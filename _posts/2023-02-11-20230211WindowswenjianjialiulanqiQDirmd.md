# Windows文件夹浏览器Q-Dir

Q-Dir是一款专为Windows操作系统设计的多窗口文件管理工具。它提供了四个独立的文件浏览器窗口，大大提升了用户在文件操作时的效率和便捷性。Q-Dir支持文件的复制、粘贴、文件夹字母快速搜索、压缩文件的查看等功能，是一款非常实用的文件管理工具。

## 主要功能

1. **四窗口界面**：Q-Dir的核心特色是其四个并排显示的文件浏览器窗口，用户可以自定义每个窗口的视图模式（如详细信息、缩略图等），并快速切换不同的文件路径。
2. **快捷操作**：支持文件的复制、粘贴、移动等操作，用户可以在不同的窗口之间轻松拖放文件。
3. **快速搜索**：支持文件夹字母快速搜索，方便用户快速定位文件。
4. **压缩文件查看**：支持查看常见的压缩文件格式，如*.zip、*.rar等。
5. **自定义颜色**：用户可以自定义窗口的背景颜色，方便区分不同的文件夹和文件类型。

## 使用方法

1. **下载与安装**：
   - 访问Q-Dir官网下载最新版本。
   - 根据系统位数选择32位或64位版本进行下载。
   - 解压下载的压缩包，运行Q-Dir程序即可。

2. **设置窗口颜色**：
   - 点击Q-Dir工具栏上的“选项”。
   - 选择“颜色和设计”。
   - 在“颜色”选项中，设置当前窗口和备用窗口的背景颜色。
