# Windows 简易版 Oracle XE 下载与安装指南

本仓库提供了一个简易版的 Oracle XE（Express Edition）数据库的下载资源，适用于 Windows 系统。Oracle XE 是一个轻量级的数据库版本，适合开发和学习使用。

## 资源内容

- **Oracle XE 安装文件**：包含 Oracle XE 数据库的安装程序。
- **安装指南**：详细介绍了如何在 Windows 系统上安装 Oracle XE 数据库。

## 安装步骤

1. **下载 Oracle XE 安装文件**：从本仓库下载 Oracle XE 的安装文件。
2. **双击安装文件**：运行下载的 `OracleXE.exe` 文件。
3. **按照向导安装**：按照安装向导的提示，逐步完成安装过程。建议选择安装路径为 `D:\oraclexe\`。
4. **设置口令**：在安装过程中，设置数据库的管理员口令。
5. **完成安装**：点击“下一步”并完成安装。安装完成后，点击“完成”按钮。
6. **管理数据库**：安装完成后，可以在浏览器中访问管理页面，使用设置的口令登录。
7. **设置环境变量**：根据需要，设置 Oracle 相关的环境变量，如 `ORACLE_HOME`、`TNS_HOME` 等。
8. **控制台登录**：在命令行中使用 `sqlplus` 命令登录 Oracle XE 数据库。

## 注意事项

- 安装过程中请确保系统满足 Oracle XE 的最低系统要求。
- 安装完成后，建议备份数据库口令，以便后续管理使用。
- 如果在安装过程中遇到问题，可以参考官方文档或社区支持。
