# uni-app完整版交互商城源码v3

## 简介
本仓库提供了一个完整的uni-app交互商城源码v3，免费共享给所有开发者使用。该源码包含了商城的核心功能和页面结构，适合用于学习和二次开发。

## 目录结构
- **store**: 存储商城数据和状态管理。
- **static**: 静态资源文件，如图片、字体等。
- **pages**: 商城的主要页面，包括首页、商品列表、商品详情等。
- **components**: 可复用的组件，如轮播图、商品卡片等。
- **common**: 公共模块和工具函数。
- **uni.scss**: uni-app的全局样式文件。
- **pages.json**: 页面路由配置文件。
- **manifest.json**: 应用配置文件，包括应用名称、版本号等。
- **main.js**: 应用的入口文件。
- **App.vue**: 应用的根组件。

## 使用说明
1. **克隆仓库**: 使用`git clone`命令将仓库克隆到本地。
2. **安装依赖**: 进入项目目录，运行`npm install`或`yarn install`安装项目依赖。
3. **运行项目**: 使用`npm run dev`或`yarn dev`启动开发服务器，进行本地开发和调试。
4. **打包发布**: 使用`npm run build`或`yarn build`进行项目打包，生成发布版本。
