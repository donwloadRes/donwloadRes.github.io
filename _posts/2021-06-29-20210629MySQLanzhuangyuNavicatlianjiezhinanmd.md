## 一、概念

1. **数据库 (DB)**  
   数据库是一个数据仓库，可以存储大量数据，是一个安装在操作系统（如 Windows、Linux、Mac 等）之上的软件。其主要作用是存储和管理数据。

2. **数据库管理系统 (DBMS)**  
   数据库管理系统是用于管理数据库的软件，能够科学有效地管理数据，维护和获取数据。

3. **MySQL**  
   MySQL 是一个关系型数据库管理系统，具有体积小、速度快、总体拥有成本低、开源等特点。

## 二、安装 MySQL

1. **下载安装包**  
   从 MySQL 官网下载适合的安装包。

2. **配置环境变量**

## 注意事项

- 安装 MySQL 时，务必使用管理员身份打开命令提示符。
- MySQL 8.0 版本不需要新建 `my.ini` 文件。
- 每次登录 MySQL 之前，确保 MySQL 服务已启动。
- 在登录 MySQL 时，可以在 `-p` 后直接加密码登录。
- 在 MySQL 中进行操作的语句后面一定要加分号。
