# Python 3.6 安装 PCL 指南

本仓库提供了一个详细的指南，帮助你在 Python 3.6 环境中安装 Point Cloud Library (PCL)。无论你是通过命令行还是使用 PyCharm 进行开发，本指南都将为你提供一步步的操作说明。

## 内容概述

本指南分为以下几个主要步骤：

1. **安装 PCL 库**：详细介绍了如何在 Windows 系统上安装 PCL 库，包括下载和安装必要的文件。
2. **配置环境变量**：如果安装过程中遇到环境变量问题，本指南提供了手动配置环境变量的方法。
3. **安装 OpenNI2**：介绍了如何安装 OpenNI2 库，这是 PCL 依赖的一部分。
4. **安装 Python PCL 库**：提供了通过 pip 安装 Python PCL 库的详细步骤。
5. **在 PyCharm 中使用**：如果你使用 PyCharm 进行开发，本指南还提供了如何在 PyCharm 中配置和使用 PCL 库的说明。

## 使用说明
