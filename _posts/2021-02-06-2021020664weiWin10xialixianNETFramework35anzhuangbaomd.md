# 64位Win10下离线.NET Framework 3.5安装包

## 简介

本仓库提供了一个适用于64位Windows 10系统的离线.NET Framework 3.5安装包。该安装包旨在解决在Windows 10系统中安装SQL Server 2008时遇到的0x80240022错误问题。通常情况下，用户在尝试从控制面板直接安装.NET Framework 3.5时也会遇到相同的问题，此时使用离线安装包可以有效解决这一问题。

## 使用说明

1. **下载安装包**：
   - 点击仓库中的`dotnetfx35.exe`文件进行下载。

2. **安装步骤**：
   - 下载完成后，双击运行`dotnetfx35.exe`文件。
   - 按照安装向导的提示完成.NET Framework 3.5的安装。

3. **注意事项**：
   - 请确保在安装过程中系统保持联网状态，以便安装程序能够正常下载所需的依赖文件。
   - 如果安装过程中遇到任何问题，请参考Windows官方文档或联系技术支持。

## 适用场景

- 在Windows 10系统中安装SQL Server 2008时遇到0x80240022错误。
- 从控制面板直接安装.NET Framework 3.5失败的情况。
