## 示例代码

以下是一个简单的示例代码，展示了如何在8x8 LED点阵上显示爱心图案：

```c
#include "max7219.h"

int main(void) {
    // 初始化max7219
    max7219_init();

    // 显示爱心图案
    uint8_t heart_pattern[8] = {
        0b00000000,
        0b01100110,
        0b11111111,
        0b11111111,
        0b11111111,
        0b01111110,
        0b00111100,
        0b00011000
    };

    max7219_display_pattern(heart_pattern);

    while (1) {
        // 主循环
    }
}
```
