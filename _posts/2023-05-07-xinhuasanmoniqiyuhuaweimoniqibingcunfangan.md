# 新华三模拟器与华为模拟器并存方案

## 简介
本资源文件提供了一个详细的方案，帮助用户在同一台电脑上同时安装和使用新华三模拟器（H3C Cloud Lab）与华为模拟器（ensp）。该方案解决了两个模拟器无法共存的问题，并附带了相关的下载地址。

## 方案背景
新华三模拟器和华为模拟器都是网络工程师常用的模拟工具，但由于技术原因，两者在同一台电脑上无法同时运行。本方案通过特定的安装步骤和配置，实现了两者的兼容运行。

## 方案逻辑
1. **卸载现有软件**：首先需要卸载电脑上已安装的VirtualBox、H3C Cloud Lab和ensp软件，确保系统干净。
2. **安装H3C Cloud Lab**：下载并安装H3C Cloud Lab，安装过程中取消默认勾选的VirtualBox安装选项。
3. **安装VirtualBox 5.2.x**：安装ensp通常使用的旧版本VirtualBox（5.2.32或5.2.44）。
4. **安装ensp**：下载并安装华为模拟器ensp，确保其能够正常运行。

## 注意事项
- 在安装H3C Cloud Lab时，务必取消默认的VirtualBox安装选项，以避免版本冲突。
- 安装VirtualBox 5.2.x版本时，选择适合ensp的版本，确保ensp能够正常识别和运行。
- 安装完成后，分别测试H3C Cloud Lab和ensp的功能，确保两者都能正常工作。

## 结论
通过本方案，用户可以在同一台电脑上同时使用新华三模拟器和华为模拟器，极大地提高了网络工程师的工作效率和实验灵活性。
