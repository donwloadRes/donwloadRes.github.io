# .NET Framework 4.8 资源证书导入指南

## 简介
本资源提供了一个名为 `Microsoft.NET4.8证书.zip` 的文件，该文件包含了在 Windows 7 系统上安装 .NET Framework 4.8 时所需的证书。如果你在安装过程中遇到“已处理证书链，但是在不受信任提供程序信任的根证书中终止”的错误提示，可以通过导入本资源提供的证书来解决该问题。

## 资源使用方法
### 获取资源文件
1. 点击本资源中的 `Microsoft.NET4.8证书.zip` 文件进行下载。
2. 将下载的 `Microsoft.NET4.8证书.zip` 文件解压到任意目录。

### 安装证书
1. 打开解压后的文件夹，找到证书文件。
2. 右键点击证书文件，选择“安装证书”。
3. 在证书导入向导中，选择“当前用户”或“本地计算机”（根据你的需求选择），然后点击“下一步”。
4. 选择“将所有的证书都放入下列存储”，然后点击“浏览”。
5. 选择“受信任的根证书颁发机构”，然后点击“确定”。
6. 点击“下一步”，然后点击“完成”。

### 安装 .NET Framework 4.8
1. 导入证书后，重新执行 .NET Framework 4.8 的安装程序。

## 注意事项
- 请确保在导入证书之前已经下载了 .NET Framework 4.8 的安装程序。
- 如果在导入证书过程中遇到任何问题，请参考 Windows 的证书管理文档或联系技术支持。
