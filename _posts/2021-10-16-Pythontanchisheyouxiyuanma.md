# Python贪吃蛇游戏源码

## 简介

本仓库提供了一个简单的Python贪吃蛇游戏源码。运行该代码后，你将能够开始一个经典的贪吃蛇游戏。通过按上下左右箭头键，你可以控制蛇的移动方向，目标是吃到食物并尽可能地延长蛇的身体，直到游戏结束。

## 如何运行

1. 确保你已经安装了Python环境。
2. 下载本仓库中的源码文件。
3. 在终端或命令行中导航到源码文件所在的目录。
4. 运行以下命令启动游戏：
   ```bash
   python snake_game.py
   ```

## 游戏操作

- **上箭头键**：向上移动蛇
- **下箭头键**：向下移动蛇
- **左箭头键**：向左移动蛇
- **右箭头键**：向右移动蛇

## 游戏目标

- 控制蛇吃到食物，每吃到一次食物，蛇的身体会变长。
- 避免蛇撞到墙壁或自己的身体，否则游戏将结束。
