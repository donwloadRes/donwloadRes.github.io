# 基于WinForm的强大流程图设计器

## 项目简介

本项目是一款专为开发者和业务需求设计的流程图制作工具，媲美专业级应用如VISIO与PPT的便捷性。采用.NET Framework 4.6与C#语言在Winform平台上精心打造，提供了丰富的功能以满足从基本绘制到复杂流程设计的需求。尽管主要提供的是可执行版本，但对渴望深度定制的用户，源代码获取方式则需联系作者本人。

## 主要特性

- **全面兼容**：直接运行，无需繁琐安装步骤。
- **图形丰富**：内置20种图形类型与2种不同的连接线样式，覆盖多数流程图设计需求。
- **高度自定义**：图形支持自由缩放、移动、连接，文本编辑具备全面的字体控制（大小、风格、颜色等）以及边框和填充色的个性化调整。
- **智能布局**：自动计算连接线路径，在移动图形时连接线能自动调整，确保界面整洁。
- **数据交换**：轻松将设计保存为XML文件，便于数据交换与持久化；同时支持导出为图片，方便分享与展示。
- **无缝集成**：特别适配于Winform应用程序，允许集成进现有或新开发的系统中，并支持二次开发，扩展无限可能。

## 使用说明

1. **运行环境**：确保您的系统已安装.NET Framework 4.6或更高版本。
2. **启动应用**：下载并双击程序文件即可直接启动流程图设计器。
3. **设计流程**：利用工具栏上的形状和连接工具，开始您的设计之旅。编辑文本、调整样式，享受创作的乐趣。
4. **保存与导出**：完成设计后，您可以将其保存为XML文件进行数据存储，或者导出为图片格式以供其他用途。
