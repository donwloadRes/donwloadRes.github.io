# 手动修改美化7zip图标 - 附替换文件

## 简介
本资源文件提供了一个手动修改和美化7zip图标的解决方案，并附带了替换文件。通过本资源，您可以轻松地将7zip的默认图标替换为更美观的图标，使其与您的系统风格更加协调。

## 使用方法
1. **下载资源文件**：首先，下载本资源文件并解压缩。
2. **备份原文件**：在替换图标之前，建议您备份7zip安装目录下的原`7z.dll`文件，以防需要恢复。
3. **替换文件**：将解压缩后的`7z.dll`文件复制到7zip的安装目录下，替换原有的`7z.dll`文件。
4. **重建图标缓存**：替换完成后，您可能需要重建系统图标缓存以查看效果。可以通过运行以下命令来重建图标缓存：
   ```
   taskkill /f /im explorer.exe
   attrib -h -i %userprofile%\AppData\Local\IconCache.db
   del %userprofile%\AppData\Local\IconCache.db /a
   start explorer.exe
   ```
5. **查看效果**：重启资源管理器后，您应该能够看到7zip的图标已经更新为美化后的图标。

## 注意事项
- 请确保在替换文件之前备份原文件，以防出现问题时可以恢复。
- 本资源文件适用于大多数版本的7zip，但如果您使用的是非常规版本或自定义安装路径，可能需要手动调整路径。
