本章习题覆盖了从基本图表制作到高级可视化技巧的多个层面，包括但不限于：
- **数据准备**：处理和清洗用于可视化的数据。
- **基础图表**：条形图、折线图、散点图的绘制及其变体。
- **进阶图形**：箱型图、热力图、直方图以及复杂组合图的应用。
- **自定义样式**：学习调整颜色、标签、图例等以提升视觉效果。
- **交互式可视化简介**：探索简单的交互性实现方法。
