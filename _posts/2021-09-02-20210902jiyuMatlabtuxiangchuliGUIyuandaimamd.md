# 基于Matlab图像处理GUI源代码

## 简介
本仓库提供了一套基于Matlab的图像处理GUI源代码，支持多种图像格式如JPG和PNG，并提供了丰富的图像处理功能，包括RGB值调整、图像旋转、滤波、边缘提取等。该源代码完全开源，用户可以根据需要进行修改和扩展。

## 功能特点
- **图像格式支持**：支持JPG、PNG等多种图像格式。
- **RGB值调整**：允许用户调整图像的RGB值，实现色彩的个性化调整。
- **图像旋转**：提供图像旋转功能，支持任意角度的旋转。
- **滤波处理**：内置多种滤波算法，如均值滤波、高斯滤波等。
- **边缘提取**：支持边缘提取功能，帮助用户更好地分析图像细节。

## 使用方法
1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```
2. **打开Matlab**：
   启动Matlab并导航到克隆的仓库目录。

   运行主程序文件，启动图像处理GUI界面。
4. **加载图像**：
   通过GUI界面加载需要处理的图像文件。
