# QT 实时显示心电波形图

## 项目介绍

本项目提供了一个基于QT框架的示例代码，展示了如何在主线程中开启子线程来初始化和读写串口数据，并通过信号将数据传送给主线程，实现实时显示心电波形图的功能。用户可以通过界面上的开关按钮来控制串口的打开、关闭以及波形图的显示和暂停。

## 功能描述

- **子线程初始化与串口读写**：在主线程中开启子线程，负责初始化和读写串口数据。
- **数据提取与信号传递**：子线程读取串口数据并提取有效数据，通过信号将数据传送给主线程。
- **界面控制**：主线程中创建界面对象，通过开关按钮槽函数控制串口的打开、关闭，图形的显示以及暂停。
- **实时刷新波形**：主线程实时刷新波形图，确保心电波形图的实时显示。

## 使用说明

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **打开项目**：
   使用QT Creator打开项目文件（通常是`.pro`文件）。

3. **编译与运行**：
   编译项目并运行，即可看到界面上的开关按钮和实时显示的心电波形图。

4. **操作界面**：
   - 点击“打开串口”按钮，启动子线程并开始读取串口数据。
   - 点击“关闭串口”按钮，停止读取串口数据。
   - 点击“暂停显示”按钮，暂停波形图的刷新。
   - 点击“继续显示”按钮，恢复波形图的刷新。
