# Google Pixel USB 驱动程序（Windows 版本）

## 简介
本仓库提供了一个适用于 Windows 系统的 Google Pixel USB 驱动程序（usb-driver-r13-windows）。通过安装此驱动程序，您可以轻松地将搭载 Android 系统的手机连接到电脑，并在两者之间传输数据。

## 功能
- **连接手机与电脑**：使用 ADB Driver 将您的 Android 手机与电脑连接。
- **数据传输**：在手机和电脑之间方便地传输数据。

## 使用方法
1. **下载驱动程序**：从本仓库下载 `usb-driver-r13-windows` 文件。
2. **安装驱动程序**：按照安装向导完成驱动程序的安装。
3. **连接手机**：使用 USB 数据线将您的 Android 手机连接到电脑。
4. **验证连接**：打开命令提示符（CMD），输入 `adb devices` 命令，确认手机已成功连接。

## 注意事项
- 确保您的电脑操作系统为 Windows。
- 在安装驱动程序之前，请确保手机已启用 USB 调试模式。
