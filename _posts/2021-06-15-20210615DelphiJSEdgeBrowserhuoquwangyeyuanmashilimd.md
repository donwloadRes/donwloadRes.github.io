# Delphi + JS + EdgeBrowser 获取网页源码示例

## 简介

近期在使用 Delphi 结合 EdgeBrowser 开发一个学习工具时，遇到了需要获取 EdgeBrowser 打开网页源码的需求。在国内各大网站上搜索了多天，始终没有找到合适的解决方案。经过一番努力和尝试，终于通过 Delphi 结合 JavaScript 成功实现了获取 EdgeBrowser 网页源码的方法。为了帮助像我一样的初学者，特此制作了一个示例 Demo，该 Demo 能够获取网页源码并将其粘贴到记事本中，供大家参考学习。

## 功能描述

- **获取网页源码**：通过 Delphi 调用 JavaScript 脚本，获取 EdgeBrowser 中当前网页的源码。
- **粘贴到记事本**：将获取到的网页源码自动粘贴到记事本中，方便查看和保存。

## 使用说明

1. **下载资源文件**：下载本仓库中的资源文件，解压后打开 Delphi 项目。
2. **运行项目**：在 Delphi 中编译并运行项目，启动程序。
3. **打开网页**：在程序中使用 EdgeBrowser 打开任意网页。
4. **获取源码**：点击程序中的“获取源码”按钮，程序将自动获取当前网页的源码，并将其粘贴到记事本中。

## 注意事项

- 本示例仅适用于 Delphi 结合 EdgeBrowser 的开发环境。
- 由于 EdgeBrowser 的限制，部分网页可能无法获取源码，具体取决于网页的安全设置。
