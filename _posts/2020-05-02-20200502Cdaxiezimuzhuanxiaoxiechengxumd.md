# C++ 大写字母转小写程序

## 描述

## 使用方法

1. 下载资源文件。
2. 打开C++开发环境（如Visual Studio、Code::Blocks等）。
3. 将下载的文件导入到项目中。
4. 编译并运行程序。
5. 输入大写字母，程序将输出对应的小写字母。

- 该程序仅适用于大写字母转小写的功能。
