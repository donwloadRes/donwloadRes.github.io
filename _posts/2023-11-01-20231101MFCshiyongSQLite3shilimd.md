# MFC 使用 SQLite3 示例

## 简介

本仓库提供了一个使用 Visual Studio 2010 编写的 MFC 对话框小例子，展示了如何在 MFC 应用程序中集成 SQLite3 数据库，并实现基本的增、删、改、查操作。特别地，本示例还演示了如何使用回调函数进行查询以及同步查询。

## 功能概述

- **增加数据**：向 SQLite 数据库中插入新记录。
- **删除数据**：从数据库中删除指定记录。
- **修改数据**：更新数据库中的现有记录。
- **查询数据**：
  - 使用回调函数进行查询。
  - 使用同步查询方式获取数据。

## 使用说明

1. **环境要求**：
   - Visual Studio 2010 或更高版本。
   - 已安装 SQLite3 库。

2. **项目导入**：
   - 下载本仓库的源代码。
   - 使用 Visual Studio 2010 打开项目文件（.sln）。

3. **编译与运行**：
   - 编译项目并生成可执行文件。
   - 运行生成的可执行文件，即可体验 MFC 与 SQLite3 的集成示例。

## 注意事项

- 本示例仅作为学习参考，实际项目中请根据需求进行适当调整。
- 确保 SQLite3 库已正确配置，否则可能导致编译错误。
