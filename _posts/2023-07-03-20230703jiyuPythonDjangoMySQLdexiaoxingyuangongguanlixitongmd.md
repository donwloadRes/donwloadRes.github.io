# 基于Python+Django+MySQL的小型员工管理系统

## 项目简介

本仓库提供了一个基于Python、Django和MySQL的小型员工管理系统。该系统旨在帮助小型企业或团队管理员工信息，提供基本的员工信息录入、查询、修改和删除功能。系统设计简洁，易于上手，适合初学者学习和使用。

## 功能特点

- **员工信息管理**：支持员工信息的录入、查询、修改和删除。
- **简单易用**：界面简洁，操作方便，适合小型团队使用。
- **数据存储**：使用MySQL数据库进行数据存储，确保数据的安全性和可靠性。

## 使用说明

1. **环境配置**：
   - 确保已安装Python 3.x。
   - 安装Django框架：`pip install django`。
   - 安装MySQL数据库，并确保数据库服务已启动。

2. **数据库配置**：
   - 在Django项目的`settings.py`文件中配置MySQL数据库连接信息。

3. **运行项目**：
   - 在项目根目录下运行命令：`python manage.py runserver`。
   - 打开浏览器，访问`http://127.0.0.1:8000/`即可进入系统。

## 注意事项

- 请确保MySQL数据库已正确配置，并创建相应的数据库和表结构。
- 系统功能较为简单，适合小型团队或个人使用，不适合大规模企业。
