# 微信小程序点餐系统源码

欢迎使用微信小程序点餐系统源码！本项目提供了一个完整的点餐系统解决方案，结合了Java后端和微信小程序前端，旨在帮助开发者快速搭建一个功能齐全的点餐平台。

## 项目概述

本项目包含以下主要部分：

- **后端服务**：使用Java开发，提供API接口供微信小程序调用。
- **微信小程序**：提供用户界面，实现点餐、下单、支付等功能。

## 功能特点

- **用户管理**：用户注册、登录、个人信息管理。
- **菜单浏览**：展示餐厅菜单，支持分类浏览。
- **购物车功能**：添加、删除、修改购物车中的商品。
- **订单管理**：创建订单、查看订单状态、订单支付。
- **支付集成**：集成微信支付功能，实现快速支付。
- **后台管理**：管理员可以管理菜单、订单、用户等。

## 技术栈

- **后端**：Java, Spring Boot, MyBatis, MySQL
- **前端**：微信小程序, WXML, WXSS, JavaScript

## 快速开始

### 后端部署

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/wechat-miniapp-ordering-system.git
   ```

2. **配置数据库**：
   - 创建MySQL数据库并导入`database.sql`文件。
   - 修改`application.properties`中的数据库连接信息。

3. **启动后端服务**：
   ```bash
   cd backend
   mvn spring-boot:run
   ```

### 微信小程序部署

1. **导入项目**：
   - 使用微信开发者工具导入`frontend`目录。

2. **配置API地址**：
   - 修改`config.js`中的API地址为后端服务的地址。

3. **运行小程序**：
   - 在微信开发者工具中点击“编译”并预览。
