# Web酒店管理系统（附源码及资源）

## 项目简介

本项目提供了一个完整的Web酒店管理系统，包括前台页面和后台页面。前台页面用于客户浏览与选择预订客房，后台页面则只能由酒店管理人员进行操作。系统实现了用户与酒店管理人员的登录、客房信息与酒店信息的展示、客房预订界面与新闻资讯界面等功能。

## 系统功能

### 前台模块
- 用户登录
- 客房信息展示
- 酒店信息展示
- 客房预订界面
- 新闻资讯界面

### 后台模块
- 客户信息管理
- 酒店客房信息管理
- 增删改查功能

## 系统主要内容

1. **酒店主页**：展示酒店概要等相关信息。
2. **客房预订界面**：实现客房的预订功能。

## 使用技术

- HTML
- CSS
- JavaScript
- Sublime Text编辑器

## 安装与运行

1. 下载源码及资源文件。
2. 使用Sublime Text编辑器打开项目。
3. 根据需要修改相关配置文件。
4. 运行项目，访问前台页面进行预订，后台页面进行管理操作。

## 注意事项

- 系统中设置了英文界面与酒店的地理位置联系方式等信息。
- 系统还提供了相关的新闻资讯，可以直接点击跳转至相关页面。
