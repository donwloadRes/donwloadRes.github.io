# C# 扫描蓝牙 WinForm 示例

## 简介

本仓库提供了一个使用 C# 编写的 WinForm 示例程序，用于扫描并显示附近的蓝牙设备。该示例程序展示了如何使用 C# 和 WinForm 技术来实现蓝牙设备的扫描功能。

## 功能特点

- **蓝牙设备扫描**：程序能够扫描并列出附近的蓝牙设备。
- **简单易用**：界面简洁，操作方便，适合初学者学习和参考。
- **开箱即用**：下载后可以直接运行，无需额外配置。

## 使用方法

1. **下载资源**：点击仓库中的下载按钮，获取示例程序的源代码和相关资源文件。
2. **打开项目**：使用 Visual Studio 或其他支持 C# 的 IDE 打开项目文件。
3. **运行程序**：编译并运行程序，即可看到扫描到的蓝牙设备列表。

## 注意事项

- 确保您的计算机支持蓝牙功能，并且蓝牙驱动程序已正确安装。
- 运行程序时，请确保蓝牙已开启。
