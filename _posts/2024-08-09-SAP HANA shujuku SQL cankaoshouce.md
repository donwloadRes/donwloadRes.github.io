## 内容概述

1. **数据类型**: 详细介绍了 SAP HANA 数据库支持的各种数据类型，包括数值类型、字符类型、日期类型等。
2. **谓词**: 解释了 SQL 查询中常用的谓词，如 `WHERE`、`HAVING` 等，并提供了使用示例。
3. **操作符**: 介绍了 SQL 中的各种操作符，包括算术操作符、比较操作符、逻辑操作符等。
4. **表达式**: 讲解了 SQL 表达式的构成和使用方法，帮助读者理解如何在查询中构建复杂的表达式。
5. **函数**: 列出了 SAP HANA 数据库支持的常用函数，包括聚合函数、字符串函数、日期函数等。
6. **语句**: 详细描述了 SQL 语句的语法和使用场景，如 `SELECT`、`INSERT`、`UPDATE`、`DELETE` 等。
