# Unity VR 开发教程：OpenXR + XR Interaction Toolkit 手部动画资源包

## 简介

本资源包是为 Unity VR 开发教程系列中的“OpenXR + XR Interaction Toolkit (二) 手部动画”部分提供的资源文件。该资源包包含了手部模型的动画及相关配置，帮助开发者快速实现 VR 应用中的手部动画效果。

## 内容

- **手部模型**：包含左手和右手模型，适用于 Oculus Quest 2 等 VR 设备。
- **动画文件**：预设的手部动画，包括捏合和抓握动作。
- **Animator Controller**：用于控制手部动画的 Animator Controller。
- **脚本示例**：提供了基于 Input System 和 XR Input Subsystem 的手部动画控制脚本。

## 使用说明

1. **导入资源包**：
   - 在 Unity 中，点击菜单栏的 `Assets -> Import Package -> Custom Package`，选择下载的资源包文件进行导入。

2. **配置场景**：
   - 将左手和右手模型分别拖入场景中的 `LeftHand Controller` 和 `RightHand Controller`。
   - 为手部模型添加相应的 Animator Controller。

3. **编写脚本**：
   - 参考提供的脚本示例，编写或修改脚本以控制手部动画。
   - 脚本示例包括基于 Input System 和 XR Input Subsystem 的实现方式。

4. **调整场景**：
   - 根据实际需求调整手部模型的旋转角度和位置，确保与现实世界中的手部位置一致。
   - 调整 URP 配置文件中的阴影设置，优化手部模型的阴影效果。

## 注意事项

- 本资源包适用于 Unity 2021.3.5 及以上版本。
- 使用本资源包前，请确保已正确配置 OpenXR 和 XR Interaction Toolkit 环境。
- 如果使用其他手部模型，请参考资源包中的动画制作思路，自定义手部动画。
