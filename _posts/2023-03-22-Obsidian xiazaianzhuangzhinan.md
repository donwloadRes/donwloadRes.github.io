# Obsidian 下载安装指南

## 简介
本仓库提供Obsidian的下载安装资源，帮助用户快速获取并安装Obsidian，以便更好地管理和组织个人知识。

## 资源内容
- **Windows版本**：提供直接安装的Windows版本Obsidian。
- **Linux版本**：包括snap和AppImage两种格式的Linux版本Obsidian。

## 安装步骤

### Windows版本
1. 下载提供的Windows安装包。
2. 双击安装包，按照提示完成安装。

### Linux版本
#### Snap格式
1. 下载提供的snap安装包。
2. 打开终端，输入以下命令进行安装：
   ```bash
   sudo apt install snap
   sudo snap install --dangerous obsidian_0.12.12_amd64.snap
   ```

#### AppImage格式
1. 下载提供的AppImage安装包。
2. 右键点击AppImage文件，选择“属性”。
3. 在“权限”选项卡中，勾选“允许作为程序执行文件”。
4. 双击AppImage文件即可运行Obsidian。

## 注意事项
- 确保系统满足Obsidian的最低运行要求。
- 安装过程中如有问题，请参考官方文档或社区论坛。
