# JDK8下载安装与Win10下Java环境变量配置

本仓库提供JDK8的下载资源，并详细介绍了在Windows 10系统下如何进行JDK8的安装以及Java环境变量的配置。以下是详细的步骤说明：

## 1. JDK8下载

### 1.1 官方下载地址

JDK8的官方下载地址为：https://www.oracle.com/java/technologies/javase-downloads.html

### 1.2 选择合适的版本

在下载页面中，选择适合您操作系统的JDK8版本进行下载。通常，大多数用户会选择64位的JDK8版本。

### 1.3 下载步骤

1. 打开下载页面后，向下滚动找到JDK8版本。
2. 点击“JDK Download”按钮，跳转到下载页面。
3. 选择适合您操作系统的安装包，并同意协议开始下载。

## 2. JDK8安装

### 2.1 安装步骤

1. 下载完成后，双击安装包开始安装。
2. 按照安装向导的提示一步步完成安装。
3. 默认安装位置通常在C盘，建议更改到其他盘以避免占用C盘空间。

## 3. Java环境变量配置

### 3.1 配置JAVA_HOME

1. 在电脑桌面右键点击“此电脑”，选择“属性”。
2. 选择“高级系统设置”，点击“环境变量”。
3. 在“系统变量”区域点击“新建”。
4. 在“变量名”处填上“JAVA_HOME”，在“变量值”处填上JDK的安装路径（例如：D:\Java\jdk1.8.0_20）。
5. 点击“确定”保存。

### 3.2 配置Path

1. 在“系统变量”中找到“Path”，点击“编辑”。
2. 选择右边的“新建”，将`%JAVA_HOME%\bin`复制到新建的条目中。
3. 再次点击“新建”，将`%JAVA_HOME%\jre\bin`复制到新建的条目中。
4. 点击“确定”保存。

### 3.3 配置CLASSPATH

1. 在“系统变量”区域点击“新建”。
2. 在“变量名”处填上“CLASSPATH”，在“变量值”处填上：
   ```
   .;%JAVA_HOME%\bin;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar;
   ```
3. 点击“确定”保存。

## 4. 验证环境变量是否配置成功

1. 回到电脑桌面，按快捷键“Win+R”，输入“cmd”打开命令提示符。
2. 输入以下命令检查Java环境是否配置成功：
   ```
   java
   javac
   java -version
   ```
3. 如果能够正常输出相关信息，则说明JDK安装和Java环境变量配置成功。

通过以上步骤，您可以顺利完成JDK8的下载、安装以及环境变量的配置，确保Java开发环境正常运行。
