# JDK 1.6 免安装版本

## 简介

本资源文件提供了一个免安装版本的JDK 1.6，适用于Windows和Linux操作系统。JDK 1.6是Java语言的软件开发工具包，主要用于移动设备、嵌入式设备上的Java应用程序开发。JDK 1.6包含了Java的运行环境（JVM+Java系统类库）和Java工具，是整个Java开发的核心。

- **免安装**：无需安装，直接解压即可使用。

## 使用方法

1. **下载资源文件**：从本仓库下载JDK 1.6的压缩包。
2. **解压文件**：将下载的压缩包解压到任意目录。
3. **配置环境变量**：
   - 在Windows系统中，右键点击“我的电脑”，选择“属性” -> “高级系统设置” -> “环境变量”。
   - 在“系统变量”中添加新的变量：
     - `JAVA_HOME`：指向解压后的JDK目录。
     - `PATH`：在原有路径后添加`%JAVA_HOME%\bin`。
   - 在Linux系统中，编辑`~/.bashrc`或`~/.bash_profile`文件，添加以下内容：
     ```bash
     export JAVA_HOME=/path/to/jdk1.6
     export PATH=$JAVA_HOME/bin:$PATH
     ```
4. **验证安装**：打开命令行工具，输入`java -version`，如果显示JDK 1.6的版本信息，则配置成功。
