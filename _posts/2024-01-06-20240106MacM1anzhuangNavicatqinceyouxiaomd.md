# Mac M1 安装 Navicat 亲测有效

本资源文件提供了在 Mac M1 芯片上安装 Navicat 的详细步骤和解决方案。以下是安装过程的简要说明：

## 安装步骤

1. **下载 Navicat**  
   从官方网站或其他可信来源下载适用于 Mac M1 的 Navicat 安装包。

2. **安装 Navicat**  
   双击下载的安装包，按照提示完成安装。将 Navicat 拖拽到应用程序文件夹中。

3. **解决安装后无法打开的问题**  
   如果在安装后双击图标提示“移至垃圾桶”，请按照以下步骤解决：
   - 打开系统偏好设置，选择“安全与隐私”。
   - 将“允许以下位置下载的应用”调整为“App Store 和被认可的开发者”。

4. **终端命令解决权限问题**  
   打开终端，输入以下命令并回车：
   ```
   sudo xattr -r -d com.apple.quarantine /Applications/Navicat.app
   ```
   其中 `/Applications/Navicat.app` 是 Navicat 的安装路径。

5. **测试连接**  
   安装完成后，打开 Navicat，连接到您的数据库进行测试。

## 注意事项

- 确保下载的 Navicat 版本与 Mac M1 芯片兼容。
- 安装过程中可能需要输入管理员密码。
- 如果遇到其他问题，请参考相关技术文档或联系 Navicat 支持团队。

通过以上步骤，您可以在 Mac M1 上成功安装并使用 Navicat。
