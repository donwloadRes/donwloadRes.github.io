# 中文翻译《线性代数导论》第五版 2.2节

## 资源描述

本资源文件是《线性代数导论》第五版 2.2节的中文翻译。本节内容详细阐述了一种解决线性方程组的系统方法，即“消元法”。通过一个 2 × 2 的例子，展示了消元法的具体步骤和应用。

在消元之前，未知数 x 和 y 在两个方程中均有出现。经过消元后，第一个未知数 x 从第二个方程 8y = 8 中消失。消元过程如下：

- 消元前：
  - x − 2y = 1
  - 3x + 2y = 11

- 消元后：
  - x − 2y = 1
  - 8y = 8

新方程 8y = 8 立即得出 y = 1。将 y = 1 代入第一个方程中，得到 x − 2 = 1，因此 x = 3。最终求解结果为 (x, y) = (3, 1)。

消元法产生了一个上三角方程组，这是目标。非零系数 1, −2, 8 形成了一个三角形。这个方程组从底向上求解，首先得出 y = 1，然后得出 x = 3。这个快速过程被称为回代，适用于任何大小的上三角方程组。

重点：原先的方程组具有相同的解 x = 3 与 y = 1。图 2.5 揭示了每个方程组都是一对直线，在交点处相交。
