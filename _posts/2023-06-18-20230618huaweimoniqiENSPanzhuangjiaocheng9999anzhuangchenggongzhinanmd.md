# 华为模拟器ENSP安装教程【99.99%安装成功指南】

欢迎使用华为模拟器ENSP（Enterprise Network Simulation Platform）安装教程，本教程旨在帮助您高效、顺利地安装这一强大的网络仿真工具。ENSP是由华为提供的免费软件，专为企业级网络路由器、交换机等设备的仿真搭建提供图形化操作平台，适用于网络技术的学习与实验。

## 系统要求
- **操作系统**: 推荐使用Windows 10教育版。
- **硬件需求**: 中高端处理器与足够的内存空间，以保证流畅的仿真体验。

## 安装步骤概览
1. **准备阶段**：
   - 下载软件包（包括ENSP、VirtualBox、WinPcap和Wireshark）。
   - 验证系统兼容性，并关闭Windows防火墙。

2. **安装前置软件**：
   1. **VirtualBox**: 右键以管理员身份运行安装程序。
   2. **WinPcap**: 安装时若已存在旧版本，考虑重新安装确保兼容。
   3. **Wireshark**: 同样需以管理员身份安装，建议安装至非C盘目录。

3. **ENSP安装**：
   - 解压缩下载的ENSP安装包。
   - 右键点击“eNSP_Setup.exe”，以管理员身份启动安装。
   - 跟随安装向导，同意许可协议，自定义安装路径（避免中文），直至完成安装。

4. **完成与启动**：
   - 安装完毕后，直接启动ENSP软件，开始您的网络仿真实验之旅。

## 注意事项
- 确保所有安装过程中的软件均为最新版本。
- 避免安装路径中含有中文字符，以免引起不必要的错误。
- 如遇先前安装过ENSP，请彻底清理注册表后再进行新版本安装。
