# 黑马程序员 Python 可视化案例：P105-107 地图可视化笔记

## 简介

本资源文件是黑马程序员 Python 可视化案例系列的一部分，专注于地图可视化。该笔记涵盖了从 P105 到 P107 的课程内容，详细介绍了如何使用 Python 进行地图可视化。通过本笔记，您将学习如何使用 `pyecharts` 库构建基础的地图可视化图表，并应用于实际案例中。

## 内容概述

1. **测试地图实现代码**
   - 演示如何使用 `pyecharts` 创建基础的地图可视化图表。
   - 提供了详细的代码示例和结果展示。

2. **全国疫情地图实现代码**
   - 展示了如何将疫情数据可视化到全国地图上。
   - 包括数据读取、处理和地图绘制的完整流程。

3. **省级疫情地图可视化实现代码**
   - 进一步细化到省级疫情地图的可视化。
   - 提供了河南省疫情地图的实现代码和结果展示。

## 使用说明

1. **环境准备**
   - 确保您已经安装了 Python 和 `pyecharts` 库。
   - 可以通过 `pip install pyecharts` 命令安装 `pyecharts`。

2. **代码运行**
   - 按照笔记中的代码示例，逐步运行代码。
   - 注意数据文件的路径设置，确保数据文件能够正确读取。

3. **结果展示**
   - 运行代码后，您将看到生成的地图可视化图表。
   - 可以通过浏览器打开生成的 HTML 文件查看结果。
