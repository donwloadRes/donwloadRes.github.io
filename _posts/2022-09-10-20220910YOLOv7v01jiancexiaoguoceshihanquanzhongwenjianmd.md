# YOLOv7-v0.1 检测效果测试（含权重文件）

## 简介

本资源文件提供了YOLOv7-v0.1版本的源码、权重文件以及环境依赖库的下载链接。通过运行Python脚本`detect.py`，用户可以使用预训练权重在指定图片上展示YOLOv7的检测性能。

## 内容

1. **源码下载**：包含YOLOv7-v0.1版本的完整源码，用户可以根据需要进行修改和测试。
2. **权重文件**：提供了预训练的权重文件，用户可以直接使用这些权重进行目标检测效果的测试。
3. **环境依赖库**：列出了运行YOLOv7-v0.1所需的所有依赖库，用户可以根据需要进行安装。

## 使用方法

1. **下载源码**：从提供的链接下载YOLOv7-v0.1的源码。
2. **下载权重文件**：从提供的链接下载预训练的权重文件。
3. **安装环境依赖库**：根据提供的依赖库列表，使用`pip3 install -r requirements.txt`命令安装所有依赖库。
4. **测试检测效果**：运行以下命令进行目标检测效果的测试：
   ```bash
   python3 detect.py --weights yolov7.pt --conf 0.25 --img-size 640 --source inference/images/horses.jpg
   ```

## 注意事项

- 确保所有依赖库已正确安装，否则可能会导致程序无法正常运行。
- 测试时请确保图片路径正确，否则可能会出现文件找不到的错误。
