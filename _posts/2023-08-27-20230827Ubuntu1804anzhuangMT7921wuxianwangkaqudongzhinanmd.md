# Ubuntu 18.04 安装 MT7921无线网卡驱动指南

本资源文件提供了在Ubuntu 18.04系统上安装MT7921无线网卡驱动的详细步骤和相关资源。MT7921是一款常见的无线网卡，但在某些Linux系统上可能需要手动安装驱动才能正常使用。

## 内容概述

1. **系统状态检查**：
   - 检查BIOS的Secure Boot是否关闭。
   - 安装编译内核所需的软件flex和bison。

2. **驱动安装步骤**：
   - 下载并解压驱动文件。
   - 编译并安装驱动。
   - 重启系统以应用更改。

3. **常见问题与解决方案**：
   - 提供了解决安装过程中可能遇到的问题的建议。

## 使用说明

1. **下载资源**：
   - 下载本资源文件中提供的驱动文件。

2. **安装依赖**：
   - 在终端中运行以下命令以安装必要的依赖：
     ```bash
     sudo apt-get install -y flex bison
     ```

3. **编译与安装**：
   - 解压下载的驱动文件，进入解压后的目录。
   - 运行以下命令以编译和安装驱动：
     ```bash
     make clean
     make defconfig-wifi
     make
     sudo make install
     ```

4. **重启系统**：
   - 安装完成后，重启系统以使驱动生效。

## 注意事项

- 确保BIOS中的Secure Boot已关闭，否则驱动可能无法正常加载。
- 如果在安装过程中遇到问题，请参考提供的常见问题与解决方案部分。
