本仓库提供了一个适用于Unity3D项目的常用3500汉字字库资源文件。该字库包含了中文常用汉字的集合，适用于开发中需要显示中文文本的游戏或应用程序。

## 使用方法
1. **下载资源文件**：
   - 点击仓库页面中的“Clone or download”按钮，选择“Download ZIP”下载整个仓库的压缩包。
   - 或者使用Git命令克隆仓库到本地：
     ```bash
     git clone https://github.com/your-repo-url.git
     ```

2. **导入资源文件**：
   - 将下载或克隆的资源文件解压后，导入到你的Unity3D项目中。
   - 在Unity编辑器中，通过“Assets”菜单选择“Import Package” -> “Custom Package”，然后选择下载的字库文件进行导入。

3. **在项目中使用字库**：
   - 在Unity3D项目中，通过TextMesh Pro或其他文本组件引用导入的字库文件。
   - 确保你的文本组件设置正确，以便使用该字库显示中文文本。

## 注意事项
- 请确保你的Unity3D项目已安装TextMesh Pro插件，以便更好地使用该字库资源。
- 如果字库文件缺失或显示异常，请检查导入路径和文件完整性。
