# Dev C++ 简介与安装指南

## 概述
Dev-C++ 是一个适用于 Windows 环境的轻量级 C/C++ 集成开发环境（IDE），特别适合初学者使用。它是一款自由软件，遵循 GPL 许可协议，集成了 GCC 编译器、GDB 调试器和 AStyle 格式整理器等众多工具。

## 主要特点
- **轻量级**：适合初学者和非商业级开发者使用。
- **集成开发环境**：集成了编译器、调试器和格式整理器。
- **多语言支持**：支持多国语言，包括简体中文。
- **自由软件**：遵循 GPL 许可协议，源代码开放。

## 安装步骤
1. **下载安装包**：从官方网站或提供的下载链接获取 Dev C++ 安装包。
2. **启动安装程序**：双击安装包文件，开始安装过程。
3. **选择安装语言**：安装过程中选择英文（安装完成后可设置为简体中文）。
4. **同意条款**：阅读并同意 Dev C++ 的各项条款。
5. **选择安装组件**：建议选择“Full”以安装所有组件。
6. **选择安装路径**：选择合适的安装路径，路径中最好不要包含中文。
7. **等待安装完成**：安装过程可能需要几分钟，完成后启动 Dev C++。
8. **首次配置**：首次启动时选择简体中文，并根据个人喜好配置字体和主题风格。

## 使用说明
- **编写代码**：启动 Dev C++ 后，即可开始编写 C/C++ 代码。
- **编译与调试**：使用集成工具进行代码编译和调试。
- **格式整理**：利用 AStyle 工具整理代码格式，提高代码可读性。

## 注意事项
- **路径选择**：安装路径中避免使用中文，以免出现兼容性问题。
- **更新与维护**：定期检查官方网站，获取最新版本和更新。

通过以上步骤，您可以顺利安装并开始使用 Dev C++ 进行 C/C++ 编程。
