# 在VS2022(QT)或QTCreator上配置并显示PCL点云文件

## 简介
本资源文件详细介绍了如何在Visual Studio 2022（结合QT）或QTCreator上配置并显示PCL（Point Cloud Library）点云文件。通过本教程，您将能够成功安装PCL库，并在开发环境中配置相关环境变量和依赖项，最终实现点云文件的可视化显示。

## 主要内容
1. **安装PCL**
   - 下载PCL库文件
   - 配置系统环境变量
   - 安装OpenNI2

2. **安装QT**
   - 下载并安装QT
   - 配置QT环境变量

3. **在Visual Studio 2022上配置PCL**
   - 添加属性表
   - 配置VC++目录
   - 配置预处理器
   - 配置附加依赖项
   - 配置调试环境

4. **在Visual Studio 2022上配置QT**
   - 下载并安装Qt Visual Studio Tools
   - 关联QT
   - 将PCL嵌入QT

5. **在QTCreator上配置PCL**
   - 安装msvc编译器
   - 配置pro文件

## 使用说明
1. **下载资源文件**
   - 下载本仓库中的资源文件，包含配置PCL和QT所需的文件和脚本。

2. **按照教程步骤操作**
   - 根据上述主要内容中的步骤，逐步配置您的开发环境。

3. **运行示例项目**
   - 配置完成后，运行提供的示例项目，验证PCL点云文件是否能够正常显示。
