# Python绘制新疆地区地形图

## 内容概述

- **数据处理**：使用xarray库加载DEM（数字高程模型）数据，并筛选出新疆地区的经纬度范围。
- **数据可视化**：使用matplotlib和cartopy库绘制地形图，并添加地理边界和颜色条。
- **代码示例**：提供了完整的Python代码，展示了如何从数据加载、处理到最终的地形图绘制。

## 使用方法

1. **安装依赖库**：
   ```bash
   pip install xarray matplotlib numpy cartopy
   ```

2. **下载DEM数据**：
   从合适的来源下载DEM数据，并将其路径替换到代码中的`filename`变量。

3. **运行代码**：
   将提供的Python代码保存为`.py`文件，并在安装了依赖库的环境中运行。
