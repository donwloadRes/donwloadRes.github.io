### 1. WinCC全局脚本动作 - 隐藏控件

该脚本演示了如何在WinCC中使用全局脚本动作来隐藏控件。通过触发变量`IM_in`，脚本会根据变量的值来控制画面中某个控件的可见性。

```c
int value;
value = GetTagBit("IM_in"); // IM_in为触发的变量
if (value == 1) {
    SetVisible("lianxi.pdl", "按钮8", TRUE); // lianxi.pdl为控制对象所在画面名，“按钮8”为对象名称
} else {
    SetVisible("lianxi.pdl", "按钮8", FALSE);
}
```

**说明**：在触发器里选择变量`IM_in`。
