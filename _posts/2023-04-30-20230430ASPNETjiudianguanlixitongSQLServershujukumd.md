# ASP.NET 酒店管理系统 - SQL Server 数据库

## 项目简介

本仓库提供了一个基于ASP.NET开发的酒店管理系统，该系统使用SQL Server数据库进行数据存储。该项目是一个期末考核项目，旨在帮助开发者理解和掌握ASP.NET与SQL Server的结合应用。

## 项目特点

- **技术栈**：ASP.NET + SQL Server
- **功能模块**：包括客房管理、预订管理、客户管理、员工管理等核心功能。
- **开发文档**：项目附带详细的开发文档，方便开发者参考和学习。

## 使用说明

1. **下载资源**：请从仓库中下载完整的项目文件。
2. **数据库配置**：根据项目文档中的说明，配置SQL Server数据库。
3. **运行项目**：使用Visual Studio或其他支持ASP.NET的IDE打开项目，并运行调试。

## 注意事项

- 请确保已安装SQL Server数据库，并配置好相关连接字符串。
- 开发文档中包含了详细的配置步骤和代码说明，建议仔细阅读。
