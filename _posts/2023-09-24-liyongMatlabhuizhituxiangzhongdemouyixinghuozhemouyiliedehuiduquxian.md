# 利用Matlab绘制图像中的某一行或者某一列的灰度曲线

本资源文件提供了一个利用Matlab绘制图像中某一行或某一列的灰度曲线的示例代码。通过该代码，用户可以轻松地提取图像中特定行或列的灰度值，并将其绘制成曲线图。

## 功能介绍

- **灰度曲线绘制**：用户可以选择图像中的任意一行或一列，代码将提取该行或列的灰度值，并绘制出相应的灰度曲线。
- **图像显示**：代码还提供了图像显示功能，用户可以在绘制灰度曲线之前先查看原始图像。

## 使用方法

1. **读取图像**：使用`imread`函数读取图像文件。
2. **显示图像**：使用`imshow`函数显示读取的图像。
3. **提取灰度值**：使用矩阵索引提取特定行或列的灰度值。
4. **绘制曲线**：使用`plot`函数绘制提取的灰度值曲线。

## 示例代码

```matlab

imshow(imgData);

% 提取第一列的灰度值
A = imgData(:, 1);
plot(A, '-');
```
