# ESP8266 2.7.2 离线安装包使用指南

## 简介
本仓库提供了一个名为 `esp8266-2.7.2 离线安装包.zip` 的资源文件，该文件是用于在 Arduino IDE 中安装 ESP8266 开发板的离线安装包。由于通过 Arduino IDE 的开发板管理器直接下载 ESP8266 时经常遇到下载困难的问题，因此我们提供了这个已经下载好的文件，并附上了详细的使用方法。

## 资源文件说明
- **文件名**: `esp8266-2.7.2 离线安装包.zip`
- **文件内容**: 包含 ESP8266 2.7.2 版本的离线安装包，SHA256 校验无误，用户可以自行比对。

## 使用方法
1. **下载文件**: 从本仓库下载 `esp8266-2.7.2 离线安装包.zip` 文件。
2. **解压缩**: 将下载的 ZIP 文件解压缩到任意目录。
3. **打开 Arduino IDE**: 启动 Arduino IDE。
4. **添加开发板管理器 URL**:
   - 打开 Arduino IDE 的“首选项”（Preferences）。
   - 在“附加开发板管理器网址”（Additional Board Manager URLs）中添加以下 URL：
     ```
     http://arduino.esp8266.com/stable/package_esp8266com_index.json
     ```
   - 点击“确定”保存设置。
5. **安装开发板**:
   - 进入“工具”（Tools）菜单，选择“开发板”（Board），然后点击“开发板管理器”（Board Manager）。
   - 在开发板管理器中，搜索“ESP8266”。
   - 选择“ESP8266 by ESP8266 Community”，然后点击“安装”（Install）。
   - 在安装过程中，Arduino IDE 会提示需要下载 ESP8266 的安装包。此时，选择“使用本地文件”（Use local file），然后选择之前解压缩的 `esp8266-2.7.2 离线安装包.zip` 文件。
6. **完成安装**:
   - 等待安装完成，安装过程中需要保持网络畅通，因为会有联网验证的步骤。
   - 安装完成后，您就可以在 Arduino IDE 中选择 ESP8266 开发板进行开发了。

## 注意事项
- 虽然是离线安装包，但在安装过程中仍然需要联网验证，因此请确保在安装过程中网络畅通。
- 如果安装过程中遇到任何问题，请检查网络连接或尝试重新安装。
