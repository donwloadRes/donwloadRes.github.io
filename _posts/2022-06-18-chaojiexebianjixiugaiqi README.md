# 超级exe编辑修改器 README

欢迎使用超级exe编辑修改器 v18.00.0！本版本是一个强大的工具，专为需要深度编辑和修改EXE文件的用户设计。我们重点增强了智能化功能，确保您的编程和编辑体验更加高效、灵活。

## 主要更新亮点：

- **智能模板系统**：
    - 经过重新设计的非模态修改模板对话框，带来更流畅的操作体验。
    - 用户现可添加自定义变量，实现高度定制化的模板。
    - 引入智能推荐功能，根据输入内容推荐最合适的模板。
    - 支持通过自动完成对话框直接插入模板，以及拖拽操作，提升效率。
    - 新增针对不同语法高亮语言的模板分类管理。
    - 增强配置选项，允许更细致地控制模板行为。

- **增强的HTML/XML标签处理**：
    - 自动完成并封闭HTML与XML标签，可根据个人偏好设置在同一行或新行封闭。
    - 高级配置中提供了更多编辑器相关设置以优化此功能。

- **便捷的查找结果交互**：
    - 支持双击快速定位包含目标字符串的行。
    - 查找结果窗口支持浮动吸附或自动隐藏模式，且加入了丰富的右键菜单操作选项。

- **字体支持优化**：
    - 全面支持比例字体，改善了非西文（如中文）的显示问题。
    - 对非等线体字体进行特殊优化，特别是在语法高亮时的视觉效果。

- **性能与便利性升级**：
    - 最近文件列表容量扩大至32个，方便快速访问历史文档。
    - 提升了FTP账号管理的速度及用户体验，包括打开对话框时的自动连接功能。
    - 适应现代系统，新增对64位环境下32位和64位外壳集成DLL的支持，确保兼容性和稳定性。
