# C# + SQL Server 医院管理系统（电子病历系统）源码

## 项目简介

本项目是一个基于C#和SQL Server开发的医院管理系统，也称为电子病历系统。该系统旨在帮助医院管理病人和医生的信息，提供便捷的数据录入、存储、修改、查询和删除功能。系统还具备数据维护功能，确保在紧急情况下医生能够通过IC卡快速了解病人病情，并进行相应的治疗。此外，系统还集成了药方药库管理，方便病人就医和取药。

## 功能介绍

### 主要功能模块

1. **门诊医生管理**：医生信息的录入、修改、查询和删除。
2. **系统管理**：用户权限管理，确保只有授权人员可以修改数据。
3. **门诊管理**：病人信息的录入、修改、查询和删除。
4. **财务管理**：医院财务数据的记录和管理。
5. **药房管理**：药品信息的录入、修改、查询和删除。
6. **住院管理**：住院病人信息的录入、修改、查询和删除。
7. **门诊收费**：门诊费用的计算和记录。
8. **药库管理**：药品库存的管理和维护。

### 特色功能

- **IC卡支持**：医生可以通过IC卡快速了解病人病情，进行紧急治疗。
- **药方药库集成**：根据医生的处方自动开药，方便病人取药。

## 开发环境

- **开发工具**：Visual Studio 2010
- **数据库**：SQL Server 2008 R2

## 使用说明

1. **安装环境**：确保您的计算机上已安装Visual Studio 2010和SQL Server 2008 R2。
2. **导入数据库**：将项目中的SQL脚本导入到SQL Server 2008 R2中，创建数据库和表结构。
3. **打开项目**：使用Visual Studio 2010打开项目文件，编译并运行程序。
4. **配置连接字符串**：根据您的数据库配置，修改项目中的连接字符串，确保程序能够正确连接到数据库。
