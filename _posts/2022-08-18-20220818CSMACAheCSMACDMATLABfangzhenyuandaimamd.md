# CSMA/CA 和 CSMA/CD MATLAB 仿真源代码

## 简介

本仓库提供了用于仿真 CSMA/CA（载波侦听多路访问/冲突避免）和 CSMA/CD（载波侦听多路访问/冲突检测）的 MATLAB 源代码。代码中包含了详细的注释，方便用户理解和修改。此外，用户可以随意设置节点数，并且仿真结果带有炫酷的图形展示效果。

## 功能特点

- **详细注释**：代码中包含了详细的注释，帮助用户理解每个步骤和功能。
- **节点数可调**：用户可以根据需要随意设置仿真中的节点数。
- **图形展示**：仿真结果以图形化的方式展示，直观且易于理解。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **打开 MATLAB**：
   启动 MATLAB 并导航到克隆的仓库目录。

3. **运行仿真**：
   打开相应的 MATLAB 脚本文件（例如 `csma_ca.m` 或 `csma_cd.m`），根据需要调整节点数等参数，然后运行脚本。

4. **查看结果**：
   仿真运行后，结果将以图形化的方式展示在 MATLAB 图形窗口中。
