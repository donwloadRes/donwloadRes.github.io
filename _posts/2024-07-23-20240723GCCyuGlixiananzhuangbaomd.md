# GCC与G++离线安装包

## 简介

本仓库提供了一个适用于无网环境的GCC与G++离线安装包，版本为4.8.5。压缩包内含一个名为`install.sh`的安装脚本，运行该脚本即可完成GCC与G++的安装。此安装包设计简洁高效，能够有效避免各类依赖问题，特别适合在没有网络连接的环境中使用。

## 资源文件描述

- **文件名**: gcc_g++_offline_installer.tar.gz
- **版本**: 4.8.5
- **内置脚本**: `install.sh`

## 使用方法

1. **下载压缩包**: 从本仓库下载`gcc_g++_offline_installer.tar.gz`文件。
2. **解压缩**: 将下载的压缩包解压到目标目录。
3. **运行安装脚本**: 进入解压后的目录，运行`install.sh`脚本。

```bash
tar -xzvf gcc_g++_offline_installer.tar.gz
cd gcc_g++_offline_installer
./install.sh
```

4. **完成安装**: 脚本运行完毕后，GCC与G++将成功安装到系统中。

## 注意事项

- 本安装包适用于无网环境，确保在安装过程中没有网络连接。
- 安装脚本会自动处理依赖关系，无需手动配置。
- 安装完成后，建议重启系统以确保所有环境变量生效。
