# JDK 1.8 下载安装与环境变量配置（Win7）

本文详细介绍了如何在Windows 7系统中下载JDK 1.8，包括从Oracle官网及国内镜像源下载，然后进行安装步骤，并重点讲解了环境变量的配置过程，最后通过命令行验证JDK安装是否成功。

## 1. JDK下载

### 1.1 下载途径
- **Oracle官网**：访问Oracle官网进行下载。
- **国内镜像源**：推荐使用华为镜像源、阿里云镜像源等国内镜像源，下载速度更快。

### 1.2 下载步骤
1. 打开Oracle官网。
2. 点击产品【Products】，选择【Java】。
3. 点击【Download Java】。
4. 找到Java SE 8，点击【JDK Download】。

## 2. JDK安装

### 2.1 安装步骤
1. 双击下载的JDK安装包。
2. 按照安装向导提示，点击“下一步”进行安装。
3. 选择安装路径（默认路径即可），继续点击“下一步”。
4. 安装完成后，点击“关闭”。

## 3. 环境变量配置

### 3.1 配置JAVA_HOME
1. 右键点击“此电脑”，选择“属性”。
2. 点击“高级系统设置”。
3. 在“系统属性”窗口中，点击“环境变量”。
4. 在“系统变量”部分，点击“新建”。
5. 变量名输入`JAVA_HOME`，变量值输入JDK的安装路径（例如：`C:\Program Files\Java\jdk1.8.0_91`）。

### 3.2 配置PATH
1. 在“系统变量”中找到`Path`变量，点击“编辑”。
2. 在变量值的末尾添加`%JAVA_HOME%\bin`，点击“确定”。

## 4. 验证安装

### 4.1 验证步骤
1. 打开命令提示符。
2. 输入`java -version`，回车。
3. 如果显示JDK版本信息，说明安装成功。

## 5. 注意事项

- 下载JDK时，建议使用国内镜像源以提高下载速度。
- 配置环境变量时，确保路径正确无误。
- 安装完成后，务必通过命令行验证JDK是否安装成功。

通过以上步骤，您可以在Windows 7系统中成功下载、安装并配置JDK 1.8，为Java开发环境打下坚实基础。
