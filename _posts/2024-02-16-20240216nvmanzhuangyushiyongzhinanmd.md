# nvm 安装与使用指南

本资源文件提供了关于如何安装和使用 nvm（Node Version Manager）的详细指南，帮助用户实现 Node.js 版本的切换，并解决安装 Node.js 后找不到 npm 的问题。

## 内容概述

1. **nvm 简介**  
   nvm 是一个用于管理 Node.js 版本的工具，允许用户在同一台机器上安装和切换不同版本的 Node.js。

2. **nvm 下载与安装**  
   提供了 nvm 的下载链接和安装步骤，包括安装前的环境清理和安装过程中的注意事项。

3. **nvm 常用命令**  
   列出了 nvm 的常用命令，如查看 nvm 版本、列出所有已安装的 Node.js 版本、安装指定版本的 Node.js 等。

4. **解决找不到 npm 的问题**  
   针对安装 Node.js 后找不到 npm 的问题，提供了手动下载和安装 npm 的解决方案。

5. **附：nvm 安装包**  
   提供了 nvm 的安装包，方便用户快速安装和使用。

## 使用说明

1. **下载 nvm**  
   根据提供的链接下载 nvm 安装包，并按照指南进行安装。

2. **安装 Node.js**  
   使用 nvm 安装所需的 Node.js 版本，并确保 npm 也一并安装。

3. **切换 Node.js 版本**  
   使用 nvm 命令切换不同的 Node.js 版本，以满足不同项目的需求。

4. **解决 npm 问题**  
   如果安装 Node.js 后找不到 npm，请参考指南中的解决方案进行手动安装。

## 注意事项

- 安装前请确保已清理电脑上的 Node.js 环境。
- 安装过程中请注意路径选择，尽量避免安装到 C 盘。
- 如果遇到 npm 相关问题，请参考指南中的详细步骤进行解决。

通过本指南，您将能够轻松管理多个 Node.js 版本，并解决常见的 npm 问题。
