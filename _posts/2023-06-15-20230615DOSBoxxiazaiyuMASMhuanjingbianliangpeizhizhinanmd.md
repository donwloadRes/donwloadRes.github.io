# DOSBox 下载与 MASM 环境变量配置指南

本资源文件提供了详细的步骤，帮助用户下载并配置 DOSBox 以及 MASM 环境变量，以便在现代操作系统上运行老式的 MS-DOS 程序和汇编语言。

1. **DOSBox 和 MASM 的下载**：
   - 提供了 DOSBox 和 MASM 的下载链接及密码。
   - 详细说明了如何安装 DOSBox 并将 MASM 相关文件放置在指定文件夹中。

2. **配置 MASM 环境变量**：
   - 指导用户如何通过系统属性配置 MASM 的环境变量。
   - 提供了具体的步骤，包括编辑系统变量中的 PATH 变量，将 MASM 的路径添加进去。

3. **DOSBox 配置文件的修改**：
   - 解释了如何找到并修改 DOSBox 的配置文件 "DOSBox 0.74 Options.bat"。
   - 详细说明了在配置文件中添加挂载命令和设置环境变量的步骤。

4. **验证配置**：
   - 提供了验证配置是否成功的步骤，包括在 DOSBox 中输入 MASM、DEBUG 和 LINK 命令，确保它们能够正常运行。

## 使用说明

1. **下载与安装**：
   - 下载 DOSBox 和 MASM 相关文件。
   - 安装 DOSBox，并将 MASM 相关文件放置在指定文件夹中。

2. **配置环境变量**：
   - 右击计算机，选择属性，进入高级系统设置。
   - 在系统变量中找到 PATH 变量，编辑并添加 MASM 的路径。

3. **修改 DOSBox 配置文件**：
   - 找到 DOSBox 的安装目录，打开 "DOSBox 0.74 Options.bat" 文件。
   - 在 [autoexec] 部分添加挂载命令和环境变量设置。

4. **验证配置**：
   - 启动 DOSBox，输入 MASM、DEBUG 和 LINK 命令，确保它们能够正常运行。

通过以上步骤，用户可以在现代操作系统上成功配置 DOSBox 和 MASM 环境变量，顺利运行老式的 MS-DOS 程序和汇编语言。
