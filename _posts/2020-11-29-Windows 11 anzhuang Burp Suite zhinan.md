# Windows 11 安装 Burp Suite 指南

本资源文件提供了在 Windows 11 操作系统上安装 Burp Suite 的详细步骤和指南。Burp Suite 是一款广泛使用的网络安全工具，主要用于渗透测试和网络安全评估。通过本指南，您将能够顺利地在 Windows 11 系统上安装和配置 Burp Suite。

## 内容概述

1. **安装 JDK**
   - 下载并解压 JDK 文件。
   - 配置系统环境变量。
   - 验证 JDK 安装是否成功。

2. **安装 Burp Suite**
   - 下载并解压 Burp Suite 文件。
   - 配置 Burp Suite 启动脚本。
   - 启动 Burp Suite 并进行初始设置。

3. **配置代理和证书**
   - 配置浏览器代理设置。
   - 下载并安装 Burp Suite 证书。
   - 验证代理和证书配置是否成功。

## 使用说明

1. **下载资源文件**
   - 下载本仓库中的资源文件，包括 JDK 和 Burp Suite 的安装包。

2. **按照指南操作**
   - 按照上述内容概述中的步骤，逐步完成 Burp Suite 的安装和配置。

3. **验证安装**
   - 启动 Burp Suite 并进行简单的测试，确保安装和配置无误。

## 注意事项

- 在配置环境变量时，请确保路径设置正确。
- 在安装 Burp Suite 时，请注意选择正确的 JDK 路径。
- 在配置代理和证书时，请确保浏览器设置正确，避免网络连接问题。

通过本指南，您将能够顺利地在 Windows 11 系统上安装和使用 Burp Suite，为网络安全测试提供有力支持。
