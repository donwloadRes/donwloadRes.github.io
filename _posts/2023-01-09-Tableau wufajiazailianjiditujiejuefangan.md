# Tableau 无法加载联机地图解决方案

## 简介
本仓库提供了一个名为 `AutoNavi.rar` 的资源文件，该文件包含了用于 Tableau 的高德地图数据源。通过使用这个数据源，您可以在 Tableau 中轻松地集成和可视化高德地图的相关数据，从而为您的分析和报告增添地理信息支持。

## 文件说明
- **文件名**: `AutoNavi.rar`
- **内容**: 包含高德地图数据源，适用于 Tableau 中的地图加载问题。

## 使用方法
1. **下载文件**: 下载 `AutoNavi.rar` 文件并解压缩。
2. **导入 Tableau**: 将解压后的文件导入到 Tableau 的 `Mapsources` 文件夹中。
3. **设置地图背景**: 在 Tableau 中，选择 `地图` -> `背景地图` -> `AutoNavi`，即可使用高德地图作为背景地图。

## 解决的问题
在 Tableau 中使用地理位置数据时，可能会遇到无法加载联机地图的问题。本资源文件提供了一个离线地图源，帮助用户在没有网络连接的情况下也能正常显示地图。

## 注意事项
- 确保您的 Tableau 版本支持自定义地图源。
- 如果仍然无法加载地图，请检查网络连接或更新 Tableau 软件版本。
