# CoFlash - VSCode上的STM32烧录工具

## 简介
CoFlash 是一个专为 VSCode 设计的 STM32 烧录工具，旨在提供简单易用的烧录体验。无论你是 STM32 的初学者还是经验丰富的开发者，CoFlash 都能帮助你快速完成烧录任务。

## 特点
- **简单易用**：CoFlash 的设计理念是让烧录过程尽可能简单，用户无需复杂的配置即可开始使用。
- **集成于 VSCode**：直接在 VSCode 中使用，无需切换到其他工具，提高开发效率。
- **支持多种 STM32 系列**：CoFlash 支持广泛的 STM32 系列，满足不同用户的需求。

## 安装与使用
1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/CoFlash.git
   ```
2. **打开 VSCode**：
   在 VSCode 中打开克隆下来的项目文件夹。
3. **配置烧录工具**：
   根据项目文档进行必要的配置，确保烧录工具能够正确识别你的 STM32 设备。
4. **开始烧录**：
   使用提供的命令或快捷键启动烧录过程。
