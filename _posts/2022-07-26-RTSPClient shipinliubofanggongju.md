# RTSPClient 视频流播放工具

## 简介

`RTSPClient.tar.gz` 是一个基于 Linux 平台和 QT 框架开发的视频流播放工具。该工具利用 FFmpeg 库实现了对 RTSP 视频流的播放功能，适用于需要在 Linux 环境下进行视频流实时播放的场景。

## 功能特点

- **跨平台支持**：基于 Linux 系统开发，适用于大多数 Linux 发行版。
- **QT 界面**：使用 QT 框架构建用户界面，提供友好的操作体验。
- **FFmpeg 集成**：利用 FFmpeg 库处理视频流，支持多种视频格式和编码。
- **RTSP 协议**：支持通过 RTSP 协议获取视频流，适用于监控、直播等场景。

## 使用说明

1. **下载资源文件**：
   - 下载 `RTSPClient.tar.gz` 文件到本地。

2. **解压缩文件**：
   - 使用以下命令解压缩文件：
     ```bash
     tar -xzvf RTSPClient.tar.gz
     ```

3. **编译与运行**：
   - 进入解压后的目录，按照目录中的 `README` 文件或 `INSTALL` 文件的说明进行编译和运行。

4. **配置与使用**：
   - 根据需要配置 RTSP 视频流的地址和其他参数。
   - 启动程序后，即可在 QT 界面中播放视频流。

## 依赖项

- **Linux 操作系统**
- **QT 开发环境**
- **FFmpeg 库**

## 注意事项

- 确保系统中已安装 QT 和 FFmpeg 库，否则可能无法正常编译和运行。
- 在使用过程中，请确保网络连接正常，以便能够顺利获取 RTSP 视频流。
