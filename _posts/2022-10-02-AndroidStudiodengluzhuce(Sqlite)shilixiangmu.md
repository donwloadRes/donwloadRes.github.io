# AndroidStudio登录注册(Sqlite)示例项目

## 项目简介

本项目是一个基于AndroidStudio开发的登录注册示例应用，使用SQLite作为本地数据库来存储用户信息。通过本项目，您可以学习如何在Android应用中实现用户登录和注册功能，并了解SQLite数据库的基本操作。

## 功能特点

- **用户注册**：用户可以通过输入用户名和密码进行注册，注册信息将存储在SQLite数据库中。
- **用户登录**：已注册用户可以通过输入正确的用户名和密码进行登录。
- **SQLite数据库**：使用SQLite数据库来存储和管理用户信息，确保数据的安全性和持久性。

## 使用说明

1. **克隆项目**：
   ```bash
   git clone https://github.com/yourusername/AndroidStudioLoginRegister.git
   ```

2. **导入项目**：
   打开AndroidStudio，选择“File” -> “Open”，然后选择克隆的项目目录。

3. **运行项目**：
   连接Android设备或启动模拟器，点击AndroidStudio中的“Run”按钮，即可在设备上运行应用。

4. **注册与登录**：
   - 首次使用时，请先进行注册。
   - 注册成功后，可以使用注册的用户名和密码进行登录。

## 依赖库

本项目使用了以下依赖库：
- AndroidX库：用于支持最新的Android组件和功能。
- SQLiteOpenHelper：用于管理SQLite数据库的创建和版本控制。
