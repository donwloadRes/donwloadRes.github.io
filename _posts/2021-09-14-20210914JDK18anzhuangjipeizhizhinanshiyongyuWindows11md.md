# JDK 1.8 安装及配置指南 (适用于 Windows 11)

本仓库提供了一个详细的指南，帮助你在 Windows 11 系统上安装和配置 JDK 1.8。无论你是 Java 开发新手还是有经验的开发者，这份指南都将为你提供清晰的步骤和必要的配置说明。

## 内容概述

1. **下载安装包**
   - 从 Oracle 官网下载 JDK 1.8 安装包。
   - 提供了百度网盘的下载链接（提取码：q94n）。

2. **安装 JDK**
   - 详细步骤指导你如何安装 JDK 1.8。
   - 包括选择安装路径和配置 JRE 路径。

3. **配置 JDK 环境变量**
   - 如何在 Windows 11 系统中配置 JDK 环境变量。
   - 包括设置 `JAVA_HOME` 和 `PATH` 变量。

4. **验证配置是否成功**
   - 通过命令行工具验证 JDK 1.8 是否成功安装和配置。

## 使用说明

1. **下载安装包**
   - 访问 Oracle 官网或使用提供的百度网盘链接下载 JDK 1.8 安装包。

2. **安装 JDK**
   - 双击下载的安装包，按照提示完成安装。
   - 在安装过程中，可以选择自定义安装路径。

3. **配置环境变量**
   - 打开系统环境变量配置界面。
   - 新建 `JAVA_HOME` 变量，并设置其值为 JDK 的安装路径。
   - 编辑 `PATH` 变量，添加 `%JAVA_HOME%\bin`。

4. **验证配置**
   - 打开命令提示符，输入 `java -version` 和 `javac -version` 命令。
   - 如果显示 JDK 1.8 的版本信息，则说明配置成功。

## 注意事项

- 确保在配置环境变量时，所有窗口的“确定”按钮都要点击，否则配置可能不成功。
- 如果在安装过程中遇到问题，可以参考 CSDN 博客文章中的详细步骤和截图。

通过本指南，你可以轻松地在 Windows 11 系统上安装和配置 JDK 1.8，为你的 Java 开发环境打下坚实的基础。
