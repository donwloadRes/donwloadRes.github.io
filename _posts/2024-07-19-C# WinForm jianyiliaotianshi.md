# C# WinForm 简易聊天室

## 简介

本项目是一个使用C# WinForm和Socket通信技术搭建的简易聊天室。通过这个项目，你可以学习如何在C# WinForm应用程序中实现基于Socket的客户端-服务器通信，从而构建一个简单的聊天室功能。

## 功能特点

- **Socket通信**：使用Socket进行客户端和服务器之间的通信。
- **简易聊天室**：实现基本的聊天功能，包括发送和接收消息。
- **WinForm界面**：使用C# WinForm构建用户界面，提供友好的聊天体验。

## 如何使用

1. **克隆仓库**：
   ```sh
   git clone https://github.com/your-repo-url.git
   ```

2. **打开项目**：
   使用Visual Studio或其他C#开发环境打开项目文件。

3. **运行服务器**：
   在解决方案中找到服务器项目，编译并运行服务器程序。

4. **运行客户端**：
   在解决方案中找到客户端项目，编译并运行客户端程序。

5. **开始聊天**：
   在客户端程序中输入服务器地址和端口，连接到服务器后即可开始聊天。
