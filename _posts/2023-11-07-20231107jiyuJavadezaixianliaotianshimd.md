# 基于Java的在线聊天室

## 项目介绍

本项目是一个基于Java和JSP技术开发的在线聊天室应用。用户可以通过注册和登录功能进入聊天界面，与其他用户进行实时聊天。聊天内容使用Session进行保存，而用户注册信息则存储在MySQL数据库中。

## 功能特点

- **注册与登录**：用户可以注册新账号并登录到聊天室。
- **实时聊天**：注册成功后，用户可以进入聊天界面与其他用户进行实时交流。
- **数据存储**：聊天内容通过Session保存，用户注册信息则存储在MySQL数据库中。

## 技术栈

- **Java**：后端开发语言。
- **JSP**：用于构建动态网页。
- **MySQL**：数据库管理系统，用于存储用户注册信息。

## 安装与运行

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/online-chatroom.git
   ```

2. **配置数据库**：
   - 创建一个MySQL数据库。
   - 导入项目中的数据库脚本。

3. **配置项目**：
   - 修改数据库连接配置，确保能够连接到你的MySQL数据库。

4. **运行项目**：
   - 部署项目到支持Java和JSP的服务器（如Tomcat）。
   - 启动服务器并访问项目URL。
