# Qt 系统监控工具：磁盘空间、CPU 利用率、内存使用率

## 简介

本项目提供了一个基于 Qt 的系统监控工具，能够实时获取并显示磁盘空间大小、CPU 利用率以及内存使用率。该工具已经封装成一个类，可以直接调用，已经在多个商业项目中得到验证和使用。

## 功能特点

1. **全平台支持**：
   - Windows
   - Linux
   - ARM Linux

2. **功能全面且稳定**：
   - 磁盘空间大小获取：快速计算硬盘容量，支持进度条显示占用比例。
   - CPU 利用率获取：支持多核 CPU，计算结果非常准确。
   - 内存使用率获取：实时显示内存使用情况。

3. **用户界面友好**：
   - 多彩数码管实时显示当前时间。
   - 自定义颜色下拉框，选择即可看到效果。

## 使用方法

1. **集成到项目中**：
   - 将封装好的类文件添加到你的 Qt 项目中。
   - 直接调用类中的方法即可获取系统监控数据。

2. **示例代码**：
   ```cpp
   // 示例代码，具体调用方法请参考类文件中的注释
   SystemMonitor monitor;
   double diskSpace = monitor.getDiskSpace();
   double cpuUsage = monitor.getCpuUsage();
   double memoryUsage = monitor.getMemoryUsage();
   ```

## 注意事项

- 本工具已经在多个商业项目中使用，稳定性有保障。
- 在 Linux 平台下，CPU 占用率的计算非常准确，支持多核处理器的监控。
- 硬盘容量的计算速度极快，适合实时监控场景。
