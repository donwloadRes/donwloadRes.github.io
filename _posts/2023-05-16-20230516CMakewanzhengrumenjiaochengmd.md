# CMake 完整入门教程

## 简介

CMake 是一个跨平台的安装(编译)工具，它可以用简单的语句来描述所有平台的安装(编译过程)。尽管网络上关于 CMake 的教程很多，但大多数教程都比较零散，难以找到一个完整、详细的中文版教程。为了帮助大家更好地理解和使用 CMake，我收集了网络上的相关信息并进行了整理，最终形成了这份文档。

## 内容概述

这份文档涵盖了 CMake 的基础知识、常用命令、项目配置、跨平台编译等方面的内容。无论你是 CMake 的初学者，还是有一定经验的开发者，这份文档都能为你提供有价值的参考。

## 适用人群

- 对 CMake 感兴趣的初学者
- 希望深入了解 CMake 的开发者
- 需要在不同平台上进行项目编译的开发者
