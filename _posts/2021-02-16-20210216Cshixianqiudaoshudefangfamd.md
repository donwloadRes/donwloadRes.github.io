## 示例代码
以下是求倒数的C++示例代码：

```cpp
#include <iostream>

int main() {
    double number;
    std::cout << "请输入一个非零数字: ";
    std::cin >> number;

    if (number != 0) {
        double reciprocal = 1 / number;
        std::cout << "该数字的倒数是: " << reciprocal << std::endl;
    } else {
        std::cout << "输入的数字不能为零，无法求倒数。" << std::endl;
    }

    return 0;
}
```
