# JDK 1.8 中文 API 文档

本仓库提供 JDK 1.8 的中文 API 文档，方便开发者在使用 Java 1.8 进行开发时查阅相关类和方法的详细说明。

## 资源文件
- **文件名**: `jdk-1.8-api-zh_CN.zip`
- **描述**: JDK 1.8 中文 API 文档

## 使用方法
1. 下载本仓库中的 `jdk-1.8-api-zh_CN.zip` 文件。
2. 解压缩文件，得到 HTML 格式的 API 文档。
3. 打开 `index.html` 文件，即可开始查阅 JDK 1.8 的中文 API 文档。
