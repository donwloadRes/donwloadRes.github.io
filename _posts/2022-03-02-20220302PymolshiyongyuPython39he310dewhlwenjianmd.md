# Pymol 适用于 Python 3.9 和 3.10 的 whl 文件

## 简介
本仓库提供适用于 Python 3.9 和 3.10 的 Pymol whl 文件下载。Pymol 是一款强大的分子可视化软件，广泛应用于生物化学、结构生物学、药物设计等领域。通过本仓库提供的 whl 文件，用户可以方便地在 Python 3.9 和 3.10 环境中安装和使用 Pymol。

## 安装步骤
1. 打开 Anaconda 的 PowerShell。
2. 使用 `pip install pmw` 安装依赖。
3. 使用 `pip install + whl 文件路径` 安装 Pymol。
4. 在 Anaconda 的 Scripts 文件夹中找到 pymol 并启动使用。

## 注意事项
- 确保已安装 Python 3.9 或 3.10 环境。
- 安装过程中可能需要其他依赖，请根据提示进行安装。
