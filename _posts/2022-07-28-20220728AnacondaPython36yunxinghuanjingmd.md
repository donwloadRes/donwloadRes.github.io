## 描述

## 适用场景
- 需要使用 Anaconda 4.8 和 Python 3.6 环境的开发者。
- 遇到 `from PyQt5 import QtCore` 导入错误的问题，且其他解决方案无效的用户。

## 注意事项
- 该环境仅适用于 Anaconda 4.8 和 Python 3.6，其他版本可能会出现不兼容的情况。
- 请在下载和使用前确保你的系统环境与该压缩包中的环境匹配。

## 使用方法
1. 下载并解压压缩包。
2. 按照解压后的文件夹中的说明进行环境配置。
3. 运行你的 Python 脚本，确保 `from PyQt5 import QtCore` 不再出现错误。
