# 交换机路由器批量脚本生成工具

## 简介

本仓库提供了一个用于生成交换机和路由器批量配置脚本的工具。通过该工具，您可以轻松生成适用于大规模网络设备的配置脚本，从而简化网络管理任务。

## 功能特点

- **批量生成脚本**：支持批量生成配置脚本，适用于大规模网络设备的配置。
- **自定义变量**：脚本支持定义无限变量，可根据实际需求灵活调整。
- **易于使用**：只需安装Python 3.0+和xlrd库，即可快速上手使用。

## 使用方法

1. **安装Python 3.0+**：
   确保您的系统已安装Python 3.0或更高版本。

2. **安装xlrd库**：
   通过pip安装xlrd库，命令如下：
   ```bash
   pip install xlrd
   ```

3. **修改脚本文件**：
   根据您的实际需求，修改脚本文件中的变量和配置内容。

4. **运行脚本**：
   运行脚本生成配置文件，并将其应用于您的网络设备。

## 注意事项

- 请确保在修改脚本文件时，仔细检查变量和配置内容，以避免错误配置。
- 建议在测试环境中先进行脚本测试，确保生成的配置文件符合预期。
