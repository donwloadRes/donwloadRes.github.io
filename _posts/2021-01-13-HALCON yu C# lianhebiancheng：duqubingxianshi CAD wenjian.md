# HALCON 与 C# 联合编程：读取并显示 CAD 文件

本资源文件提供了一个示例项目，展示了如何使用 HALCON 与 C# 联合编程，实现读取并显示 CAD 文件的功能。通过本项目，您可以学习到如何在 C# 环境中调用 HALCON 的图像处理功能，并将其应用于实际的 CAD 文件处理中。

## 资源内容

- **HALCON 与 C# 联合编程示例代码**：包含了一个完整的 C# 项目，展示了如何使用 HALCON 读取并显示 CAD 文件。
- **测试 CAD 文件**：项目目录下提供了一个 CAD 文件，供您进行测试和练习。

## 使用说明

1. **下载资源文件**：请将整个文件夹下载到您的本地计算机。
2. **配置 HALCON 引用**：如果项目中的 `halcondotnet.dll` 引用不正确，请手动添加您电脑中相应的 HALCON 引用。
3. **运行项目**：将下载的文件夹放置在桌面上，然后打开 C# 项目并运行。由于 CAD 文件使用的是相对路径，因此必须将文件夹放在桌面上才能正常运行。

## 注意事项

- **路径问题**：请确保将文件夹放置在桌面上，否则 CAD 文件可能无法正确读取。
- **HALCON 引用**：如果遇到 `halcondotnet.dll` 引用问题，请根据您的 HALCON 安装路径进行手动配置。
