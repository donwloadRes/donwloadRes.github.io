# QT for Android环境搭建与配置

## 简介
本资源文件提供了详细的步骤和指南，帮助开发者搭建和配置QT for Android的开发环境。通过本指南，您将能够顺利地在Windows平台上配置QT for Android所需的JDK、SDK、NDK、ANT等工具，并成功运行您的第一个QT for Android应用程序。

## 主要内容
1. **环境要求**：
   - Windows 7 x64或更高版本
   - QT 5.12.0或更高版本
   - JDK 1.8.0_241
   - Android SDK
   - Android NDK r21e
   - Apache Ant 1.10.5

2. **搭建步骤**：
   - JDK的安装与环境变量配置
   - Android SDK的安装与配置
   - Android NDK的安装
   - Apache Ant的安装与环境变量配置
   - QT 5.12.0的安装
   - QT中配置Android环境

3. **测试与运行**：
   - 新建QT工程并进行测试
   - 在Android设备上运行QT应用程序

## 注意事项
- 所有工具的安装路径中不能包含中文和空格，否则可能导致配置失败。
- 在配置过程中，确保各个组件的版本高度契合，避免因版本不匹配导致的错误。
