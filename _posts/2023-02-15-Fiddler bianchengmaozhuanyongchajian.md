# Fiddler 编程猫专用插件

## 简介

本仓库提供了一个专为编程猫设计的 Fiddler 插件，旨在帮助开发者更方便地进行各种 hook 注入、JS 调试、数据解码等操作。该插件能够显著提升开发效率，适用于需要对编程猫平台进行深入调试和分析的开发者。

## 功能特点

- **Hook 注入**：支持在编程猫平台中进行各种 hook 操作，方便开发者对特定功能进行拦截和修改。
- **JS 调试**：提供强大的 JavaScript 调试功能，帮助开发者快速定位和解决代码中的问题。
- **数据解码**：支持对编程猫平台中的数据进行解码，便于开发者分析和处理数据。

## 使用方法

1. **下载插件**：从本仓库下载最新的 Fiddler 编程猫专用插件。
2. **安装插件**：将下载的插件文件导入到 Fiddler 中。
3. **启动 Fiddler**：打开 Fiddler，确保插件已成功加载。
4. **开始调试**：在 Fiddler 中进行各种 hook 注入、JS 调试和数据解码操作。

## 注意事项

- 请确保您已安装并配置好 Fiddler 工具。
- 使用插件时，请遵守相关法律法规和编程猫的使用条款。
