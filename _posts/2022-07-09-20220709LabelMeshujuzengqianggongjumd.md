# LabelMe数据增强工具

## 概述

本仓库提供了一套高效的数据增强解决方案，专为使用LabelMe标注的图像设计。通过简单的步骤，您可以在已有的一张标记好的图像基础上，自动生成包含多种变换效果（如翻转、加噪、模糊、调整曝光等）的图像及相应的JSON标签文件。这套方案极大地扩展了训练数据集的多样性，从而提升深度学习模型，如Mask R-CNN、Faster R-CNN等，在目标检测与分割任务中的性能。

## 功能特点

- **单一到多变**：只需提供一张由LabelMe工具完成标注的图片（.png + .json），即可批量生成20张经过不同数据增强处理的新图像及其标签。
- **丰富变换**：支持包括水平/垂直翻转、随机噪声添加、图像模糊、曝光度调整等多种常见的数据增强技术。
- **无缝对接COCO格式**：提供脚本将增强后的数据集转换成COCO数据集格式，便于直接用于主流的目标检测与实例分割框架。
- **简易操作**：整个流程基于Python脚本执行，无需复杂的配置，适用于快速扩增训练数据集。

## 使用步骤

### 准备工作

确保您的环境中已安装Python以及相关依赖库，如`Pillow`, `numpy`, `labelme`等。

1. **下载仓库**：首先，克隆或下载此仓库到本地。
   
2. **安装依赖**：如果需要，可以通过pip安装必要的库，例如：
   ```bash
   pip install -r requirements.txt
   ```

### 数据增强

1. **放置初始标注文件**：确保你的原始`.png`和`.json`文件位于合适的位置。
   
2. **运行数据增强脚本**：
   在终端中，定位到脚本所在目录并执行以下命令来生成增强图像及标签：
   ```bash
   python a.py
   ```
   这将会根据初始标签生成20张增强后的图像及对应JSON文件。

### 转换为COCO格式

如果您希望将这些增强后的数据集转换为COCO格式，以便于在相应框架下使用，执行以下命令：
```bash
python labelme2COCO.py
```
这一步骤会生成COCO标准的数据结构，包括images、annotations和categories等文件夹或文件。

## 注意事项

- 确保在执行脚本前备份原始数据，以防意外覆盖。
- 根据实际需求，您可能需要对脚本进行适当的修改以适应特定的项目设置。

通过以上步骤，您可以有效扩展和优化您的机器学习模型训练数据，提高模型的泛化能力。希望这个工具能为您的计算机视觉项目带来便利。
