# 数据可视化：基于 Echarts + Python Flask 实时大屏监控解决方案

## 项目简介

本项目是一个专门针对银行监管系统的数据可视化应用示例。通过结合强大的Python Flask微框架与丰富的Echarts前端图表库，实现了动态、实时的数据大屏幕展示。它非常适合用于银行内部的运营监控、风险管理以及数据分析等场景，提供直观、高效的视觉化界面，帮助决策者快速掌握关键业务指标。

## 技术栈

- **Python**: 后端开发语言，使用Flask作为Web服务器框架。
- **Flask**: 轻量级的Web服务软件，简单易用，适合快速构建应用。
- **Echarts**: 开源JavaScript可视化库，提供了丰富的图表类型，支持高度的定制及交互式操作。
- **HTML/CSS/JavaScript**: 前端展示的基础技术，结合Echarts实现动态效果。

## 快速启动

1. **环境准备**：确保你的开发环境中已安装Python 3.x版本。
2. **安装依赖**：在项目根目录下运行以下命令来安装所需的Python库：
   ```bash
   pip install -r requirements.txt
   ```
3. **运行服务器**：
   执行`main.py`文件以启动Flask应用：
   ```bash
   python main.py
   ```
4. **访问大屏**：浏览器中输入地址即可查看数据可视化大屏：
   ```plaintext
   http://localhost:88/static/xxx/index.html
   ```
   注意这里的端口号(`88`)应与`main.py`配置中的`port`参数保持一致。

## 使用注意事项

- 在首次部署或更新数据源时，请参照文档中关于数据接口和Echarts配置的部分进行适当调整。
- 主题切换功能可通过前端代码实现，具体实现方法可在项目的前端代码注释中找到说明。
- 确保Flask应用运行无误，并且静态资源路径正确设置，以便浏览器能够正确加载Echarts图表。
  
## 特性亮点

- **动态实时**：设计用于实时更新数据，适用于需要持续监控的金融环境。
