# DOSBox Debug 调试教程资源文件介绍

本资源文件提供了使用 DOSBox 进行 Debug 调试的详细教程，适用于需要学习和实践汇编语言的用户。通过本教程，您将能够掌握在 DOSBox 环境中进行汇编程序的调试方法。

## 内容概述

1. **DOSBox 安装与配置**  
   详细介绍了如何下载和安装 DOSBox，并进行必要的配置，以便在现代操作系统上模拟 DOS 环境。

2. **MASM 工具的安装与使用**  
   提供了 MASM 工具的下载链接，并指导用户如何在 DOSBox 中安装和使用 MASM 进行汇编程序的编译。

   提供了一个简单的加法运算汇编源程序示例，帮助用户理解如何编写和保存汇编代码。

4. **DOSBox 中的 Debug 调试**  
   详细讲解了如何在 DOSBox 中运行 Debug 工具，并使用 Debug 进行汇编程序的调试。包括常用 Debug 命令的使用方法。

## 使用说明

1. **下载与安装**  
   按照教程中的步骤下载并安装 DOSBox 和 MASM 工具。

2. **配置 DOSBox**  
   根据教程配置 DOSBox，确保能够正确挂载和访问您的文件系统。

3. **编写汇编程序**  
   使用提供的示例代码或自行编写汇编程序，并保存为 `.asm` 文件。
