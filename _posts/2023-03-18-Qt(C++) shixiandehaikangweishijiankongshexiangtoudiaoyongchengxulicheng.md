# Qt(C++) 实现的海康威视监控摄像头调用程序例程

## 简介

本仓库提供了一个使用 Qt(C++) 实现的海康威视监控摄像头调用程序例程。该例程展示了如何通过 Qt 框架调用海康威视的摄像头，并实现基本的监控功能。

## 功能特点

- 使用 Qt(C++) 开发，跨平台性强。
- 支持海康威视监控摄像头的基本调用。
- 提供了简单的用户界面，方便用户操作。

## 使用说明

1. **克隆仓库**
    ```bash
    git clone https://github.com/your-repo-url.git
    ```

2. **安装依赖**
    确保你已经安装了 Qt 开发环境以及海康威视的 SDK。

3. **编译运行**
    进入项目目录，使用 Qt Creator 打开项目文件，编译并运行程序。

4. **配置摄像头**
    根据海康威视的文档，配置摄像头的 IP 地址、用户名和密码等信息。
