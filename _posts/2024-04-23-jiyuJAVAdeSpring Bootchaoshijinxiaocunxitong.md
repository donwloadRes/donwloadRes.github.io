# 基于JAVA的Spring Boot超市进销存系统

## 项目简介

本项目是一个基于Spring Boot框架的超市进销存管理系统，采用B/S模式，后台数据库使用MySQL，服务器使用Tomcat。系统主要功能包括首页、个人中心、员工管理、客户管理、供应商管理、承运商管理、仓库信息管理、商品类别管理、商品信息管理、采购信息管理、入库信息管理、出库信息管理、销售订单管理、部门管理、岗位管理等。通过这些功能的实现，系统能够满足日常超市进销存管理的操作需求，并为管理员提供高效、安全的数据管理功能。

## 功能模块

1. **首页**：系统的主界面，提供导航和快速访问功能。
2. **个人中心**：用户个人信息管理，包括修改密码、个人信息查看等。
3. **员工管理**：管理超市员工信息，包括添加、删除、修改员工信息。
4. **客户管理**：管理客户信息，包括客户信息的添加、删除、修改。
5. **供应商管理**：管理供应商信息，包括供应商信息的添加、删除、修改。
6. **承运商管理**：管理承运商信息，包括承运商信息的添加、删除、修改。
7. **仓库信息管理**：管理仓库信息，包括仓库信息的添加、删除、修改。
8. **商品类别管理**：管理商品类别信息，包括商品类别的添加、删除、修改。
9. **商品信息管理**：管理商品信息，包括商品信息的添加、删除、修改。
10. **采购信息管理**：管理采购信息，包括采购订单的添加、删除、修改。
11. **入库信息管理**：管理商品入库信息，包括入库记录的添加、删除、修改。
12. **出库信息管理**：管理商品出库信息，包括出库记录的添加、删除、修改。
13. **销售订单管理**：管理销售订单信息，包括销售订单的添加、删除、修改。
14. **部门管理**：管理部门信息，包括部门的添加、删除、修改。
15. **岗位管理**：管理岗位信息，包括岗位的添加、删除、修改。

## 技术栈

- **框架**：Spring Boot
- **数据库**：MySQL
- **服务器**：Tomcat
- **前端**：HTML, CSS, JavaScript
- **后端**：Java

## 安装与运行

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   ```

2. **导入项目**：
   使用IDE（如IntelliJ IDEA或Eclipse）导入项目。

3. **配置数据库**：
   在`application.properties`文件中配置MySQL数据库连接信息。

4. **运行项目**：
   启动Tomcat服务器，运行Spring Boot应用程序。

5. **访问系统**：
   打开浏览器，访问`http://localhost:8080`。

## 注意事项

- 系统对用户输入的任何信息都进行了一定的验证，确保数据的安全性和有效性。
- 管理员可以通过系统对各项功能进行管理，提高操作效率。
