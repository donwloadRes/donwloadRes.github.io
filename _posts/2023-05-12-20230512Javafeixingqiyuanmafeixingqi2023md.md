# Java飞行棋源码【飞行棋2023】

## 项目简介

本仓库提供了一个完整的Java飞行棋游戏源码，名为“飞行棋2023”。该源码实现了经典的飞行棋游戏逻辑，适合Java初学者学习和参考，也适合开发者进行二次开发和扩展。

## 功能特点

- **经典游戏逻辑**：实现了飞行棋的基本规则，包括掷骰子、棋子移动、碰撞、胜利判定等。
- **图形化界面**：使用Java Swing库实现了简单的图形化界面，方便玩家进行游戏。
- **多人对战**：支持2-4人同时进行游戏，玩家可以通过本地或网络进行对战。
- **可扩展性**：源码结构清晰，易于理解和修改，适合开发者进行功能扩展和优化。

## 使用说明

1. **环境要求**：
   - Java Development Kit (JDK) 8 或更高版本
   - 任何支持Java的集成开发环境（IDE），如Eclipse、IntelliJ IDEA等

2. **导入项目**：
   - 将本仓库克隆到本地，或下载ZIP文件并解压。
   - 使用IDE打开项目文件夹。

3. **运行游戏**：
   - 在IDE中找到主类文件（通常为`Main.java`或`Game.java`），运行该类即可启动游戏。

4. **游戏操作**：
   - 游戏启动后，按照提示进行操作，如选择玩家人数、掷骰子、移动棋子等。
