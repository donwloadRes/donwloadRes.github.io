# Sublime 的 ConvertToUTF8 插件安装指南

## 简介
ConvertToUTF8 是一个 Sublime Text 插件，用于编辑和保存编码为 GBK、BIG5、EUC-KR、EUC-JP、Shift_JIS 等的文件。通过本插件，您可以轻松解决 Sublime Text 中遇到的乱码问题。

## 安装方法

### 方法一：使用 Package Control 安装
1. 打开 Sublime Text，按下 `Ctrl + Shift + P` 打开命令面板。
2. 输入 `install`，选择 `Package Control: Install Package`。
3. 在搜索框中输入 `UTF8`，找到并安装 `ConvertToUTF8` 插件。
4. 安装完成后，左下角会提示正在安装，稍等片刻即可。

### 方法二：手动安装
1. 下载 ConvertToUTF8 插件并解压，将解压后的文件夹名修改为 `ConvertToUTF8`。
2. 将 `ConvertToUTF8` 文件夹移动到 Sublime Text 的 `Packages` 文件夹下（可通过 `Preferences > Browse Packages` 找到）。
3. 完成之后，按照正常安装的方式进行安装即可。

## 使用说明
安装完成后，Sublime Text 将自动支持编辑和保存 GBK、BIG5 等编码的文件，无需额外配置。

## 注意事项
- 确保 Sublime Text 已安装 Package Control，否则无法使用方法一进行安装。
- 手动安装时，请确保文件夹路径和名称正确，否则插件可能无法正常加载。

通过以上步骤，您可以轻松解决 Sublime Text 中的乱码问题，提升编辑体验。
