# POSIX官方正式标准（内含8个文档）

## 概述

本仓库提供了一套完整的POSIX（Portable Operating System Interface for Unix）官方正式标准文档集，对于系统开发者、程序员以及对Unix/Linux操作系统深入理解有兴趣的人来说，这是一套不可或缺的手册。POSIX是由IEEE制定的一系列标准，旨在确保不同操作系统之间的兼容性和可移植性，特别是针对Unix-like系统。

## 包含文档

这套资源包含了以下八个核心文档，全面覆盖了POSIX的标准规范：

1. **POSIX Base Definitions** - 基础定义，提供了环境和数据类型的标准化定义。
2. **POSIX Base Specifications Issue 7** - 第七版基础规格说明，详细描述了接口的行为和系统服务。
3. **POSIX Rationale (Informative)** - 理由说明（信息性），解释了标准设计背后的理由和决策过程。
4. **POSIX® Realtime and Embedded Application Support** - 实时和嵌入式应用支持，为实时和特殊需求的嵌入式系统定义了额外的接口。
5. **POSIX® Shell and Utilities** - 命令行shell和工具，标准化了shell编程语言及常用命令的行为。
6. **POSIX System Interfaces** - 系统接口，定义了应用程序与操作系统之间交互的接口。
7. **POSIX Technical Corrigendum 1** - 技术勘误表1，修正了前版中的技术错误。
8. **POSIX Technical Corrigendum 2** - 技术勘误表2，进一步的技术修正和更新。
