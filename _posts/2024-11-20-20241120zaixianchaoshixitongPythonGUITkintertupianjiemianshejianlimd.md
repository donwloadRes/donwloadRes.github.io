# 在线超市系统 - Python GUI Tkinter 图片界面设计案例

## 简介

本资源文件提供了一个使用Python和Tkinter库开发的在线超市系统图形用户界面（GUI）设计案例。该系统允许用户在界面上选择商品、确认商品并进行支付操作。通过本案例，您可以学习如何使用Tkinter库创建交互式GUI应用程序，并了解如何在界面中集成图片和按钮功能。

## 功能概述

1. **主界面设计**：
   - 显示超市主界面，包含“在线选购”和“退出系统”按钮。
   - 主界面中展示超市的图片和标题。

2. **商品选择界面**：
   - 用户点击“在线选购”按钮后，进入商品选择界面。
   - 界面中展示多种商品的图片和对应的价格按钮。
   - 用户可以选择商品并点击相应按钮进行确认。

3. **确认选购界面**：
   - 用户确认选择商品后，进入确认选购界面。
   - 界面中显示所选商品的图片和确认购买的提示信息。
   - 提供“二维码支付”和“返回”按钮。

4. **支付界面**：
   - 用户点击“二维码支付”按钮后，弹出支付界面。
   - 界面中显示二维码图片和“完成支付”按钮。

## 开发环境

- **操作系统**：Windows 10
- **Python版本**：Python 3.5.5
- **开发工具**：VS Code

## 使用说明

1. **安装依赖库**：
   - 确保已安装Python和Tkinter库。
   - 如果未安装PIL库，请在命令提示符中运行以下命令进行安装：
     ```
     pip install pillow
     ```

2. **运行程序**：
   - 下载资源文件并解压。
   - 在VS Code或其他Python IDE中打开项目文件夹。
   - 运行主程序文件，启动在线超市系统。

3. **界面操作**：
   - 在主界面点击“在线选购”按钮进入商品选择界面。
   - 选择商品后点击相应按钮进行确认。
   - 在确认选购界面点击“二维码支付”按钮进行支付。

## 注意事项

- 本案例中的图片路径为本地路径，请根据实际情况修改图片路径。
- 支付界面中的二维码为示例图片，实际使用时需替换为真实支付二维码。
