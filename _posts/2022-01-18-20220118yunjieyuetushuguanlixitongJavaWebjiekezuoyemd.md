# 云借阅图书管理系统 - Java Web 结课作业

## 项目简介

本项目是一个基于Java Web的结课作业，名为“云借阅图书管理系统”。该系统旨在提供一个在线的图书借阅平台，方便用户进行图书的借阅、归还、查询等操作。通过该系统，用户可以轻松管理自己的借阅记录，图书馆管理员也可以方便地管理图书库存和借阅情况。

## 功能特点

- **用户管理**：支持用户注册、登录、个人信息管理等功能。
- **图书管理**：管理员可以添加、删除、修改图书信息，用户可以浏览图书目录。
- **借阅管理**：用户可以在线借阅图书，查看借阅记录，管理员可以查看所有借阅情况。
- **归还管理**：用户可以在线归还图书，管理员可以处理归还操作。
- **搜索功能**：支持按书名、作者、分类等条件搜索图书。

## 技术栈

- **前端**：HTML, CSS, JavaScript
- **后端**：Java, Servlet, JSP
- **数据库**：MySQL
- **服务器**：Tomcat

## 使用说明

1. **环境配置**：
   - 确保本地安装了Java开发环境（JDK）。
   - 安装并配置Tomcat服务器。
   - 安装MySQL数据库，并导入项目中的数据库脚本。

2. **项目部署**：
   - 将项目导入到IDE（如Eclipse或IntelliJ IDEA）中。
   - 配置数据库连接信息。
   - 将项目部署到Tomcat服务器。

3. **运行项目**：
   - 启动Tomcat服务器。
   - 在浏览器中访问项目首页。

## 注意事项

- 请确保数据库连接信息正确无误。
- 项目运行前，请先创建数据库并导入相关数据表。
- 如有任何问题，请参考项目中的日志文件进行排查。
