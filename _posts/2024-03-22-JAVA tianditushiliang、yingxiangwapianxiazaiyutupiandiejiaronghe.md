# JAVA 天地图矢量、影像瓦片下载与图片叠加融合

本项目是一个基于Spring Boot和Maven构建的JAVA应用，旨在实现天地图矢量和影像底图瓦片图片的获取、叠加、图片融合拼接等功能。通过该项目，您可以轻松地下载天地图的矢量和影像瓦片，并将它们进行叠加和融合，生成所需的图片。

## 功能特点

- **瓦片下载**：支持从天地图服务器下载矢量和影像底图的瓦片图片。
- **图片叠加**：可以将多个瓦片图片进行叠加，生成复合图片。
- **图片融合**：支持将叠加后的图片进行融合拼接，生成更大范围的地图图片。

## 使用方法

1. **克隆项目**：将本项目克隆到本地。
2. **配置环境**：确保您的开发环境已配置好Java和Maven。
3. **运行测试类**：运行测试类`TianDiTuDownload_ds`，该类包含了瓦片下载、图片叠加和融合的示例代码。
4. **自定义功能**：根据您的需求，修改和扩展代码，实现更多功能。
