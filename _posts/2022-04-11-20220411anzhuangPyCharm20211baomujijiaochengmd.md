# 安装 PyCharm 2021.1 保姆级教程

## 概览

本文档为您提供详尽的指南，以帮助您顺利安装 PyCharm 2021.1 版本，这是一款广受欢迎的 Python 集成开发环境（IDE）。无论您是编程新手还是经验丰富的开发者，这份教程都将引导您完成整个安装过程，并包含必要的激活步骤以及配置中文界面的指导。

### 系统需求

- **Python 版本**：确保您的系统已安装 Python 2.7 或 Python 3.6 至 3.11。
- **PyCharm 版本**：本教程基于 PyCharm 2021.1 版本。

### 下载资源

- 安装文件及辅助材料可通过资源仓库获取，包括 PyCharm 安装包和 Python 安装包。
- 百度网盘提取码：wk9g

### 安装步骤

#### 1. Python 安装
1. 安装 `python-3.8.5-amd64`，勾选“添加到环境变量”，并自定义安装路径。
2. 在命令行输入 `python`，验证安装成功。

#### 2. PyCharm 安装
1. 解压缩 PyCharm 2021.1 安装包，以管理员身份运行安装程序。
2. 完成安装后，使用管理员权限运行 `Block Host [Run Administrator] cmd` 以破解。
3. 将 `BetterIntelliJ-1.20.jar` 复制到 C 盘根目录，不可删除。
4. 把中文包放入 PyCharm 安装目录下的 `lib` 文件夹。
5. 打开 PyCharm，配置插件，导入 `BetterIntelliJ-1.20.jar` 并调整 JVM 设置加入 `-javaagent:C:/BetterIntelliJ-1.20.jar`。
6. 重新启动 PyCharm，完成语言设置为中文，并激活软件使用提供的激活码。

### 注意事项

- 激活过程需谨慎处理，确保遵守软件的许可协议。
- 确保所有的文件放置正确，避免因误操作导致的问题。
- 完成上述步骤后，您将拥有一个完全配置好的中文界面 PyCharm 2021.1 开发环境。

开始您的 Python 编程之旅吧！
