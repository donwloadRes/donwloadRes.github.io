# 解决Solidworks2019许可问题资源文件

## 简介

本资源文件旨在帮助用户解决Solidworks2019在启动时遇到的许可问题，具体表现为“无法获得下列许可，期望的Solidworks监控程序已关闭（-97，121，0）”。该问题通常由于系统文件缺失或被误删导致，通过本资源文件提供的解决方案，用户可以快速恢复Solidworks2019的正常使用。

## 问题描述

在使用Solidworks2019时，部分用户可能会遇到以下错误提示：

```
无法获得下列许可，期望的Solidworks监控程序已关闭（-97，121，0）
```

该错误通常是由于系统中缺少必要的文件（如`sw_d.exe`）或文件被误删导致的。

## 解决方案

1. **检查文件缺失**：
   - 首先，检查Solidworks安装目录下是否存在`sw_d.exe`文件。如果不存在，请继续下一步。

2. **从安装包中恢复文件**：
   - 如果您有Solidworks的安装包，请在其中搜索`sw_d.exe`文件，并将其复制到Solidworks的安装目录下。

3. **使用本资源文件**：
   - 如果您找不到安装包，可以使用本资源文件中提供的`sw_d.exe`文件，将其复制到Solidworks的安装目录下。

4. **重启Solidworks**：
   - 完成文件复制后，重新启动Solidworks，检查问题是否已解决。

- 请确保复制的文件与您的Solidworks版本匹配。
