# MPU-6050 校准与示例资源文件

本仓库提供了一个名为 `mpu6050校准与示例.zip` 的资源文件，该文件包含了 MPU-6050 6轴加速度计/陀螺仪的 Arduino 库及相关示例代码。

## 资源文件描述

MPU-6050 6轴加速度计/陀螺仪的 Arduino 库由 Electronic Cats 改编自 Arduino 库管理器。要安装该库，您可以使用 Arduino 库管理器并搜索“mpu6050”，然后安装该库。

### 手动安装步骤

1. 使用 Arduino 库管理器（“草图” -> “包括库” -> “管理库...”）安装该库。
2. 或者，您可以使用“下载 ZIP”按钮从 GitHub 下载 zip 文件。
3. 然后，使用 IDE（“草图” -> “包含库” -> “添加 .ZIP 库...”）将此 Git 存储库克隆到您的草图簿/库文件夹中。

## 注意事项

该库基于 [jrowberg/i2cdevlib](https://github.com/jrowberg/i2cdevlib/tree/master/Arduino/MPU6050) 的工作。

希望这个资源文件对您的项目有所帮助！
