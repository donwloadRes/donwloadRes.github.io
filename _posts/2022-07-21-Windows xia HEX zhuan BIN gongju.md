# Windows 下 HEX 转 BIN 工具

## 简介

本仓库提供了一个在 Windows 系统下将 HEX 文件转换为 BIN 文件的工具。该工具名为 `hex2bin.exe`，使用简单方便，适用于需要在 Windows 环境下进行 HEX 文件转换的用户。

## 使用说明

1. **下载工具**：
   - 从本仓库下载 `hex2bin.exe` 文件。

2. **使用方法**：
   - 将 `hex2bin.exe` 放置在你需要转换的 HEX 文件所在的目录下。
   - 打开 Windows 的命令提示符（CMD），并导航到该目录。
   - 输入以下命令以查看使用说明：
     ```
     hex2bin.exe
     ```
   - 输入以下命令以将指定的 HEX 文件转换为 BIN 文件：
     ```
     hex2bin.exe 123.hex
     ```
     其中 `123.hex` 是你要转换的 HEX 文件名，转换后会生成一个同名的 BIN 文件 `123.bin`。

## 注意事项

- 确保 `hex2bin.exe` 和你要转换的 HEX 文件在同一个目录下。
- 转换后的 BIN 文件会保存在与 HEX 文件相同的目录下。
