# VUE项目开发：使用jessibuca实现直播流播放器

本仓库提供了一个基于VUE框架开发的直播流播放器项目，使用了jessibuca插件来实现直播流的播放功能。该项目旨在帮助开发者快速集成直播流播放功能到VUE项目中。

## 项目简介

该项目通过引入jessibuca插件，实现了在VUE项目中播放直播流的功能。jessibuca是一个轻量级的JavaScript库，专门用于播放H.264/H.265编码的直播流。通过本项目，开发者可以轻松地将直播流播放功能集成到自己的VUE应用中。

## 功能特点

- **直播流播放**：支持H.264/H.265编码的直播流播放。
- **多分屏模式**：支持1分屏、4分屏、6分屏和9分屏模式，方便用户同时观看多个直播源。
- **全屏播放**：支持全屏播放模式，提供更好的观看体验。
- **抓拍功能**：支持视频抓拍功能，用户可以随时保存当前视频画面。
- **自定义工具栏**：提供自定义工具栏，方便用户进行播放控制和操作。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/vue-jessibuca-player.git
   ```

2. **安装依赖**：
   ```bash
   cd vue-jessibuca-player
   npm install
   ```

3. **运行项目**：
   ```bash
   npm run serve
   ```

4. **配置直播流地址**：
   在项目中找到相应的配置文件，配置直播流的URL地址。

5. **启动播放器**：
   运行项目后，打开浏览器访问本地服务器地址，即可看到直播流播放器界面。
