# 让 Mac 像 Windows 一样使用 Alt + Tab 切换窗口 -- AltTab 工具介绍

## 概览

如果你是从 Windows 转移到 macOS 平台的用户，是否时常怀念着那熟悉的 Alt + Tab 快捷键切换窗口的便捷？不用担心，AltTab 应运而生，专门为了满足这样的需求。它是一款专为 macOS 设计的软件，完美模拟并超越了 Windows 中的 Alt + Tab 功能，让你在 Mac 上也能享受到高效的窗口切换体验。

## 主要特点

- **无缝转换体验**：AltTab 完美复刻了 Windows 中的 Alt + Tab 切换逻辑，使得习惯于 Windows 快捷操作的用户在 Mac 上同样游刃有余。
  
- **高度自定义**：提供了丰富的设置选项，允许用户自定义界面，包括显示应用程序图标、缩略图、调整标题大小等，甚至可以选择几乎任何键作为触发快捷方式。
  
- **附加功能**：除了基本的窗口切换外，还能实现窗口的最小化、关闭、全屏显示以及隐藏和退出应用程序等操作。
  
- **支持暗黑模式**：适应 macOS 的暗黑模式，让软件融入你的个性化系统设置。
  
- **黑名单应用**：可设置不想被 AltTab 列出或触发的应用程序，保持专注不受干扰。
  
- **简体中文界面**：对于中文用户来说，友好的界面和语言设置，让操作更加无障碍。

## 安装与使用

1. **下载与安装**：前往指定资源地址下载最新版本的 `.pkg` 文件，并按照指引完成安装。
   
2. **快捷键使用**：
   - 按住 `⌥ Option + ⇥ Tab` 显示窗口缩略图。
   - 按住 `⌥ Option`，通过 `↹ Tab` 或 `⇧ Shift + ↹ Tab` 来选择目标窗口。
   - 释放 `⌥ Option` 键以切换到选定的窗口。

3. **个性化设置**：安装后，通过软件的设置菜单进行个性化定制，以符合你的使用习惯。

对于那些希望在 macOS 上获得更贴近 Windows 使用习惯的用户，AltTab 是一个不可或缺的工具，它不仅能提升你的工作效率，也让过渡到新平台的过程更加平滑。立即尝试，你会发现操作从未如此顺手。
