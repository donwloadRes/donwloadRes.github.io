# Unity SoftMask 资源文件

## 简介

本仓库提供了一个名为 `UnitySoftMask.zip` 的资源文件，该文件包含了一个用于 Unity 项目的 SoftMask 组件。SoftMask 组件可以帮助你在游戏中实现软遮罩效果，使得 UI 元素能够以柔和的方式进行遮罩。

## 使用方法

1. **下载资源文件**：首先，下载 `UnitySoftMask.zip` 文件。

2. **解压文件**：将下载的 `UnitySoftMask.zip` 文件解压到你的本地目录。

3. **导入项目**：将解压后的文件夹直接放入你的 Unity 项目中。

4. **添加组件**：在 Unity 编辑器中，选择你想要应用 SoftMask 效果的游戏对象，然后在 Inspector 窗口中点击“Add Component”按钮，搜索并添加“SoftMask”组件。

5. **配置组件**：根据需要调整 SoftMask 组件的属性，以实现你想要的遮罩效果。

## 注意事项

- 该资源文件为非插件形式，无需额外安装或配置。
- 确保将文件夹正确放置在 Unity 项目的 Assets 目录下。
- 如果你在使用过程中遇到任何问题，可以参考 Unity 官方文档或社区资源进行排查。
