# BurpSuite 工具安装与激活指南

## 简介
本仓库提供了一个用于 BurpSuite 工具激活的资源文件 `BurpLoaderKeygen.jar.zip`。BurpSuite 是一款广泛使用的网络安全测试工具，而 `BurpLoaderKeygen.jar` 是一个用于激活 BurpSuite 的注册机。

## 资源文件
- **文件名**: `BurpLoaderKeygen.jar.zip`
- **描述**: 包含用于激活 BurpSuite 的注册机文件 `BurpLoaderKeygen.jar`。

## 安装与激活步骤
1. **下载资源文件**:
   - 从本仓库下载 `BurpLoaderKeygen.jar.zip` 文件。

2. **解压文件**:
   - 将 `BurpLoaderKeygen.jar.zip` 解压到任意目录。

3. **准备 BurpSuite 主程序**:
   - 确保你已经下载了 BurpSuite 主程序 `burpsuite_pro_v2023.4.5.jar`。
   - 将 `burpsuite_pro_v2023.4.5.jar` 和 `BurpLoaderKeygen.jar` 放置在同一目录下。

4. **执行激活命令**:
   - 打开命令行工具（CMD）。
   - 切换到包含 `BurpLoaderKeygen.jar` 和 `burpsuite_pro_v2023.4.5.jar` 的目录。
   - 执行以下命令：
     ```sh
     java -jar BurpLoaderKeygen.jar
     ```

5. **启动 BurpSuite**:
   - 在命令行中，继续执行以下命令启动 BurpSuite：
     ```sh
     java -jar burpsuite_pro_v2023.4.5.jar
     ```

## 注意事项
- 请确保你的 Java 环境已正确安装并配置。
