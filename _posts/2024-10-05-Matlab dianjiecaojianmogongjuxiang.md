# Matlab 电解槽建模工具箱

## 简介
该工具箱是用于电解建模的 Matlab 工具，适用于质子交换膜（PEM）和碱性系统。主要功能是根据测量数据对 UI 曲线进行参数化。该工具箱旨在简化流程并提供易于使用的命令，以快速确定 UI 曲线参数值及其不确定性范围。

## 功能特点
- **UI 曲线参数化**：根据测量数据对电解槽的 UI 曲线进行参数化。
- **适用于 PEM 和碱性系统**：支持质子交换膜（PEM）和碱性电解槽系统的建模。
- **易于使用**：提供简单易用的命令，方便用户快速确定参数值及其不确定性范围。

## 安装步骤
1. **下载文件**：从发布页面下载提供的 `electrolyzerModellingToolbox.mltbx` 文件。
2. **安装工具箱**：双击 `.mltbx` 文件开始安装。
3. **使用工具箱**：安装完成后，工具箱函数可用于所有 Matlab 项目。

## 系统要求
- 用户需要在计算机上安装 MATLAB 才能安装电解槽建模工具箱。

## 注意事项
- 确保 MATLAB 已正确安装在您的计算机上。
- 安装过程中请勿关闭 MATLAB 或中断安装过程。
