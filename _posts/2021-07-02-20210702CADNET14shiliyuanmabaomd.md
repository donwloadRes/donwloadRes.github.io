# CAD .NET 14 示例源码包

欢迎使用CAD .NET 14 demo源码包，本资源集成了CAD .NET 14试用版本以及详尽的示例代码，旨在帮助开发者快速上手，实现C#和VB.NET在处理CAD文件时的功能，全面支持.dxf和.dwg两种常见格式。通过这个包，您可以学习如何在各种项目场景下读写和操作CAD文件。

## 文件结构概览

资源包遵循清晰的文件结构设计，便于开发者迅速定位所需示例：

- **bin**: 包含必要的DLL库，分为CLR4兼容版本，以及针对DWGExportLib的不同平台（Win32, Win64）的支持。
- **demo_sources**: 核心部分，包含了多个演示工程，覆盖了从基本的添加实体到复杂的编辑控制等各类应用场景。
    - 各子目录如`AddEntities`, `ImportDemo`, `EditorControlDemo`等，分别对应不同的功能演示，每个目录下有C#和VB.NET两个版本的源码。
    - 每个演示项目都配有完整的解决方案结构，包括属性设置、项目引用等，方便即刻运行体验。
- **doc**: 提供了帮助文档，包含mshelp和mshelp2目录，进一步解释库的使用方法和技术细节。
- **Files**, **Images**, 和 **shx** 目录则存放着示例所需的辅助文件，如字体文件和图像等。

## 快速入门

1. **环境配置**：确保你的开发环境支持.NET Framework 4.x或以上版本。
2. **解压资源**：将此压缩包解压至本地，推荐保留原文件夹结构以利于导航。
3. **打开项目**：导航至`demo_sources`下的任意项目（例如“GettingStarted”），根据喜好选择C#或VB.NET版本，使用Visual Studio或兼容的IDE打开.sln文件。
4. **编译与运行**：直接编译并运行程序，观察示例效果，并可按需修改源码进行定制化开发。

## 功能亮点

- **多语言支持**：同时提供了C#和VB.NET版本的源码，满足不同开发者的需求。
- **完整功能展示**：涵盖了从导入导出、编辑操作到界面集成等多种使用场景，让您的CAD应用开发更加得心应手。
- **易学易用**：示例代码简洁明了，即便是CAD文件处理的新手也能快速理解其工作原理。

通过本资源包，您不仅能够掌握CAD .NET 14的基本使用技巧，还能深入探索高级功能，为构建专业的CAD应用程序奠定坚实的基础。祝您开发顺利！
