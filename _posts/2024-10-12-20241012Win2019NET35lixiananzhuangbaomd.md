# Win2019-NET3.5 离线安装包

## 资源描述

本仓库提供了一个名为 `Win2019-NET3.5.zip` 的资源文件，该文件包含了在 Windows Server 2019 上安装 .NET Framework 3.5 所需的离线安装包。

## 使用场景

在云服务器上安装 IIS 时，可能会需要 .NET Framework 3.5 的环境支持。然而，直接在云服务器上安装 .NET 3.5 可能会失败，因此需要使用离线安装包进行安装。

- 请确保在安装前备份重要数据。
- 安装过程中可能需要管理员权限。
- 安装完成后，建议重启服务器以确保环境配置生效。
