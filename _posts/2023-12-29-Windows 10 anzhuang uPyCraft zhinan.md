# Windows 10 安装 uPyCraft 指南

本资源文件提供了在 Windows 10 系统上安装 uPyCraft 的详细步骤和相关资源。uPyCraft 是一个适用于 MicroPython 的集成开发环境（IDE），特别适合初学者和爱好者使用。

## 内容概述

1. **uPyCraft 简介**  
   uPyCraft 是一个运行在 Windows 平台上的 MicroPython 编程 IDE，界面简洁，操作便利，内置了许多基础操作库，为 MicroPython 爱好者提供了一个简单实用的平台。

2. **安装步骤**  
   - 下载 uPyCraft 的 EXE 可执行文件。
   - 直接运行下载的文件，无需安装 Python 即可使用。

3. **注意事项**  
   - 确保系统已正确安装适用于 ESP8266 的 USB 驱动。
   - 连接硬件设备时，确保设备通过 USB 与电脑连接。

## 使用说明

1. **下载 uPyCraft**  
   从提供的资源链接中下载 uPyCraft 的 EXE 文件。

2. **运行安装**  
   双击下载的 EXE 文件，按照提示完成安装。

3. **连接设备**  
   使用 USB 数据线将 MicroPython 设备（如 ESP32、ESP8266）连接到电脑。

4. **开始编程**  
   打开 uPyCraft，开始编写和调试 MicroPython 代码。
