# Dev C++ 下载地址和安装教程

## 简介
Dev C++ 是一款免费开源的 C/C++ 集成开发环境（IDE），内嵌 GCC 编译器（GCC 编译器的 Windows 移植版），是 NOI、NOIP 等比赛的指定工具。Dev C++ 的优点是体积小（只有几十兆）、安装卸载方便、学习成本低，缺点是调试功能较弱。

## 安装步骤
1. **下载安装包**：
   - 从官方网站或百度网盘下载 Dev C++ 的安装包。
   - 安装包为 `.exe` 程序。

2. **开始安装**：
   - 双击安装包文件，启动安装程序。
   - 选择安装语言为英文（English），安装完成后可设置为简体中文。

3. **同意条款**：
   - 阅读并同意 Dev C++ 的各项条款。

4. **选择安装组件**：
   - 选择“FULL”，全部安装。

5. **选择安装路径**：
   - 自定义安装路径，或使用默认路径。

6. **等待安装完成**：
   - 安装过程通常需要几分钟时间。

7. **配置 Dev C++**：
   - 首次使用时，需要进行简单的配置，包括设置语言、字体和主题风格。
   - 选择简体中文作为界面语言。
   - 字体和主题风格可使用默认设置。

## 使用说明
- 安装完成后，即可开始编写 C/C++ 程序。
- Dev C++ 支持多国语言，包括简体中文，适合初学者使用。

## 注意事项
- 安装过程中，建议选择英文作为初始语言，安装完成后再切换为简体中文。
- 安装路径尽量选择不含中文的路径，以避免潜在的兼容性问题。

通过以上步骤，您可以顺利下载并安装 Dev C++，开始您的 C/C++ 编程之旅。
