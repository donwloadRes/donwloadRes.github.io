# Notepad++ 插件推荐

本文档包含了一系列推荐的 Notepad++ 插件，这些插件可以显著提升你的文本编辑体验。无论你是开发者、作家还是普通用户，这些插件都能帮助你更高效地完成工作。

## 插件列表

以下是推荐的插件列表及其简要描述：

### 编辑类
- **BetterMultiSelection**：多光标选择器，提供更多编辑功能。
- **Code Alignment**：通过填充空格自动对齐代码。
- **CoolFormat**：支持多种语言的代码格式化工具。
- **NppAStyle**：Artistic Style 插件，支持代码格式化。
- **SurroundSelection**：自动补全括号。
- **XBrackets Lite**：自动补全括号和引号。

### 显示类
- **ElasticTabstops**：对齐显示 Tab 宽度。
- **HexEditor**：16 进制查看器。
- **JSON Viewer**：JSON 查看器。
- **Markdown Panel**：显示 Markdown 文本。

### 执行类
- **NppEventExec**：条件触发的执行 console 命令。
- **NppExec**：执行 console 命令。
- **RunMe**：shell 运行当前文件。

### 查看、跳转类
- **Compare**：比较两个文本的差异。
- **Location Navigate**：光标位置导航，编辑过的部分高亮标记显示。
- **NavigateTo**：通过关键词筛选搜索标签卡。
- **Regex Trainer**：正则表达式训练器。
- **Source Cookifier**：代码符号跳转插件。

### 辅助功能
- **GitSCM**：Git 集成插件。
- **Linefilter3**：快速对多行内容进行筛选。
- **Merge files in one**：将多个文件合并成为一个文件。
- **NppQrCode**：选中文本生成二维码。

## 安装方法

1. 下载插件文件。
2. 将插件文件拷贝到 Notepad++ 安装目录下的 `plugins` 目录。
3. 重启 Notepad++，插件即可生效。

## 注意事项

- 部分插件可能需要特定的环境或配置才能正常工作。
- 请确保 Notepad++ 版本与插件兼容。

希望这些插件能帮助你提升工作效率，享受更流畅的文本编辑体验！
