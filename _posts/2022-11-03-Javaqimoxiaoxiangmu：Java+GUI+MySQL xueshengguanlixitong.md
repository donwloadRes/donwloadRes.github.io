# Java期末小项目：Java+GUI+MySQL 学生管理系统

## 项目简介

本项目是一个基于Java语言的学生管理系统，结合了图形用户界面（GUI）和MySQL数据库。系统主要功能包括用户登录、学生信息查询、添加、删除和修改等操作。通过该系统，用户可以方便地管理学生信息，实现对学生信息的有效管理。

## 功能模块

1. **用户登录**：用户可以通过输入用户名和密码进行登录，系统会验证用户的身份。
2. **主界面展示**：登录成功后，用户将进入主界面，主界面提供了查询和操作选项。
3. **学生信息查询**：用户可以根据学生姓名查询学生信息。
4. **学生信息添加**：用户可以添加新的学生信息到系统中。
5. **学生信息删除**：用户可以选择删除系统中的学生信息。
6. **学生信息修改**：用户可以修改系统中已有的学生信息。

## 技术栈

- **Java**：项目主要使用Java语言进行开发。
- **GUI**：使用Java Swing库实现图形用户界面。
- **MySQL**：使用MySQL数据库进行数据存储和管理。

## 使用说明

1. **登录系统**：启动系统后，输入正确的用户名和密码进行登录。
2. **主界面操作**：登录成功后，用户可以在主界面进行学生信息的查询、添加、删除和修改操作。
3. **退出系统**：用户可以通过点击“取消”按钮退出系统。

## 项目结构

- **LoginFrame.java**：用户登录界面的实现。
- **Main.java**：主界面的实现，包含学生信息查询、添加、删除和修改功能。
- **Model.java**：数据模型类，负责与数据库进行交互。

## 运行环境

- JDK 1.8及以上
- MySQL 5.7及以上
- Eclipse或其他Java IDE

## 安装与配置

1. **导入项目**：将项目导入到Eclipse或其他Java IDE中。
2. **配置数据库**：在MySQL中创建相应的数据库和表结构，并配置数据库连接信息。
3. **运行项目**：运行`LoginFrame.java`文件，启动系统。

## 注意事项

- 请确保数据库连接信息正确，否则系统无法正常运行。
- 在添加、删除或修改学生信息时，请确保输入信息的准确性。
