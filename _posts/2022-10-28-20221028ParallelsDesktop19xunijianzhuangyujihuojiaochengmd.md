# Parallels Desktop 19 虚拟机安装与激活教程

本仓库提供了一个详细的图文教程，帮助用户安装和激活 Parallels Desktop 19 虚拟机。教程内容包括软件的下载、安装、激活以及启动 Windows 系统的步骤。

## 内容概述

1. **下载安装 Parallels Desktop 软件**
   - 下载完成后，双击打开安装文件，输入开机密码或指纹认证，等待安装初始化。
   - 安装完成后，Parallels Desktop 19 将自动启动。

2. **激活 Parallels Desktop 19**
   - 将下载好的“激活”目录复制到桌面或其它目录中，打开“激活”目录，双击其中的“Install_Patch.command”文件。
   - 输入开机密码，以获得权限进行激活。
   - 在界面内遇到需要输入的时候，全部输入 y 即可。
   - 激活完成后，可在“账户和许可证”中查看是否激活成功。

3. **启动 Parallels Desktop 19**
   - 此版本激活为不完美版本，每次启动需要通过运行“启动器”，将安装包内的“启动器”复制到应用程序。
   - 运行后，在提示框中点击“继续操作”，然后输入开机密码即可。

4. **安装 Windows 系统**
   - 启动 Parallels Desktop 后，如果未有安装后虚拟机，软件会自动弹出界面，提示在线下载推荐并适配的 Windows 系统。
   - 点击“从 Microsoft 获取 Windows 11”即可。
   - 如果想安装自己下载的系统镜像，则点击“通过映像文件安装 Windows、Linux 或 macOS”。

## 注意事项

- 激活完成后，此激活文件夹不能删除和移动位置。
- 如果启动用发现无网络，请在虚拟机的右上角切换一个网络，然后再切回共享网络，即可。

通过本教程，您可以顺利完成 Parallels Desktop 19 的安装与激活，并成功启动 Windows 系统。希望本教程对您有所帮助！
