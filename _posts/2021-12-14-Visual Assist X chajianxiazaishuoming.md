# Visual Assist X 插件下载说明

## 简介
本仓库提供最新版本的 Visual Assist X 插件，适用于 Visual Studio 2019、2017、2015 和 2013 版本。该插件经过亲测，确保在上述版本的 Visual Studio 中可用。

## 功能特点
- **智能代码补全**：自动识别关键字、系统函数和成员变量，提供输入提示。
- **语法检查**：自动更正大小写错误，自动标示错误。
- **代码重构**：提供高效的代码重构功能，提升开发效率。
- **快速导航**：轻松移动到项目和解决方案中的任何文件、方法、符号或引用。

## 适用版本
- Visual Studio 2019
- Visual Studio 2017
- Visual Studio 2015
- Visual Studio 2013

## 安装步骤
1. **卸载旧版本**：如果之前安装过 Visual Assist X，请先卸载。
2. **手动检查并删除旧版本文件夹**：在扩展文件夹中手动检查并删除旧版本的 Visual Assist X 文件夹。
3. **替换 DLL 文件**：将提供的 `va_x.dll` 文件替换到对应的路径下。
4. **完成安装**：安装完成后，可能会显示“License: trial”，但所有功能应正常工作，无任何限制。

## 文件路径
- Visual Studio 2008: `C:\Program Files (x86)\Visual Assist X\`
- Visual Studio 2010: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\10.0\Extensions\Whole Tomato Software\Visual Assist\__version__\`
- Visual Studio 2011/2012: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\11.0\Extensions\__random_dir__\`
- Visual Studio 2013: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\12.0\Extensions\__second_random_dir__\`
- Visual Studio 2015: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\14.0\Extensions\__random_dir__\`
- Visual Studio 2017: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\15.0__%XXX%\Extensions\__second_random_dir__\`
- Visual Studio 2019: `%USERPROFILE%\AppData\Local\Microsoft\VisualStudio\16.0__%XXX%\Extensions\__second_random_dir__\`
