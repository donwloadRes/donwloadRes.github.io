# 常用AD库（较全）

## 简介

本仓库提供了一个常用AD库资源文件，包含了AD封装库和常用元件，非常适合初学者使用。该资源文件内容较全，能够满足大多数电子设计的需求。

## 资源内容

- **AD封装库**：包含多种常用元件的封装，方便用户在设计电路时直接调用。
- **常用元件**：涵盖了初学者在电子设计中常用的各种元件，如电阻、电容、晶体管等。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **下载资源文件**：

   将下载的资源文件导入到你的AD软件中，即可开始使用。
