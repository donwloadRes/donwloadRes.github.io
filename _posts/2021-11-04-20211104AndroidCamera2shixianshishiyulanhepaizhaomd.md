# Android Camera2 实现实时预览和拍照

本资源文件提供了一个使用 Android Studio 开发的示例项目，展示了如何使用 Camera2 API 实现实时预览和拍照功能。通过本项目，您可以轻松学习如何在 Android 设备上使用 Camera2 API 进行拍照，并将照片保存到手机存储中。

## 功能特点

- **实时预览**：使用 Camera2 API 实现相机实时预览功能，用户可以在屏幕上看到实时拍摄画面。
- **拍照功能**：通过简单的点击按钮，即可触发拍照操作，并将照片保存到手机存储中。
- **照片保存**：照片保存格式为 `fpg`，保存路径在代码中有详细注释，方便开发者理解和修改。
- **代码简洁**：本项目代码经过优化，力求简洁易懂，适合初学者学习和参考。

## 使用说明

1. **导入项目**：将本项目导入到 Android Studio 中。
2. **运行项目**：在 Android 设备上运行项目，确保设备支持 Camera2 API。
3. **实时预览**：启动应用后，您将看到相机实时预览画面。
4. **拍照**：点击拍照按钮，照片将被保存到手机存储中，保存路径可在代码中查看。

## 注意事项

- 本项目仅作为学习参考，实际开发中可能需要根据具体需求进行调整。
- 请确保您的设备支持 Camera2 API，否则可能无法正常运行。
