# 0.96寸OLED资源

## 简介
本资源提供了一个用于0.96寸OLED屏幕的开发库，该库采用软件模拟I2C通信方式，具有高可移植性和易用性。无论你是初学者还是经验丰富的开发者，都可以轻松地将该库集成到你的项目中，快速实现OLED屏幕的显示功能。

## 特点
- **软件模拟I2C**：采用软件模拟I2C通信方式，无需硬件I2C接口，方便在各种平台上使用。
- **高可移植性**：代码结构清晰，易于移植到不同的硬件平台和开发环境。
- **使用方便**：提供了简洁的API接口，方便开发者快速上手，实现屏幕的初始化、显示文本、绘制图形等功能。

## 适用场景
- 嵌入式系统开发
- 物联网设备
- 电子DIY项目
- 教育实验

## 使用说明
1. **下载库文件**：将本资源中的所有文件下载到你的项目目录中。
2. **包含头文件**：在你的主程序中包含`oled.h`头文件。
3. **初始化屏幕**：调用`OLED_Init()`函数初始化OLED屏幕。
4. **显示内容**：使用提供的API函数，如`OLED_ShowString()`、`OLED_DrawBMP()`等，在屏幕上显示文本或图形。

## 示例代码
以下是一个简单的示例代码，展示了如何初始化OLED屏幕并显示一行文本：

```c
#include "oled.h"

int main(void) {
    // 初始化OLED屏幕
    OLED_Init();

    // 显示文本
    OLED_ShowString(0, 0, "Hello, OLED!");

    while (1) {}
}
```
