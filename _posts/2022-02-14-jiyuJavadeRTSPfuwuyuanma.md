# 基于Java的RTSP服务源码

## 简介
本仓库提供了一个基于Java的RTSP服务源码，支持RTP/UDP和RTP/TCP模式的VLC播放。通过该源码，开发者可以快速搭建和部署自己的RTSP服务，实现视频流的实时传输和播放。

## 功能特点
- **支持RTP/UDP模式**：适用于需要高实时性的视频流传输场景。
- **支持RTP/TCP模式**：适用于网络环境不稳定或需要更可靠传输的场景。
- **兼容VLC播放器**：确保视频流能够通过VLC播放器进行播放。

## 使用说明
1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/java-rtsp-service.git
   ```
2. **导入项目**：
   将项目导入到你喜欢的Java开发环境中（如Eclipse、IntelliJ IDEA等）。
3. **配置参数**：
   根据需要修改配置文件，设置RTSP服务的相关参数。
4. **运行服务**：
   运行项目中的主类，启动RTSP服务。
5. **测试播放**：
   使用VLC播放器或其他支持RTSP协议的播放器，输入服务地址进行视频流的播放测试。
