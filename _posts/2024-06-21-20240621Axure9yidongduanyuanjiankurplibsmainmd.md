# Axure9 移动端元件库 rplibs-main

欢迎使用 Axure9 移动端元件库 rplibs-main！这个资源文件包含了微信和小程序、iOS、macOS、Android、Windows 等多平台的 Axure RP 元件库，旨在帮助你轻松创建高保真的移动端原型。

## 资源描述

本元件库的设计参考了各平台的官方设计规范文档，力求在每一个细节上都做到精准无误。无论是长度、宽度、转角还是颜色，我们都尽可能地还原了官方的设计标准，确保你的原型在视觉上与实际产品保持高度一致。

通过使用这个高保真元件库，你可以更好地与开发、测试等团队成员进行深入沟通，确保原型设计与最终产品的实现无缝对接。

## 使用建议

如果你的团队中有 UI 设计人员，并且你希望在 Axure RP 中创建低保真原型，建议你进行以下设置：

1. 打开 Axure RP 软件，进入菜单 “Project > Page Style Editor”。
2. 调整 Sketchiness 为 20 左右，以增加手绘风格的效果。
3. 将 Color 设置为 Grayscale，以减少颜色的干扰。
4. 选择一个中文手写字体作为默认字体，确保原型不会干扰到 UI 设计的视觉风格。

希望这些建议对你有所帮助。感谢你自由使用这个开源元件库，祝你在原型设计中取得更好的成果！
