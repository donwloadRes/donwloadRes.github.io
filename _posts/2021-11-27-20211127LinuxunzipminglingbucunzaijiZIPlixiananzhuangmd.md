# Linux unzip命令不存在及ZIP离线安装

## 简介
在Linux系统中，有时会遇到`unzip`命令不存在的情况，这通常是因为系统没有安装`unzip`工具。本文将介绍如何在Linux系统中进行`unzip`的离线安装，并提供相应的资源文件下载。

## 安装步骤

### 1. 下载unzip离线安装包
首先，您需要下载`unzip`的离线安装包。本仓库提供了一个`unzip`的RPM安装包，您可以直接下载并使用。

### 2. 安装unzip
将下载的离线安装包上传到Linux系统中，然后使用以下命令进行安装：
```bash
rpm -ivh unzip-6.0-5.el6.x86_64.rpm
```
请根据实际下载的文件名替换命令中的文件名。

### 3. 验证安装
安装完成后，您可以使用以下命令验证`unzip`是否成功安装：
```bash
unzip -v
```
如果能够显示版本信息，则表示`unzip`已成功安装。

## 注意事项
- 根据不同的Linux发行版和版本，安装命令可能会有所不同，请根据系统的具体情况进行适当的调整。
- 本仓库提供的安装包适用于CentOS 6.x系统，如果您使用的是其他版本的Linux系统，请自行查找相应的安装包。
