# 基于Vue的PC端综合性网上购物商城

## 项目简介

本项目是一个基于Vue的PC端综合性网上购物商城，涵盖了前台商城和后台管理系统。后端采用Koa2框架，数据库使用MySQL进行数据存储。该项目旨在提供一个完整的电商解决方案，适用于开发者和企业在PC端搭建自己的网上购物平台。

## 功能特点

- **前台商城**：
  - 用户注册与登录
  - 商品浏览与搜索
  - 购物车管理
  - 订单生成与支付
  - 用户中心（个人信息管理、订单查看等）

- **后台管理系统**：
  - 商品管理（添加、编辑、删除商品）
  - 订单管理（查看、处理订单）
  - 用户管理（管理用户信息）
  - 数据统计与分析

## 技术栈

- **前端**：
  - Vue.js
  - Vue Router
  - Vuex
  - Axios
  - Element UI

- **后端**：
  - Koa2
  - MySQL
  - Sequelize（ORM）

## 安装与运行

1. **克隆项目**：
   ```bash
   git clone https://github.com/your-repo/vue-shopping-mall.git
   ```

2. **安装依赖**：
   ```bash
   cd vue-shopping-mall
   npm install
   ```

3. **配置数据库**：
   - 在`config`目录下找到`database.js`文件，配置MySQL数据库连接信息。

4. **启动项目**：
   ```bash
   npm run serve
   ```

5. **访问项目**：
   - 前台商城：`http://localhost:8080`
   - 后台管理系统：`http://localhost:8080/admin`
