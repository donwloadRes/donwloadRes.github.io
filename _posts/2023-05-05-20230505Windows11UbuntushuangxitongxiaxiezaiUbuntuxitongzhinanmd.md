# Windows 11 + Ubuntu 双系统下卸载 Ubuntu 系统指南

本资源文件提供了一个详细的指南，帮助用户在 Windows 11 和 Ubuntu 双系统环境下卸载 Ubuntu 系统。通过本指南，您可以安全地删除 Ubuntu 分区及 BIOS 启动项，确保系统不再启动 Ubuntu，同时不会对 Windows 系统造成影响。

## 内容概述

1. **准备工作**
   - 确保备份重要数据
   - 下载必要的软件工具

2. **删除 Ubuntu 分区**
   - 使用 DiskGenius 软件删除 Ubuntu 分区
   - 保存更改并确认分区已删除

3. **删除 BIOS 启动项**
   - 使用 EasyUEFI 软件删除 EFI 启动项
   - 确保系统不再启动 Ubuntu

4. **验证与清理**
   - 重启计算机，验证 Ubuntu 系统是否已成功卸载
   - 清理残留文件（如有）

## 注意事项

- 在操作前，请确保已备份所有重要数据，以防数据丢失。
- 操作过程中，请仔细阅读每一步的说明，避免误操作。
- 如果您不熟悉磁盘管理和 UEFI 设置，建议在专业人士的指导下进行操作。

通过本指南，您可以轻松地在 Windows 11 和 Ubuntu 双系统环境下卸载 Ubuntu 系统，恢复单一的 Windows 系统环境。
