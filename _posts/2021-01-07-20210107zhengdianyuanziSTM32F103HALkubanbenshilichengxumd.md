# 正点原子STM32F103 HAL库版本示例程序

## 简介

本仓库提供了正点原子STM32F103 HAL库版本的示例程序，旨在帮助开发者快速上手使用STM32F103微控制器，并熟悉HAL库的使用方法。

## 资源内容

- **示例程序**：包含多个基于HAL库的示例程序，涵盖了GPIO、UART、I2C、SPI等常用外设的使用。
- **文档**：提供了详细的代码注释和使用说明，帮助开发者理解每个示例程序的功能和实现方式。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **导入工程**：
   使用STM32CubeIDE或其他支持HAL库的开发工具，导入本仓库中的示例工程。

3. **编译与下载**：
   根据具体示例程序的需求，配置开发板并编译代码，然后将生成的二进制文件下载到STM32F103开发板上。

4. **调试与学习**：
   通过调试工具观察程序运行情况，学习HAL库的使用方法，并根据需要修改和扩展示例程序。
