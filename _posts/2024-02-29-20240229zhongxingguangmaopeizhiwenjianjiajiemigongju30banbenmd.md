# 中兴光猫配置文件加解密工具3.0版本

## 简介
本仓库提供了一个用于中兴光猫配置文件加解密的工具，版本为3.0。该工具名为`ztecfg.exe`，支持对中兴光猫3.0版本的配置文件进行加解密操作。

## 功能说明
- **解密功能**：使用`ztecfg.exe`可以对中兴光猫的配置文件进行解密。
- **加密功能**：同样，该工具也支持对配置文件进行加密操作。

## 使用方法
1. **解密操作**：
   ```
   ztecfg.exe -d AESCBC -i .\（要解密的文件名）db_user_cfg.xml -o （解密后文件名）123.cfg
   ```
   其中：
   - `-d AESCBC`：指定解密算法为AESCBC。
   - `-i .\（要解密的文件名）db_user_cfg.xml`：指定要解密的文件路径及文件名。
   - `-o （解密后文件名）123.cfg`：指定解密后的文件名及保存路径。

2. **加密操作**：
   ```
   ztecfg.exe -e AESCBC -i .\（要加密的文件名）123.cfg -o （加密后文件名）db_user_cfg.xml
   ```
   其中：
   - `-e AESCBC`：指定加密算法为AESCBC。
   - `-i .\（要加密的文件名）123.cfg`：指定要加密的文件路径及文件名。
   - `-o （加密后文件名）db_user_cfg.xml`：指定加密后的文件名及保存路径。

## 注意事项
- 该工具仅支持中兴光猫3.0版本的配置文件加解密。
- 请确保在使用前备份好原始配置文件，以防操作失误导致数据丢失。
