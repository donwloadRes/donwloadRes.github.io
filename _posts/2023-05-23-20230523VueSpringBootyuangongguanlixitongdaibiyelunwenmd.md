# Vue+SpringBoot员工管理系统 带毕业论文

## 项目简介

本仓库提供了一个完整的员工管理系统，采用Vue.js作为前端框架，Spring Boot作为后端框架。该系统不仅包含了完整的代码实现，还附带了毕业论文，适合作为毕业设计或学习参考。

## 功能特点

- **前端技术栈**: Vue.js
- **后端技术栈**: Spring Boot
- **数据库**: 支持多种数据库（如MySQL、PostgreSQL等）
- **用户管理**: 员工信息的增删改查
- **权限管理**: 不同角色的权限控制
- **毕业论文**: 附带详细的毕业论文文档，涵盖系统设计、实现过程及技术细节

## 使用说明

1. **克隆仓库**: 
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **前端配置**:
   - 进入前端目录：`cd frontend`
   - 安装依赖：`npm install`
   - 启动前端服务：`npm run serve`

3. **后端配置**:
   - 进入后端目录：`cd backend`
   - 配置数据库连接信息
   - 启动Spring Boot应用

4. **运行系统**:
   - 访问前端地址：`http://localhost:8080`

毕业论文文档位于`docs`目录下，详细介绍了系统的架构设计、技术选型、实现过程以及测试结果。适合作为毕业设计的参考文档。
