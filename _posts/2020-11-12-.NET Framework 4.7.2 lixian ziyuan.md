# .NET Framework 4.7.2 离线 资源

## 简介

本资源提供了一个 .NET Framework 4.7.2 的离线安装程序，适用于那些无法使用网络安装程序的情况。此离线安装包比网络安装程序大，但不包括语言包。在有网络连接的情况下，建议使用网络安装程序，以实现最佳效率和带宽需求。

## 安装说明

1. 从本资源中下载 .NET Framework 4.7.2 离线安装程序。
2. 双击安装程序文件并按照提示完成安装。
3. 安装完成后，可能需要重新启动计算机。
