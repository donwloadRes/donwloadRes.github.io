# Vue2 和 Vue3 + JS + TS 的 Three.js 项目集合

## 项目介绍

本仓库包含四个基于 Vue2 和 Vue3 框架的 Three.js 项目，使用 JavaScript 和 TypeScript 编写。这些项目展示了如何在 Vue 环境中集成 Three.js，实现厂房、机台和楼层的三维显示效果。

## 项目列表

1. **厂房显示**  
   该项目展示了如何使用 Three.js 在 Vue 中创建一个三维厂房模型，并实现基本的交互功能。

2. **机台显示**  
   该项目专注于机台的三维建模和显示，展示了如何在 Vue 中动态加载和渲染 Three.js 模型。

3. **楼层显示**  
   该项目展示了如何使用 Three.js 在 Vue 中创建楼层的三维模型，并实现楼层之间的切换和交互。

4. **综合项目**  
   该项目整合了厂房、机台和楼层的三维显示，展示了如何在 Vue 中实现复杂的三维场景管理和交互。

## 技术栈

- **Vue2 / Vue3**: 前端框架，用于构建用户界面。
- **Three.js**: 用于创建和显示三维图形的 JavaScript 库。
- **JavaScript / TypeScript**: 项目主要使用 JavaScript 和 TypeScript 进行开发。

## 使用说明

1. **克隆仓库**  
   使用以下命令克隆本仓库到本地：
   ```bash
   git clone [仓库地址]
   ```

2. **安装依赖**  
   进入项目目录，安装项目依赖：
   ```bash
   cd [项目目录]
   npm install
   ```

3. **运行项目**  
   使用以下命令启动项目：
   ```bash
   npm run serve
   ```

4. **访问项目**  
   项目启动后，打开浏览器访问 `http://localhost:8080` 即可查看项目效果。
