# Vue 开发者工具 (谷歌浏览器插件)

## 资源描述

本仓库提供了一个用于谷歌浏览器的 Vue 开发者工具插件，支持 Vue 2 和 Vue 3 的开发调试。该插件可以帮助开发者更方便地调试和分析 Vue 应用。

## 插件版本

- **Vue 2 版本**: 5.3.4
- **Vue 3 版本**: 6.0.0.18 beta

## 安装说明

1. **下载插件**: 从本仓库下载对应的插件文件。
2. **安装插件**:
   - 打开谷歌浏览器，点击右上角的菜单按钮（三个竖点）。
   - 选择 `更多工具` -> `扩展程序`。
   - 在扩展程序页面，打开右上角的 `开发者模式`。
   - 点击 `加载已解压的扩展程序`，选择你下载的插件文件夹，完成安装。

## 使用说明

- **Vue 2 项目**: 安装并启用 Vue 2 版本的开发者工具。
- **Vue 3 项目**: 安装并启用 Vue 3 版本的开发者工具。

## 注意事项

- 两个版本的插件需要分别安装，分别支持 Vue 2 和 Vue 3。
- 请确保你的浏览器版本支持开发者模式下的扩展程序安装。
