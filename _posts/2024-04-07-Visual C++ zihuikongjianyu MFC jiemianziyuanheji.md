# Visual C++ 自绘控件与 MFC 界面资源合集

本仓库提供了一个丰富的资源文件，旨在帮助开发者学习和掌握 Visual C++ 自绘控件及 MFC 界面编程。资源文件包含了多种常用控件的自绘源码，以及一些经典的界面示例，是学习 VC 界面编程的绝佳资料。

## 资源内容概述

### 自绘控件源码
- **对话框**：包含多种自绘对话框的实现代码。
- **编辑框**：自绘编辑框的源码，支持多种样式和功能。
- **菜单**：自绘菜单的实现，提供丰富的菜单样式。
- **滚动条**：自绘滚动条的源码，支持自定义外观和行为。
- **进度条**：自绘进度条的实现，适用于各种进度展示场景。
- **标签页**：自绘标签页的源码，支持多标签切换。
- **列表框**：自绘列表框的实现，提供丰富的列表展示功能。

### 其他控件
- **按钮**：自绘按钮的源码，支持多种按钮样式。
- **滑块**：自绘滑块的实现，适用于音量控制等场景。
- **单选框**：自绘单选框的源码，支持自定义外观。
- **富文本框**：自绘富文本框的实现，支持丰富的文本格式。

### 界面示例
- **QQ 界面**：模拟 QQ 界面的自绘实现。
- **云界面**：模拟云服务界面的自绘实现。
- **天气预报界面**：模拟天气预报应用界面的自绘实现。

### 界面库与工具
- **Skin# 全集合**：包含多种语言（如 VC++、VB.NET、易语言、C# 等）的破解版界面库，以及皮肤制作工具和详细的使用教程。
- **GLUI C++ 界面库**：包含 GLUI 界面库的源码、开发文档和 GLUT 库。
