# C# WEBAPI 及 Winform 调用示例

## 项目介绍

本仓库提供了一个完整的 C# WebAPI 项目示例，以及如何在 Winform 中调用这些接口的方法。项目代码结构清晰，易于理解和学习，非常适合初学者参考和借鉴。

## 项目内容

- **WebAPI 项目**：包含接口的生成和相关配置。
- **Winform 调用示例**：展示了如何在 Winform 应用程序中调用 WebAPI 接口。

## 开发步骤

详细的开发步骤和说明可以参考我的博客文章：[C# WebAPI 及 Winform 调用详解](http://www.cnblogs.com/gudaozi/p/8384734.html)。

## 如何使用

1. **克隆仓库**：
   ```sh
   git clone https://github.com/your-repo-url.git
   ```

2. **打开解决方案**：
   使用 Visual Studio 打开项目解决方案文件（.sln）。

3. **运行 WebAPI 项目**：
   启动 WebAPI 项目，确保接口可以正常访问。

4. **运行 Winform 项目**：
   启动 Winform 项目，查看如何在 Winform 中调用 WebAPI 接口。
