# Windows端UI自动化---Python3 + uiautomation框架

## 简介

本资源文件提供了使用Python3和uiautomation框架进行Windows端UI自动化的详细教程和示例代码。通过本教程，您将学会如何使用UIspy.exe工具获取元素信息、打开程序、获取软件主界面和弹框提示，以及对不同元素进行操作。

## 主要内容

1. **学会使用UIspy.exe软件**
   - 该软件可以实时查看你需要操作元素的信息，例如点击、输入等操作。

2. **打开某个程序**
   - 使用Python的subprocess模块打开系统自带的程序或其他可执行文件。

3. **获取软件的主界面**
   - 通过uiautomation库捕获Windows软件的主界面，以便后续操作。

4. **获取弹框的提示信息**
   - 获取弹框的提示内容，适用于每次弹框导出的内容不一样的情况。

5. **不同类型元素的操作**
   - 对程序窗口、按钮、文件显示、输入框等不同类型的元素进行操作。

6. **元素的属性**
   - 了解元素的ClassName、Name、ProcessId、AutomationId等属性。

7. **常用的操作**
   - 包括双击、点击、右键点击、发送字符等常用操作。

## 使用方法

1. **安装Python3**
   - 确保您的系统中已安装Python3。

2. **安装uiautomation库**
   - 使用pip安装uiautomation库：
     ```bash
     pip install uiautomation
     ```

3. **下载UIspy.exe**
   - 下载UIspy.exe工具，用于实时查看和获取元素信息。

4. **运行示例代码**
   - 根据教程中的示例代码，编写和运行您的自动化脚本。

## 注意事项

- 确保您的Windows系统版本支持UIAutomation API。
- 在使用UIspy.exe工具时，请确保其路径正确。
- 在操作不同类型的元素时，注意元素的属性和操作方法。

通过本教程，您将能够掌握使用Python3和uiautomation框架进行Windows端UI自动化的基本技能，并能够编写简单的自动化脚本。
