# 最完整的Eclipse插件AmaterasUML下载安装教程

## 简介
本资源文件提供了最完整的Eclipse插件AmaterasUML的下载安装教程。AmaterasUML是一个功能强大的Eclipse插件，用于在Eclipse IDE中创建和编辑UML图。通过本教程，您可以轻松地安装和配置AmaterasUML插件，以便在Eclipse中进行UML建模。

## 安装步骤

### 1. 安装GEF插件
1. 打开Eclipse官网。
2. 在官网中搜索GEF项目并下载。
3. 在Eclipse中，选择`Help` -> `Install New Software`。
4. 复制GEF的在线安装地址并粘贴到Eclipse中。
5. 点击`Add`，自定义插件名字，粘贴地址，选择`Select All`，然后点击`Next`和`Accept`。
6. 等待安装完成，如果遇到错误，重复上述步骤。

### 2. 安装AmaterasUML插件
1. 下载AmaterasUML_1.3.4.zip文件。
2. 解压文件，将得到的3个jar包拷贝到Eclipse的`plugins`文件夹中。
3. 重启Eclipse。
4. 验证安装是否成功：在`File` -> `New` -> `Other`下查看是否有AmaterasUML选项。

### 3. 解决常见问题
如果在`File` -> `New` -> `Other`下没有看到AmaterasUML选项，可以尝试以下解决方法：
1. 删除Eclipse安装目录下的`/configuration/org.eclipse.update`文件夹。
2. 以管理员身份运行命令行，打开Eclipse安装目录，输入`eclipse -clean`。
3. 重启Eclipse，在`New` -> `Other`下应该能看到AmaterasUML选项。

## 使用说明
1. 在Eclipse中，选择`File` -> `New` -> `Other` -> `AmaterasUML` -> `Class Diagram`。
2. 命名并创建`.cld`文件，即可开始使用UML类图。

## 注意事项
- 确保Eclipse版本与插件兼容。
- 安装过程中遇到问题，可以参考官方文档或社区支持。

通过本教程，您可以顺利安装并使用AmaterasUML插件，提升在Eclipse中的UML建模效率。
