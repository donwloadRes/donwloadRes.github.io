# C# AnyCAD 三维动画模型演示 DEMO

本仓库提供了一个使用 C# 和 AnyCAD 实现的三维动画模型演示 DEMO。该 DEMO 基于 Visual Studio 2010 开发，包含了完整的源代码，支持模型的缩放和旋转操作。

## 功能特点

- **三维动画演示**：使用 AnyCAD 库实现三维模型的动画演示。
- **缩放与旋转**：用户可以通过鼠标操作对模型进行缩放和旋转，以便从不同角度观察模型。
- **完整源码**：提供了完整的源代码，方便开发者学习和二次开发。

## 使用说明

1. **环境要求**：
   - Visual Studio 2010 或更高版本。
   - AnyCAD 库（请确保已正确配置 AnyCAD 环境）。

2. **运行步骤**：
   - 克隆或下载本仓库到本地。
   - 使用 Visual Studio 打开项目文件。
   - 编译并运行项目，即可看到三维动画模型的演示效果。

3. **操作指南**：
   - **缩放**：使用鼠标滚轮进行缩放操作。
   - **旋转**：按住鼠标左键并拖动，可以旋转模型。
