# vcenter 6.7 (VCSA) 部署指南

欢迎使用vcenter 6.7 (VCSA) 部署指南，本指南详细阐述了如何在您的IT环境中成功部署VMware vCenter Server Appliance 6.7。该文档基于[CSDN博客](https://blog.csdn.net/weixin_33754913/article/details/92493241)提供的丰富信息编写，旨在帮助管理员和IT专业人员轻松完成VCSA的安装与配置。

## 文档概述

该部署指南覆盖了从下载安装介质到完成两阶段部署的全过程。包括但不限于：

### 准备工作
- **下载安装文件**: 您需要从官方渠道获取VMware vCenter Server Appliance 6.7的ISO映像文件。
- **环境要求**: 确保拥有一个配置好的ESXi主机，以及对于FQDN的正确解析。

### 部署步骤

#### 第一阶段
1. **初始化安装**: 挂载ISO文件，启动安装向导。
2. **同意许可协议**。
3. **选择部署类型**，通常建议选择嵌入式部署。
4. **指定部署位置**，可以直接输入ESXi主机地址。
5. **安全性配置**, 包括证书警告确认，以及设定虚拟机名称和密码。
6. **配置规模选择**，依据实际需求选择适合的部署大小。
7. **选择存储**，注意精简磁盘模式的选择。
8. **网络配置**，重点是FQDN的正确配置，确保其在内部网络可解析。

#### 第二阶段
1. **继续配置**: 包含NTP服务器的设置，SSO配置等，这些配置项非常关键，密码需谨慎保存。
2. **部署审核**，检查所有配置信息无误后，开始部署。
3. **完成部署**，等待安装完成，并进行验证。

### 错误处理
提供了常见问题的解决方案，例如FQDN解析问题，确保部署流程顺畅。

### 注意事项
- 测试环境可以简化某些配置，但在生产环境务必遵循最佳实践。
- 记录所有密码和重要配置，以备后续管理。
