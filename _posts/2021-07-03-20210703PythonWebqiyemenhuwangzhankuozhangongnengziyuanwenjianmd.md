# Python Web企业门户网站扩展功能资源文件

本资源文件提供了第五课“Python Web企业门户网站-常用的扩展功能”的相关内容。该课程主要介绍了如何使用Python和Django框架来构建企业门户网站，并实现一些常用的扩展功能。

## 内容概述

1. **基本框架**：介绍了门户网站的基本设计框架，包括页面制作、页面切换、数据库操作、后台系统管理、模板显示等基本知识点。
2. **信号触发器的使用**：详细讲解了信号触发器的使用方法，包括招聘信息发布、基于模型表单的应聘信息上传等功能。
3. **发送邮件**：介绍了如何在门户网站中集成邮件发送功能，以便在用户提交信息后，后台服务器能够通过邮件服务向指定邮箱发送通知。
4. **资料下载**：讲解了如何创建资料模型，并开发资料下载列表页面，方便用户下载相关资料。
