# SecureCRT+SecureFX 7.0.0.326 绿色汉化整合版 README.md

## 介绍

欢迎使用SecureCRT+SecureFX 7.0.0.326绿色汉化整合版，这是一套专为Windows用户打造的SSH客户端及文件传输解决方案。本版本集成了SecureCRT和SecureFX两个重量级工具，旨在提供高效、便捷的远程服务器管理和文件传输体验。该整合版经过精心汉化处理，支持32位与64位系统，完全绿色便携，无需安装，即刻启动您的远程工作之旅。

## 主要特性

- **SecureCRT**: 强大的终端仿真器，支持SSH（SSH1和SSH2）、Telnet、Serial等多种协议，特别是其多标签界面极大地简化了同时管理多个远程会话的任务。
  
- **SecureFX**: 专业的文件传输工具，支持SFTP、FTP、SCP等安全传输协议，确保您的文件交换过程既快捷又安全。支持SSL加密，适配各类操作系统环境，尤其是Linux服务器。

- **绿色便携**: 所有配置存储于Data目录，无需复杂设置，携带方便。解压后直接运行`SecureCRTPortable.exe`或`SecureFXPortable.exe`即可开始使用。

- **汉化体验**: 完全中文界面，让国内用户操作更为直观，减少了语言障碍，提升了工作效率。

## 使用说明

1. **下载与解压**: 首先从提供的源处下载压缩包，并将其解压到您想要存放的本地目录。
   
2. **运行**: 解压后，进入软件目录，根据您的系统类型选择运行`SecureCRTPortable.exe`（对于SecureCRT）或`SecureFXPortable.exe`（对于SecureFX）。

3. **配置**: 第一次使用可能需要简单配置，如会话设置、偏好选项等，所有个性化配置将保留在Data目录中。

4. **享受安全连接**: 现在，您可以轻松地连接到远程服务器，执行管理任务或是安全地传输文件了。
