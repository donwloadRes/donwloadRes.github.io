# Android USB 设备读写与控制示例

## 项目简介

本项目是一个开源的 Android USB 读写与控制示例，旨在帮助开发者从多个 USB 设备中选择一个指定的设备，并实现对外设的控制通信。该示例已经在实际项目中经过测试，性能稳定，是公司项目中的重要组件。

## 主要功能

1. **多设备选择**：可以从多个连接的 USB 设备中选择指定的设备进行通信。
2. **性能稳定**：经过实际项目测试，性能稳定可靠。
3. **设备过滤**：示例中已经实现了指定设备的过滤功能，使用时请根据实际情况修改为自己的 USB 设备名称。

## 使用说明

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **导入项目**：
   将项目导入到 Android Studio 中，并根据需要修改代码。

3. **修改设备名称**：
   在代码中找到设备过滤的部分，将设备名称修改为你要控制的 USB 设备的名称。

4. **运行测试**：
   运行项目，确保设备选择和通信功能正常。
