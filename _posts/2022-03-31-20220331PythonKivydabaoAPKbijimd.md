# Python Kivy 打包 APK 笔记

## 简介

本资源文件提供了关于如何使用 Python 和 Kivy 框架将 Python 应用程序打包成 Android APK 文件的详细笔记。通过本笔记，您可以了解从环境配置到最终打包的完整流程，帮助您顺利将 Python 应用发布到 Android 平台。

## 内容概述

1. **安装 Python 开发环境**
   - 使用 pip 安装 Kivy 及相关依赖
   - 安装 PyCharm 作为开发工具

2. **虚拟机配置**
   - 下载并安装 VirtualBox 虚拟机
   - 导入 Kivy 提供的 Linux 虚拟镜像系统文件
   - 设置共享文件夹以便在 Windows 和虚拟机之间传输文件

3. **打包 APK 文件**
   - 在虚拟机中配置 buildozer 工具
   - 初始化 buildozer 配置文件
   - 编辑配置选项以适应您的应用需求
   - 运行打包命令生成 APK 文件

4. **注意事项**
   - 打包过程需要访问外网，buildozer 会自动下载 Android SDK 和 NDK 等依赖
   - 确保虚拟机和插件版本一致，避免报错

5. **工程示例**
   - 提供了一些示例代码和资源链接，帮助您更好地理解和实践

## 使用说明

1. **环境准备**
   - 按照笔记中的步骤安装 Python 和 Kivy 环境
   - 配置 VirtualBox 虚拟机并导入 Kivy 提供的虚拟镜像

2. **打包流程**
   - 在虚拟机中配置 buildozer 工具
   - 初始化 buildozer 配置文件并根据需要进行编辑
   - 运行打包命令生成 APK 文件

3. **常见问题**
   - 如果在打包过程中遇到问题，请参考笔记中的注意事项和解决方案
