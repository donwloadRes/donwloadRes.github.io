# STM32CubeG4：STM32G4系列专属的全功能软件包

## 概览

STM32CubeG4 是专为 STM32G4 系列微控制器设计的完整软件解决方案，由全球领先的半导体厂商意法半导体（STMicroelectronics）倾力打造。这一创新的软件包秉承了STM32Cube系列的核心理念，即显著降低开发者的工作负荷，缩短产品上市时间，并有效控制成本。它为STM32的广大用户群提供了全面、高效、标准化的开发环境。

## 软件包内容概要

- **HAL与LL驱动**：高度抽象化的HAL (Hardware Abstraction Layer) 和底层(LL) 驱动，支持跨STM32家族的最大程度可移植性。
  
- **CMSIS核心与设备**：遵循ARM标准的Core-Mangement Model Interface for Cortex Microcontroller Software Interface Standard (CMSIS)，包括内核和针对STM32G4特定硬件的设备支持。

- **中间件库**：集成了一系列通用的中间件组件，如RTOS实时操作系统、USB接口、FatFS文件系统、图形界面以及触摸感应库等，便于快速构建复杂功能的应用。

- **完整项目实例**：覆盖STM32G4所有官方支持的开发板（包括Nucleo开发板、评估板及发现套件），提供了基础示例、应用程序演示，涵盖从入门到高级应用的全方位示例代码。

## 核心优势

- **广泛的兼容性**：确保软件资产在不同STM32G4器件间的可重用性。
