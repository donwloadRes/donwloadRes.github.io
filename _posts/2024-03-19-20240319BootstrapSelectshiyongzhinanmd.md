# Bootstrap-Select 使用指南

## 简介

本资源文件提供了关于如何在项目中使用 `bootstrap-select` 的详细指南。`bootstrap-select` 是一个基于 Bootstrap 的下拉选择框插件，提供了丰富的功能和自定义选项，适用于需要复杂下拉选择框的 Web 应用。

## 主要内容

### 1. 下载地址

提供了 `bootstrap-select` 的下载链接，方便开发者快速获取所需的文件。

### 2. 引入文件

详细说明了如何在项目中引入 `bootstrap-select` 的 CSS 和 JS 文件，确保插件能够正常工作。

### 3. 基本使用

介绍了如何在 HTML 中使用 `bootstrap-select`，包括如何定义下拉选择框、设置默认选项、以及如何处理选项内容过长的情况。

### 4. 高级功能

涵盖了一些高级功能，如手动指定默认选中的项、调整下拉框的高度等。特别指出，下拉框高度的修改需要直接编辑 `bootstrap-select.min.js` 源码。

### 5. 示例代码

提供了一些示例代码，帮助开发者快速上手并理解 `bootstrap-select` 的使用方法。

## 适用场景

- 需要复杂下拉选择框的 Web 应用
- 希望提升用户体验的表单设计
- 需要自定义下拉框样式的项目

## 注意事项

- 确保在引入 `bootstrap-select` 之前已经引入了 Bootstrap 和 jQuery。
- 对于高级功能的修改，建议在理解源码的基础上进行，避免影响插件的正常使用。
