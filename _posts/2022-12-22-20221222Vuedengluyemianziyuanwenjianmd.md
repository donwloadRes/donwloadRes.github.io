## 使用说明

### 项目设置

1. **安装依赖**  
   在项目根目录下运行以下命令，安装项目所需的依赖包：
   ```bash
   npm install
   ```

2. **开发环境**  
   运行以下命令，启动开发服务器，并启用热重载功能：
   ```bash
   npm run serve
   ```

3. **生产环境**  
   运行以下命令，编译并最小化生产环境的代码：
   ```bash
   npm run build
   ```

4. **代码整理与修复**  
   运行以下命令，整理和修复代码中的问题：
   ```bash
   npm run lint
   ```
