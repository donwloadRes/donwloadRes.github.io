# Python 3.6.5 在 Windows 上的下载和安装指南

本文提供了在 Windows 操作系统上安装 Python 3.6.5 的详细步骤，包括下载、安装、配置和验证。Python 3.6.5 是 Python 编程语言的一次重要更新，增加了许多新特性和优化，适合初学者和专业开发者使用。

## 一、Python 3.6.5 下载

Python 3.6.5 可以从官方网站或百度网盘下载。建议从官方网站下载以确保文件的完整性和安全性。

## 二、Python 3.6.5 安装

1. 找到下载好的 `.exe` 安装包，双击运行。
2. 在安装界面中，勾选“Add Python 3.6 to PATH”选项，以便将 Python 添加到系统环境变量中。
3. 可以选择默认安装（Install Now）或自定义安装（Customize installation），自定义安装可以更改安装路径。
4. 根据提示完成安装过程。

## 三、Python 3.6.5 配置

安装完成后，需要配置系统环境变量：
1. 在系统环境变量“path”中，新增 Python 的安装路径和 Scripts 目录路径。
2. 这一步是为了使用 Python 默认的 pip 工具。

## 四、Python 3.6.5 验证

1. 打开命令提示符（CMD）。
2. 输入 `python -V` 命令，查看 Python 版本号。
3. 输入 `python` 命令，进入 Python 交互模式，尝试输出 `print('hello world')`，确认安装成功。

## 五、注意事项

- Python 3.6.5 与之前的版本在语法上略有不同，建议参考官方文档或相关教程进行学习。
- 安装过程中遇到问题，可以参考官方文档或社区论坛寻求帮助。

通过以上步骤，您应该能够成功在 Windows 上安装并配置 Python 3.6.5。希望这篇指南对您有所帮助！
