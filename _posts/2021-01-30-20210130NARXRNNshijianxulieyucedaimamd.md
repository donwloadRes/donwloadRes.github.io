# NARX-RNN时间序列预测代码

## 简介

本仓库提供了一套完整的NARX-RNN（非线性自回归外生模型）时间序列预测代码，包括数据集、训练、预测以及预测效果对比。无论你是时间序列分析的新手还是有一定经验的研究者，这套代码都能帮助你快速上手并进行实践。

## 功能特点

- **数据集**：包含示例数据集，可直接用于训练和预测。
- **训练**：提供完整的训练脚本，支持自定义参数调整。
- **预测**：训练完成后，可进行时间序列预测。
- **预测效果对比**：提供预测结果与实际数据的对比分析，帮助你评估模型性能。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   cd your-repo-directory
   ```

2. **替换数据集**：
   - 将你的数据集替换到`data`目录下。
   - 确保数据格式与示例数据集一致。

3. **运行代码**：
   - 直接运行`train.py`进行模型训练。
   - 运行`predict.py`进行时间序列预测。
   - 运行`compare.py`进行预测效果对比。

4. **自定义参数**：
   - 在相应脚本中调整模型参数，以适应你的数据集和需求。
