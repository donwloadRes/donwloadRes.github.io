# GCC-C++ 5.3.1 离线安装包 (aarch64)

## 资源文件描述

在国产化环境中，标麒麟操作系统中安装软件时，可能会提示不支持C++编译环境。系统中只能查找到`gcc`，但未找到`g++`和`c++`，导致无法进行C++程序的编译。为了解决这一问题，需要重新安装`gcc-c++`。然而，在离线环境中无法通过`yum`命令进行安装，因此我整理了适用于`aarch64`架构的`gcc-c++-5.3.1`离线安装包（`aarch64.rpm`），供大家参考使用。

## 使用说明

1. **下载文件**：请下载本仓库中的`gcc-c++-5.3.1-离线安装aarch64.rpm`文件。

2. **安装步骤**：
   - 在终端中进入文件所在目录。
   - 执行以下命令进行安装：
     ```bash
     rpm -ivh gcc-c++-5.3.1-离线安装aarch64.rpm
     ```

3. **验证安装**：
   - 安装完成后，可以通过以下命令验证`gcc-c++`是否安装成功：
     ```bash
     g++ --version
     ```
   - 如果安装成功，系统将显示`g++`的版本信息。

## 注意事项

- 本安装包仅适用于`aarch64`架构的系统。
- 请确保在安装前系统中已安装了`gcc`，否则可能会导致依赖问题。
- 本安装包仅供离线环境使用，在线环境中建议通过`yum`或`dnf`等包管理工具进行安装。
