# Unity 2019 汉化指南

欢迎使用 Unity 2019 汉化资源包！本资源旨在帮助广大中文用户更顺畅地使用 Unity 2019 系列的开发环境。通过本指南，您可以轻松将您的 Unity 编辑器界面转换为简体中文，提升开发体验。

## 如何汉化

### 步骤一：下载汉化包
首先，您需要下载汉化包。虽然具体的链接在此文本中未直接提供，但通常您可以在相关论坛、社区找到适合 Unity 2019 版本的汉化文件，如 `.po` 文件，比如 `zh-cn.po` 或 `zh-hans.po`。

### 步骤二：创建或定位 `Localization` 文件夹
1. 找到您的 Unity 安装目录下的 `Editor\Data` 文件夹。
2. 若其中不存在 `Localization` 文件夹，您需手动创建一个。

### 步骤三：应用汉化包
1. 将下载好的汉化包（`.po` 文件）复制到新创建的 `Localization` 文件夹内。
   - 确保文件名正确，通常是表示中文的文件名，如 `zh-cn.po`。

### 步骤四：设置 Unity 首选项
1. 启动 Unity 编辑器。
2. 导航至 `Edit`（编辑）-> `Preferences`（首选项）。
3. 在首选项对话框中，找到 `Languages`（语言）部分。
4. 选择 `Chinese (Simplified)` 或相应简体中文选项。
5. 保存设置并重启 Unity。

### 注意事项
- 某些情况下，您可能需要根据 Unity 的具体版本调整汉化过程中的细节。
- 汉化包可能适用于特定版本的 Unity，请根据您的实际版本选用合适的汉化资源。
- 重启 Unity 是为了使汉化效果完全体现。

## 结论
通过以上步骤，您的 Unity 2019 开发环境将变为熟悉的中文界面，极大地方便了中文用户的操作与理解。记得，持续关注社区更新，以获得最新的汉化资源和支持。开发愉快！
