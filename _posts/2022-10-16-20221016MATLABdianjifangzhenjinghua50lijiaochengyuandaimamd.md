## 使用方法

1. **下载仓库**: 点击右上角的“Clone or download”按钮，选择“Download ZIP”下载整个仓库，或使用Git命令克隆仓库到本地。

   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **打开MATLAB**: 启动MATLAB软件，将下载的文件夹添加到MATLAB路径中。

3. **运行案例**: 打开任意一个案例的MATLAB脚本文件（.m文件），按照教程说明运行代码，观察仿真结果。
