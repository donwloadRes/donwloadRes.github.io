# Python数据分析实战之超市零售分析

本仓库提供了一个关于超市零售数据分析的实战资源，包含以下两个主要文件：

1. **数据文件**：`superstore_dataset2011-2015.csv`
   - 这是一个包含超市零售数据的CSV文件，涵盖了2011年至2015年的销售数据。

2. **Jupyter Notebook文件**：`python数据分析实战之超市零售分析.ipynb`
   - 这是一个基于Jupyter Notebook环境的Python数据分析实战文件，详细介绍了如何使用Python进行超市零售数据的分析。

## 使用方法

1. **下载文件**：
   - 将本仓库中的`superstore_dataset2011-2015.csv`和`python数据分析实战之超市零售分析.ipynb`两个文件下载到本地。

2. **放置文件**：
   - 将这两个文件放置在同一个文件夹中。

3. **打开Jupyter Notebook**：
   - 按住键盘上的`Shift`键，在文件夹的空白处点击鼠标右键，选择“在此处打开命令窗口”（Windows系统）或“在此处打开Shell窗口”（其他系统）。

4. **启动Jupyter Notebook**：
   - 在命令窗口或Shell窗口中输入以下命令并回车：
     ```
     jupyter notebook
     ```

5. **打开Notebook文件**：
   - 在浏览器中打开Jupyter Notebook界面后，点击`python数据分析实战之超市零售分析.ipynb`文件，即可开始学习超市零售数据的分析过程。

## 注意事项

- 确保你已经安装了Jupyter Notebook环境，如果没有安装，可以通过以下命令进行安装：
  ```
  pip install jupyter
  ```

- 在学习过程中，建议按照Notebook中的步骤逐步操作，以便更好地理解数据分析的每一个环节。
