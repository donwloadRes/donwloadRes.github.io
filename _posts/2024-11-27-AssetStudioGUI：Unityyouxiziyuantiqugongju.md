# AssetStudioGUI：Unity游戏资源提取工具

## 简介

AssetStudioGUI是一款强大的Unity游戏资源提取工具，专为动画制作人员设计。它能够从Unity开发的游戏中提取立绘和动画资源，并支持动画的导出。无论你是游戏开发者还是动画爱好者，AssetStudioGUI都能帮助你轻松获取所需的资源。

## 功能特点

- **资源提取**：支持从Unity游戏安装包中提取各种资源，包括模型、动画、音频、贴图等。
- **动画导出**：支持将提取的动画资源导出为常见的格式，方便后续的编辑和使用。
- **多版本支持**：兼容多个Unity版本，包括最新的Unity 2018.2。
- **用户友好**：界面简洁直观，操作简单，即使是初学者也能快速上手。

## 使用方法

1. **下载安装包**：首先，你需要下载目标游戏的安装包。确保该游戏是由Unity开发的。
2. **解压安装包**：将下载的游戏安装包进行解压，得到包含游戏资源的文件夹。
3. **加载资源**：打开AssetStudioGUI工具，选择`File -> Load folder`，导入刚刚解压出来的文件夹。
4. **导出素材**：在AssetStudioGUI中，你可以通过两种方式导出素材：
   - `Model -> Export All Assets`：将模型以fbx文件导出。
   - `Export -> All Assets`：导出所有素材，工具会自动将素材放入对应的文件夹。

## 注意事项

- 请确保你下载的游戏安装包是合法的，遵守相关版权规定。
- 在使用AssetStudioGUI时，建议备份原始文件，以防操作失误导致数据丢失。
