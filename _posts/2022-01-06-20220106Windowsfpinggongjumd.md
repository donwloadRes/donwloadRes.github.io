# Windows fping工具

## 简介

`windows-fping工具` 是一个用于在Windows操作系统上同时ping多个主机的实用工具。通过该工具，您可以轻松地批量检测多个主机的网络连通性，提高网络故障排查的效率。

## 功能特点

- **批量Ping**: 支持同时ping多个主机，快速获取多个主机的网络状态。
- **简单易用**: 操作简单，无需复杂的配置，适合网络管理员和普通用户使用。
- **跨平台**: 虽然主要针对Windows系统设计，但也可以通过适当的配置在其他操作系统上使用。

## 使用方法

1. **下载资源文件**: 从本仓库下载`windows-fping工具`的资源文件。
2. **解压文件**: 将下载的压缩包解压到您选择的目录中。
3. **运行工具**: 打开命令行工具（如CMD或PowerShell），导航到解压后的目录，运行`fping.exe`。
4. **输入主机列表**: 在命令行中输入您要ping的主机列表，多个主机之间用空格分隔。
5. **查看结果**: 工具将显示每个主机的ping结果，包括是否连通以及延迟时间。

## 示例

```bash
fping.exe 192.168.1.1 192.168.1.2 192.168.1.3
```
