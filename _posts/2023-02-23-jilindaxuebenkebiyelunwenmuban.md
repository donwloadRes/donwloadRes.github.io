# 吉林大学本科毕业论文模板

## 简介

本仓库提供吉林大学本科毕业论文的LaTeX模板，旨在帮助吉林大学的本科生更方便地撰写和排版毕业论文。该模板遵循吉林大学本科毕业论文的格式要求，涵盖了论文的各个部分，包括封面、摘要、目录、正文、参考文献等。

## 使用说明

1. **下载模板**：
   - 点击仓库页面右上角的“Code”按钮，选择“Download ZIP”下载整个模板文件。
   - 解压下载的ZIP文件，进入解压后的文件夹。

2. **编辑论文**：
   - 使用LaTeX编辑器（如TeXstudio、Overleaf等）打开`main.tex`文件。
   - 根据模板中的注释和示例，填写你的论文内容。

3. **编译生成PDF**：
   - 在LaTeX编辑器中编译`main.tex`文件，生成最终的PDF格式的毕业论文。

4. **自定义设置**：
   - 模板中提供了一些自定义选项，如字体、页边距等，可以根据需要进行调整。

## 文件结构

```
吉林大学本科毕业论文模板/
├── main.tex          # 主文件，包含论文的整体结构
├── references.bib    # BibTeX格式的参考文献文件

└── README.md         # 本文件
```
