# **.NET CORE SAP RFC 接口调用资源（Windows、Linux）**

## 资源概览

本资源库提供了一套全面的组件，用于在 .NET CORE 项目中调用 SAP 的 RFC 接口。通过使用这些资源，您可以高效地集成 SAP 系统，扩展您的应用程序的功能。

**核心组件：**

- **SAP 接口组件：** 专为 .NET CORE 项目设计的组件，可与 SAP RFC 接口进行交互。
- **SapNwRfc 库：** 提供了基本的连接和数据操作功能，简化了 SAP RFC 接口调用。

**开发环境要求：**

- .NET CORE 开发环境
- 引用 SapNwRfc 库
- 安装 SAP NetWeaver RFC Library 7.50 SDK

**使用说明：**

SapNwRfc 库提供了详细的文档和示例，指导您完成使用过程。通过利用此库，您可以轻松地：

- 建立与 SAP 系统的安全连接
- 执行 RFC 调用并处理响应
- 访问和处理 SAP 数据

**SAP NetWeaver RFC Library 7.50 SDK：**

该 SDK 提供了适用于 Windows 和 Linux 平台的组件，为 SAP RFC 接口调用提供了稳定的底层基础设施。它包含以下关键元素：

- **RFC 客户端库：** 用于与 SAP 系统建立连接并发送 RFC 请求。
- **RFC 服务器库：** 用于从 SAP 系统接收并处理 RFC 请求。
