# FT232RL Win7/Win10 驱动程序

## 简介
本仓库提供了适用于 Windows 7 和 Windows 10 操作系统的 FT232RL 驱动程序。FT232RL 是一款常用的 USB 转串口芯片，广泛应用于各种开发板和设备中。通过安装此驱动程序，您可以确保您的设备在 Windows 系统上正常工作。

## 文件列表
- `FT232RL_Driver_Win7.zip` - 适用于 Windows 7 的驱动程序压缩包
- `FT232RL_Driver_Win10.zip` - 适用于 Windows 10 的驱动程序压缩包

## 安装指南
1. 根据您的操作系统版本（Windows 7 或 Windows 10），下载相应的驱动程序压缩包。
2. 解压缩下载的文件到任意目录。
3. 打开设备管理器（可以通过在开始菜单中搜索“设备管理器”来找到）。
4. 找到未识别的 FT232RL 设备（通常在“其他设备”或“端口(COM 和 LPT)”下）。
5. 右键点击未识别的设备，选择“更新驱动程序软件”。
6. 选择“浏览计算机以查找驱动程序软件”。
7. 选择解压缩后的驱动程序目录，然后点击“下一步”进行安装。
8. 按照提示完成驱动程序的安装。

## 注意事项
- 请确保下载的驱动程序与您的操作系统版本相匹配。
- 在安装驱动程序之前，请断开 FT232RL 设备与计算机的连接，安装完成后再重新连接。
- 如果遇到安装问题，请参考压缩包内的安装说明或联系技术支持。
