# JDK 17 下载与安装指南（Windows）

本仓库提供了一个详细的指南，帮助您在Windows系统上下载和安装JDK 17。JDK 17是Java开发工具包的一个版本，适用于开发和运行Java应用程序。

## 内容概述

1. **下载JDK 17**
   - 从Oracle官网下载JDK 17的安装包。
   - 使用百度网盘或迅雷下载JDK 17的安装包。

2. **安装JDK 17**
   - 下载完成后，双击安装程序进行安装。
   - 可以选择默认安装路径或自定义安装路径。
   - 安装完成后，点击“关闭”按钮。

3. **配置环境变量**
   - 打开系统环境变量配置界面。
   - 新建JAVA_HOME变量，变量值为JDK的安装路径。
   - 编辑PATH变量，添加JDK的bin目录路径。
   - 配置完成后，打开命令提示符窗口，输入`java -version`验证安装是否成功。

## 安装步骤详解

### 1. 下载JDK 17

您可以从以下几个途径下载JDK 17的安装包：
- **Oracle官网**：访问Oracle官网，找到JDK 17的下载页面，选择适合您操作系统的版本进行下载。
- **百度网盘**：使用提供的百度网盘链接和提取码下载JDK 17的安装包。
- **迅雷下载**：使用提供的迅雷下载链接和提取码下载JDK 17的安装包。

### 2. 安装JDK 17

下载完成后，双击下载的安装程序，按照以下步骤进行安装：
- 点击“下一步”开始安装。
- 选择安装路径（默认路径为C盘，可点击“更改”选择其他路径）。
- 安装完成后，点击“关闭”按钮。

### 3. 配置环境变量

安装完成后，需要配置系统环境变量，以便在命令行中使用Java命令。
- 打开“此电脑”，右键点击“属性”，然后点击“高级系统设置”。
- 在系统属性对话框中，点击“环境变量”按钮。
- 在系统变量部分，新建一个名为`JAVA_HOME`的变量，变量值为JDK的安装路径（例如：`C:\Program Files\Java\jdk-17`）。
- 编辑`PATH`变量，添加`%JAVA_HOME%\bin`路径。
- 配置完成后，打开命令提示符窗口，输入`java -version`验证安装是否成功。

## 验证安装

在命令提示符窗口中输入以下命令，验证JDK 17是否安装成功：
```
java -version
```
如果显示JDK 17的版本信息，说明安装成功。

通过本指南，您应该能够顺利地在Windows系统上下载和安装JDK 17，并配置好相应的环境变量。如果在安装过程中遇到任何问题，请参考相关文档或寻求帮助。
