# Windows Server 2019 简体中文语言包

本仓库提供Windows Server 2019的简体中文语言包下载。该语言包可以帮助用户将Windows Server 2019的界面语言切换为简体中文，方便中文用户使用。

## 使用方法

1. 下载本仓库中的语言包文件。
2. 在Windows Server 2019系统中，打开“运行”窗口（按下Win + R键）。
3. 在“运行”窗口中输入`lpksetup`，然后按下回车键。
4. 在弹出的“安装或卸载显示语言”窗口中，选择“安装显示语言”。
5. 点击“浏览”按钮，选择你下载的语言包文件。
6. 按照提示完成语言包的安装。
7. 安装完成后，重新启动系统，界面语言将会切换为简体中文。

## 注意事项

- 请确保你下载的语言包与你的Windows Server 2019版本相匹配。
- 安装语言包后，系统可能需要重新启动才能生效。
- 如果在安装过程中遇到任何问题，请参考Windows官方文档或联系技术支持。
