# 基于LibVLCSharp的视频文件播放Demo

## 项目描述

本项目是一个基于LibVLCSharp开发的视频文件播放Demo。该Demo实现了视频播放的基本功能，包括播放、暂停、快进、快退和停止。项目使用Visual Studio 2019进行开发，可以直接编译和测试。

## 功能介绍

- **播放**：加载并播放视频文件。
- **暂停**：暂停当前播放的视频。
- **快进**：快速前进视频播放进度。
- **快退**：快速后退视频播放进度。
- **停止**：停止播放并重置视频进度。

## 使用说明

1. **环境要求**：
   - Visual Studio 2019
   - .NET Framework 或 .NET Core

2. **编译与运行**：
   - 打开项目解决方案文件（.sln）。
   - 在Visual Studio 2019中加载解决方案。
   - 点击“生成”菜单中的“生成解决方案”进行编译。
   - 编译成功后，点击“调试”菜单中的“开始调试”或“开始执行（不调试）”运行程序。

3. **操作说明**：
   - 运行程序后，界面将显示视频播放控件。
   - 点击“播放”按钮加载并播放视频文件。
   - 点击“暂停”按钮暂停播放。
   - 点击“快进”按钮快速前进视频进度。
   - 点击“快退”按钮快速后退视频进度。
   - 点击“停止”按钮停止播放并重置视频进度。

## 注意事项

- 确保已安装LibVLCSharp库及其依赖项。
- 视频文件路径需正确配置，以便程序能够正确加载和播放视频。
