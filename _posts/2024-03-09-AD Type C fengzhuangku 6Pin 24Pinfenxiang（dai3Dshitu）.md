# AD Type C 封装库 6Pin 24Pin分享（带3D视图）

## 介绍

本仓库提供了一个AD（Altium Designer）封装库，包含了6Pin和24Pin的Type C接口封装，并附带3D视图。这些封装库可以用于电子设计自动化（EDA）软件Altium Designer中，方便用户进行电路设计和布局。

## 资源详情

- **6Pin Type C 封装**：这是我个人在某个项目中使用过的封装，由我自己绘制。
- **24Pin Type C 封装**：这个封装来自互联网，由于技术有限，可能不提供技术支持。
