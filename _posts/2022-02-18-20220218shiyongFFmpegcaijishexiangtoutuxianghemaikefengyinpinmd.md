# 使用FFmpeg采集摄像头图像和麦克风音频

本仓库提供了一个使用FFmpeg API采集摄像头图像和麦克风音频的资源文件。通过该资源文件，您可以实现以下功能：

- 采集摄像头图像并进行预览。
- 采集麦克风音频。
- 将采集到的图像和音频编码后保存成文件。
- 实现视音频同步机制。

为了更好地理解代码的实现原理，建议您阅读我在CSDN博客上相关的一篇文章：[使用FFmpeg采集摄像头图像和麦克风音频](https://blog.csdn.net/zhoubotong2012/article/details/79338093)。

## 使用说明

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   ```

2. **编译代码**：
   根据您的开发环境，使用相应的编译工具编译代码。

3. **运行程序**：
   运行编译后的可执行文件，程序将自动采集摄像头图像和麦克风音频，并保存成文件。
