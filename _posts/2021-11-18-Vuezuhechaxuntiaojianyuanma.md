## 使用步骤

1. **安装依赖**  
   在项目根目录下运行以下命令，安装所需的依赖包：
   ```bash
   npm install
   ```

2. **运行项目**  
   安装完成后，运行以下命令启动开发服务器：
   ```bash
   npm run dev
   ```

3. **访问项目**  
   项目启动成功后，打开浏览器访问以下地址：
   ```
   http://localhost:8090/#/
   ```
