# FFmpeg + Nginx 程序包

## 简介

本仓库提供了一个基于 FFmpeg 和 Nginx 的程序包，用于实现 RTSP 协议摄像头的远程播放功能。通过该程序包，您可以轻松地将 RTSP 流媒体数据转换为 HTTP 流媒体数据，并通过 Nginx 进行分发，从而实现远程播放。

## 功能特点

- **RTSP 转 HTTP 流媒体**：将 RTSP 协议的摄像头视频流转换为 HTTP 流媒体格式，方便在浏览器或其他支持 HTTP 流的设备上播放。
- **Nginx 流媒体服务器**：利用 Nginx 作为流媒体服务器，高效分发视频流，支持多用户同时访问。
- **易于部署**：程序包已经预先配置好 FFmpeg 和 Nginx，用户只需简单配置即可快速部署。

## 使用方法

1. **下载程序包**：从本仓库下载 `ffmpeg+nginx程序包`。
2. **解压文件**：将下载的压缩包解压到您的服务器或本地环境中。
3. **配置 FFmpeg 和 Nginx**：根据您的摄像头地址和端口，修改 FFmpeg 的配置文件，指定 RTSP 流的源地址。
4. **启动服务**：运行 Nginx 和 FFmpeg 服务，开始流媒体转换和分发。
5. **访问视频流**：通过浏览器或其他支持 HTTP 流的客户端，访问 Nginx 提供的流媒体地址，即可观看远程摄像头视频。

## 注意事项

- 请确保您的服务器或设备具备足够的带宽和处理能力，以支持视频流的实时转换和分发。
- 在配置 FFmpeg 和 Nginx 时，请根据您的实际网络环境和摄像头参数进行调整。
- 本程序包仅供学习和测试使用，请勿用于商业用途。
