# 解决Windows 11家庭版无法远程问题（无需任何升级即可mstsc）

## 简介

本资源文件旨在帮助Windows 11家庭版用户解决无法使用远程桌面（mstsc）的问题。通过使用本资源文件，您无需升级操作系统即可启用远程桌面功能。

## 使用方法

1. **下载资源文件**：
   - 下载本仓库中的资源文件，解压到任意目录。

2. **安装RDPWrap**：
   - 进入解压后的目录，右击以管理员身份运行“install.bat”。

3. **配置防火墙**：
   - 按下 `Win + R`，输入“desk.cpl”回车，点击“隐私和安全性”，然后点击“Windows 安全中心”。
   - 点击“打开 Windows 安全中心”，再点击“防火墙和网络保护”，最后点击“高级设置”。
   - 在左侧点击“入站规则”，在右侧点击“新建规则”。

4. **修改配置文件**：
   - 修改 `C:\Program Files\RDP Wrapper` 目录下的 `rdpwrap.ini` 配置文件。修改前请以管理员身份运行 `cmd` 停止远程服务。
   - 根据您的系统版本选择合适的配置进行修改，参考GitHub上的配置文件。

5. **启动远程服务**：
   - 以管理员身份运行 `cmd`，输入命令启动远程服务。
   - 运行 `RDPConf.exe` 查看配置是否成功，应显示三个绿色标志。

6. **测试远程连接**：
   - 使用 `mstsc` 进行远程连接测试，确保一切正常。

## 注意事项

- 本方法适用于Windows 11家庭版，也适用于Windows 10家庭版。
- 使用第三方工具时请注意安全风险。
