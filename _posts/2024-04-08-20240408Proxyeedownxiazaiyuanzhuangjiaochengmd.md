# Proxyee-down 下载与安装教程

Proxyee-down 是一个开源的 HTTP 下载工具，由 monkeyWie 在 GitHub 上开发。它通过本地 HTTP 代理服务器的方式嗅探下载请求，支持多连接分块下载，能够显著提升下载速度。本文将详细介绍 Proxyee-down 的下载与安装步骤，帮助用户快速上手使用。

## 版本选择

Proxyee-down 提供了多个版本，包括 2.x 和 3.x 版本。考虑到安装成本和稳定性，本文推荐使用 2.54 版本，这是 2.x 系列的最后一个版本，也是最稳定的版本。

## 安装步骤

### 前期准备

1. 确保你已经安装了 Chrome 浏览器。

### 安装 Proxyee-down

1. 下载并解压 `proxyee_down.zip` 文件。
2. 运行解压后的 `proxyee_down2.54` 目录下的 `proxyee-down.exe` 文件。

### 配置 Chrome 浏览器

1. 打开 Chrome 浏览器，将 `SwitchyOmega_Chromium.crx` 拖入浏览器中。
2. 安装完成后，Chrome 右上角会出现一个图标，点击该图标，选择“选项”。
3. 在选项页面中，点击左侧的“导入导出文件”，然后选择“从备份文件中恢复”，导入 `OmegaOptions.bak` 文件。

### 使用 Proxyee-down

1. 打开 Chrome 浏览器，点击右上角的圆圈图标，选择情景模式为 `proxyee-down`。
2. 进入百度云网页版（pan.baidu.com），选择要下载的文件，点击下载。
3. 此时会调用 Proxyee-down，你将体验到满速下载的百度云文件。

## 注意事项

- 分段数保持默认即可，大多数情况下，分段数过大反而会降低下载速度。
- 浏览其他网页时，请将情景模式调回“直接连接”。

## 常见问题

- **Proxyee-down 有的文件下载失败，有的文件下载成功怎么办？**
  - 尝试重新下载，如果仍然失败，可以考虑开通会员。

- **Proxyee-down 不能下载怎么办？**
  - 请关闭其他下载工具，如百度网盘客户端。

- **备份文件找不到怎么办？**
  - 解压下载的 `Proxyee_down2.54.zip` 文件，里面包含 `OmegaOptions.bak` 文件。
