# 省市区三级联动城市数据JS文件

## 简介

本仓库提供了一个用于实现省市区三级联动的城市数据JS文件。该文件可以直接复制并放入`citys.js`文件中使用，文件名可以根据需要自定义。

## 使用方法

1. 下载本仓库中的`citys.js`文件。
2. 将下载的文件复制到你的项目目录中。
3. 根据需要自定义文件名。
4. 在你的项目中引入该JS文件，即可使用其中的省市区三级联动数据。

## 数据格式

该JS文件包含了全国各省、市、区的详细数据，数据格式如下：

```javascript
export default [
  {
    id: 1,
    name: '北京市',
    city: [
      {
        id: 1,
        name: '北京市',
        district: [
          { id: 1, name: '东城区' },
          // 其他区数据...
        ]
      }
    ]
  },
  // 其他省份数据...
]
```

## 注意事项

- 该数据文件适用于前端开发中需要实现省市区三级联动的场景。
- 数据文件中的数据为静态数据，如需动态更新，请根据实际需求进行调整。
