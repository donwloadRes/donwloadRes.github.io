# Kali Linux 上安装 Burp Suite 的详细指南

本文档详细介绍了如何在 Kali Linux 上安装 Burp Suite 专业版，包括安装 JDK 环境、设置启动快捷方式和桌面快捷键。通过本指南，您可以顺利地在 Kali Linux 系统中运行 Burp Suite。

## 目录

1. [安装 JDK 环境](#安装-jdk-环境)
2. [安装 Burp Suite](#安装-burp-suite)
3. [设置启动快捷方式](#设置启动快捷方式)
4. [设置桌面快捷键](#设置桌面快捷键)

## 安装 JDK 环境

首先，使用 root 账户登录并安装 JDK。确保您的系统已经配置了 Java 环境，否则 Burp Suite 可能无法正常运行。

## 安装 Burp Suite

将 Burp Suite 文件移动到 `/usr/bin` 目录，并通过 `burp-loader-keygen.jar` 激活。确保文件路径正确，并且权限设置无误。

## 设置启动快捷方式

创建启动快捷方式，以便在命令行中快速启动 Burp Suite。您可以通过编辑启动脚本来实现这一点。

## 设置桌面快捷键

为了方便使用，您可以设置桌面图标，使得 Burp Suite 可以通过双击图标直接启动。确保桌面快捷方式指向正确的启动脚本。

通过以上步骤，您应该能够在 Kali Linux 上成功安装并运行 Burp Suite 专业版。如果在安装过程中遇到任何问题，请参考相关文档或社区支持。
