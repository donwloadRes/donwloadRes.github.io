# 任务栏透明 TranslucentTB 2024.1

## 简介
TranslucentTB 是一个开源项目，旨在使 Windows 任务栏在 Windows 10 和 Windows 11 上半透明/透明。该工具轻量级，使用几 MB RAM，几乎不使用 CPU，是一个非常实用的任务栏美化工具。

## 功能特点
- **任务栏透明化**：使 Windows 任务栏在 Windows 10 和 Windows 11 上半透明/透明。
- **任务栏颜色设置**：提供任务栏颜色设置功能，用户可以根据个人喜好调整任务栏颜色。
- **动态模式**：支持动态模式下设置任务栏透明度，用户可以根据不同的使用场景调整任务栏的透明效果。

## 安装与使用
1. **下载安装包**：根据您的操作系统版本（Windows 10 或 Windows 11）下载相应的安装包。
2. **安装步骤**：
   - Windows 11：可使用便携式安装包。
   - Windows 10：只能使用 msixbundle 安装包。
3. **自启动设置**：
   - 按住 `Win + R` 打开“运行”对话框，输入 `shell:startup` 并回车，打开“用户启动文件夹”。
   - 右击 `TranslucentTB.exe`，选择“创建快捷方式”。
   - 将快捷方式移入第一步打开的文件夹即可实现自启动。
4. **开机即全透明效果**：
   - 在开机自启文件夹里找到快捷方式，右击选择“属性”进入。
   - 找到“目标”，在最后面先加一个空格，然后输入 `--transparent`。
   - 点击“确认”关闭窗口即可。

## 注意事项
- 该工具仅适用于 Windows 10 和 Windows 11 系统。
- 在 Windows 11 系统中，可能会出现一些适配问题，如任务栏顶部的黑线等。
