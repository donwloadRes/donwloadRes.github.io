# PHP+MySQL 实现数据库增删改查，学生信息管理系统

## 项目简介

本项目是一个基于PHP和MySQL的学生信息管理系统，实现了对学生信息的增删改查功能。通过该系统，用户可以方便地管理学生信息，包括添加新学生、删除学生信息、修改学生信息以及查询学生信息。

## 功能特点

- **添加学生信息**：用户可以通过表单输入学生的详细信息，包括姓名、性别、年龄、学历、工资、奖金和籍贯，并将这些信息保存到数据库中。
- **删除学生信息**：用户可以选择删除某个学生的信息，系统会从数据库中移除该学生的记录。
- **修改学生信息**：用户可以修改已有的学生信息，包括姓名、性别、年龄、学历、工资、奖金和籍贯，并更新到数据库中。
- **查询学生信息**：用户可以查看所有学生的信息列表，系统会从数据库中读取并显示所有学生的详细信息。

## 技术栈

- **PHP**：用于处理服务器端的逻辑和与数据库的交互。
- **MySQL**：用于存储和管理学生信息的数据库。
- **HTML/CSS**：用于构建用户界面和样式。
- **JavaScript**：用于处理前端的交互逻辑，如删除确认提示。

## 运行环境

- Apache 2.4.41
- PHP 7.3.8
- MySQL 5.7

## 安装与使用

1. **克隆项目**：将项目克隆到本地。
2. **配置数据库**：在MySQL中创建一个名为`studb`的数据库，并导入项目中的SQL文件以创建学生信息表。
3. **配置连接**：在`conn.php`文件中配置数据库连接信息，包括数据库主机、用户名和密码。
4. **运行项目**：将项目部署到Apache服务器上，并通过浏览器访问`index.php`文件以启动学生信息管理系统。

## 注意事项

- 确保服务器环境已正确配置，包括Apache、PHP和MySQL的安装与配置。
- 在修改或删除学生信息时，请谨慎操作，以免误删或误改重要数据。
