# Notepad++ JSON Viewer 插件下载

## 介绍

本仓库提供了一个用于 Notepad++ 的插件——JSON Viewer。该插件旨在增强 Notepad++ 对 JSON 文件的处理能力，使其能够更方便地查看和编辑 JSON 数据。

## 插件功能

- **JSON 格式化**：自动格式化 JSON 数据，使其更易于阅读。
- **语法高亮**：为 JSON 文件提供语法高亮显示，便于区分不同类型的数据。
- **错误检查**：自动检测 JSON 文件中的语法错误，并提供相应的提示。

## 安装方法

1. 下载本仓库中的 `JSONViewer.dll` 文件。
2. 打开 Notepad++，进入 `插件` 菜单，选择 `插件管理器`。
3. 在插件管理器中，点击 `安装`，然后选择下载的 `JSONViewer.dll` 文件进行安装。
4. 安装完成后，重启 Notepad++ 即可使用 JSON Viewer 插件。

## 使用说明

1. 打开一个 JSON 文件。
2. 在 Notepad++ 中，点击 `插件` 菜单，选择 `JSON Viewer`，然后选择相应的功能（如格式化、语法高亮等）。
3. 插件会自动处理 JSON 文件，并显示处理结果。
