# unveril2.0：微信公众号解包与反编译工具

## 简介

本仓库提供了一个名为 `unveril2.0` 的工具，专门用于解包和反编译微信公众号的内容。通过使用该工具，您可以轻松地提取微信公众号中的资源文件，并进行进一步的分析和处理。

## 功能特点

- **解包功能**：能够高效地解包微信公众号中的资源文件，提取出其中的关键内容。
- **反编译功能**：支持对解包后的文件进行反编译，帮助您深入理解公众号的内部结构和逻辑。
- **开源免费**：本工具完全开源，您可以自由地使用、修改和分发。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/unveril2.0.git
   ```

2. **安装依赖**：
   ```bash
   cd unveril2.0
   pip install -r requirements.txt
   ```

3. **运行工具**：
   ```bash
   python unveril.py --input <微信公众号文件路径> --output <输出目录>
   ```

4. **查看结果**：
   解包和反编译后的文件将保存在指定的输出目录中，您可以进一步分析这些文件。
