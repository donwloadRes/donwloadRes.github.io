# Python比赛抽签软件

## 项目简介

本项目是一个用Python编写的简易抽签工具，专为各类比赛设计。通过此软件，组织者能够轻松实现参赛队伍或个人的随机分配，以确保抽签过程的公平性和透明性。用户可以便捷地利用Excel文件来管理参与者名单，灵活调整参与抽签的对象，大大简化了传统抽签流程。

## 主要功能

1. **随机抽签**：采用Python内置的随机模块，确保每次抽签结果的不确定性。
2. **Excel支持**：允许用户通过Excel格式导入参与者名单，便于管理和修改。
3. **简单易用**：用户界面友好，即使是对Python不熟悉的用户也能快速上手操作。
4. **公平透明**：源代码开放，可验证抽签的公正性，增强参与者信任。

## 技术栈

- Python 3.x
- `pandas`（用于处理Excel文件）
- `random`（用于抽签的随机选择）

## 快速使用指南

1. **安装依赖**：首先确保你的环境中已安装Python 3.x，然后通过pip安装必要的库，如使用命令`pip install pandas`。
2. **准备数据**：创建一个Excel文件（如`.xlsx`），列名可以是“姓名”或“队伍”，并填入参与者信息。
3. **运行程序**：执行主脚本，按照提示导入Excel文件并开始抽签。
4. **查看结果**：抽签完成后，软件会显示抽签结果，也可考虑将结果保存至文件或屏幕打印。
