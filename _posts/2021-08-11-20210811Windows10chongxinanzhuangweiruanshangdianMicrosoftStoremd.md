# Windows 10 重新安装微软商店 Microsoft Store

本资源文件提供了在Windows 10系统中重新安装微软商店（Microsoft Store）的方法。微软商店是Windows 10系统中用于下载和更新应用程序的重要平台，但在某些情况下，用户可能会遇到微软商店无法正常使用的问题，如无法打开、无法更新或崩溃等。通过本资源文件，您可以轻松地重新安装微软商店，恢复其正常功能。

## 使用方法

1. **下载资源文件**：
   - 下载本资源文件中提供的微软商店安装包。

2. **手动安装**：
   - 将下载的文件全部放入一个文件夹中，确保路径中没有中文和空格。

     ```
     Add-AppxPackage *
     ```
   - 按下回车键，系统将开始安装微软商店。

3. **完成安装**：
   - 安装完成后，您可以在开始菜单中找到并打开微软商店。
