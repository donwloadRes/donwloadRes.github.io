# C# WinForm 开发简易画图工具

## 项目简介

本仓库提供了一个使用 C# 和 WinForm 技术开发的简易画图工具。该工具允许用户在 Windows 窗体应用程序中进行基本的绘图操作，适合初学者学习和实践 C# WinForm 开发。

## 功能特点

- **基本绘图功能**：支持绘制直线、矩形、椭圆等基本图形。
- **颜色选择**：用户可以选择不同的颜色进行绘图。
- **画笔粗细调整**：支持调整画笔的粗细，以绘制不同宽度的线条。
- **保存与加载**：可以将绘制的图形保存为图片文件，并支持加载已保存的图片进行编辑。

## 使用说明

1. **下载资源**：从本仓库下载资源文件。
2. **打开项目**：使用 Visual Studio 或其他支持 C# 的 IDE 打开项目文件。
3. **编译运行**：编译并运行项目，启动简易画图工具。
4. **开始绘图**：在工具界面中选择绘图工具、颜色和画笔粗细，开始绘制图形。
5. **保存与加载**：绘制完成后，可以将图形保存为图片文件，或加载已保存的图片进行编辑。

## 适用人群

- C# 初学者：通过本项目可以学习 WinForm 的基本开发流程和控件使用。
- WinForm 开发者：可以参考本项目的代码，扩展和优化自己的 WinForm 应用程序。
