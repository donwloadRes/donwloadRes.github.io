# 大学网页设计前端Web大作业

## 项目描述

本项目是一个大学网页设计前端Web大作业，使用HTML、CSS、JavaScript和jQuery技术实现的静态网站。该资源文件包含了网站的所有源代码和相关资源，适合学习和参考。

## 项目结构

```
├── index.html          # 主页HTML文件
├── css/                # CSS样式文件夹
│   ├── style.css       # 主要样式文件
│   └── ...             # 其他样式文件
├── js/                 # JavaScript脚本文件夹
│   ├── main.js         # 主要JavaScript文件
│   └── ...             # 其他JavaScript文件
├── images/             # 图片资源文件夹
│   ├── logo.png        # 网站Logo
│   └── ...             # 其他图片资源
├── README.md           # 项目说明文件
└── ...                 # 其他相关文件
```

## 使用技术

- **HTML**: 用于构建网页的基本结构。
- **CSS**: 用于网页的样式设计，包括布局、颜色、字体等。
- **JavaScript**: 用于实现网页的动态效果和交互功能。
- **jQuery**: 一个快速、简洁的JavaScript库，简化了HTML文档遍历、事件处理、动画和Ajax交互。

## 如何使用

1. **克隆仓库**: 使用以下命令克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **打开项目**: 进入项目目录，双击`index.html`文件即可在浏览器中查看网站。

3. **修改和扩展**: 你可以根据需要修改HTML、CSS和JavaScript文件，添加新的功能或样式。
