# PyQt5桌面软件实战教程

本仓库提供了一个使用Python的PyQt5库创建桌面应用的实战教程资源文件。通过本教程，您将学习如何使用PyQt5库创建一个带有计时功能的简单桌面应用。

## 内容概述

1. **演示效果**  
   展示了应用的实际运行效果。

2. **准备工作**  
   - 使用pip下载所需包：`pyqt5`
   - 下载可视化UI工具：QT Designer
   - 可视化UI工具汉化
   - PyCharm配置QT Designer和PyUIC

3. **代码实现**  
   - 文件结构
   - UI设计代码
   - 功能实现代码

4. **打包**  
   使用pyinstaller将应用打包成可执行文件。

## 使用说明

1. **安装依赖**  
   使用pip安装所需的PyQt5库：
   ```bash
   pip install pyqt5
   ```

2. **下载QT Designer**  
   下载并安装QT Designer，用于设计应用的UI界面。

3. **配置PyCharm**  
   在PyCharm中配置QT Designer和PyUIC，以便将UI文件转换为Python代码。

4. **运行代码**  
   运行`main.py`文件，启动应用。

5. **打包应用**  
   使用pyinstaller将应用打包成可执行文件：
   ```bash
   pyinstaller --onefile main.py
   ```

## 注意事项

- 确保已安装Python环境。
- 配置QT Designer和PyUIC时，注意路径设置。
- 打包应用时，确保pyinstaller已正确配置。

通过本教程，您将掌握使用PyQt5创建桌面应用的基本流程，并能够进一步扩展和优化应用功能。
