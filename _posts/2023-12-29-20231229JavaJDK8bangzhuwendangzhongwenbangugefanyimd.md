# Java JDK 8 帮助文档（中文版，谷歌翻译）

## 简介

本仓库提供了一份基于谷歌翻译的 Java JDK 1.8 API 中文帮助文档。该文档旨在帮助中文用户更方便地理解和使用 Java JDK 1.8 的相关功能。由于文档是通过谷歌翻译自动生成的，因此翻译的准确性无法完全保证，建议用户在使用时结合英文原版文档进行参考。

## 文件说明

- **文件名**: `java_jdk_8_help_doc_cn_google_translate.chm`
- **格式**: CHM（Compiled HTML Help）
- **语言**: 中文（谷歌翻译）

## 使用说明

1. **下载文件**: 点击仓库中的文件链接进行下载。
2. **打开文件**: 下载完成后，双击 `.chm` 文件以打开帮助文档。
3. **解除锁定**: 如果文件打开后显示空白页面，请右键点击文件，选择“属性”，然后在“常规”选项卡中找到“解除锁定”按钮并点击，以解除文件的锁定状态。

- **文件锁定**: 如果文件打开后显示空白页面，请按照上述步骤解除文件锁定。
