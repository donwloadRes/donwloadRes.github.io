# PyQt5实现登录、主界面跳转

## 项目简介

本仓库提供了一个使用PyQt5实现的登录界面与主界面跳转的示例代码。通过该资源文件，您可以学习如何使用PyQt5创建一个简单的登录界面，并在用户成功登录后跳转到主界面。

## 功能描述

- **登录界面**：用户可以在登录界面输入用户名和密码。
- **主界面**：用户成功登录后，系统会自动跳转到主界面。
- **界面跳转**：通过PyQt5的信号与槽机制，实现登录界面与主界面之间的无缝跳转。

## 使用说明

1. **下载资源文件**：请从本仓库下载相关资源文件。
2. **安装依赖**：确保您的环境中已安装PyQt5库。如果没有安装，可以使用以下命令进行安装：
   ```bash
   pip install PyQt5
   ```
3. **运行代码**：运行下载的Python脚本，即可启动登录界面。
4. **登录与跳转**：在登录界面输入正确的用户名和密码后，系统将自动跳转到主界面。

## 注意事项

- 本示例代码仅供学习参考，实际应用中请根据需求进行适当修改。
- 请确保输入的用户名和密码与代码中设定的值一致，以便成功登录。
