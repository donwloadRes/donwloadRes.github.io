# Keil uVision5 添加 Compiler Version 5 教程

本资源文件提供了如何在 Keil uVision5 中添加 Compiler Version 5 的详细教程。通过本教程，您可以轻松地将 Compiler Version 5 集成到您的 Keil uVision5 开发环境中，以便进行嵌入式开发。

1. **下载 Compiler Version 5 安装包**
   - 提供了官方下载地址和百度云盘下载链接。
   - 根据您的电脑系统下载对应的压缩包。

2. **安装 Compiler Version 5**
   - 将安装包解压到本地。
   - 找到 Keil MDK 的 ARM 文件夹，并在其中创建 ARMCC 文件夹。
   - 运行解压文件中的 ARMCompiler506_b960.msi 文件，按照提示完成安装。

3. **配置 Compiler Version 5**
   - 在 Keil uVision5 中进行相关配置，确保 Compiler Version 5 被正确识别和使用。

## 使用说明

1. **下载安装包**
   - 根据您的需求选择官方下载或百度云盘下载。
   - 下载完成后，解压安装包。

2. **安装步骤**
   - 打开解压后的文件夹，找到 ARMCompiler506_b960.msi 文件并运行。
   - 按照安装向导的提示，选择 ARMCC 文件夹作为安装路径，完成安装。

3. **配置 Keil uVision5**
   - 打开 Keil uVision5，进入项目设置。
