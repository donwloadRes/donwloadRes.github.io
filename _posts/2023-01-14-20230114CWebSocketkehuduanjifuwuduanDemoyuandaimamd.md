# C# WebSocket 客户端及服务端 Demo 源代码

本仓库提供了一个完整的 C# WebSocket 客户端及服务端 Demo 源代码，帮助开发者快速理解和实现 WebSocket 通信。服务端采用 SocketTCP 通讯，WebSocket 通讯与传统通讯的主要差异在于消息解析部分，具体实现细节请参考服务端源代码。

## 资源文件描述

- **标题**: C# WebSocket 客户端及服务端 Demo 源代码
- **描述**: C# WebSocket 客户端及服务端 Demo，服务端采用 SocketTCP 通讯，WebSocket 通讯与传统通讯的差异就是多了一些消息解析，详情见服务端源代码！

## 使用说明

1. **下载源代码**: 克隆或下载本仓库的源代码到本地。
2. **打开项目**: 使用 Visual Studio 或其他 C# 开发工具打开项目。
3. **运行服务端**: 首先运行服务端程序，确保服务端正常启动并监听指定端口。
4. **运行客户端**: 启动客户端程序，连接到服务端，进行 WebSocket 通信测试。

## 注意事项

- 服务端和客户端的通信依赖于 SocketTCP 协议，WebSocket 通信的实现主要在于消息的解析和封装。
- 请确保服务端和客户端的配置正确，包括 IP 地址和端口号。
- 代码中包含了详细的注释，方便开发者理解和修改。
