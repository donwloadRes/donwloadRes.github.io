# Kafka 可视化管理工具 (Kafka Tool - Windows 64位)

**简介**

本仓库提供了一款高效且用户友好的Kafka可视化工具，专为Windows 64位系统设计。这款工具极大地简化了Apache Kafka的管理和监控工作流程，对于开发者和运维人员来说，是一个不可或缺的辅助工具。

**功能特性**

1. **直观查看Topic内容**: 通过其图形化界面，您可以轻松浏览和理解Kafka集群中的Topic数据，无需复杂的命令行操作。
   
2. **自定义数据展示**: 用户可以根据需要定制消息的显示格式，提高数据审查的效率和便利性。

3. **Topic管理**: 支持创建和删除Topic的功能，简化了Kafka的管理工作，使环境配置更为便捷。

4. **模拟消息发送**: 不仅限于查看，此工具还允许您模拟发送消息到指定的Topic，便于测试和验证Kafka的配置与消息处理逻辑。

**安装与使用**

- 下载提供的`Kafka Tool_windows_64bit.rar`压缩包，并解压到您的本地目录。
- 运行解压后的应用程序文件，根据界面提示进行相应设置，连接到您的Kafka集群。
- 利用该工具的强大功能，管理您的Kafka环境，无论是监控Topic状态，还是进行消息的生产和消费测试。

**重要说明**

- 请确保您的系统已正确安装Java运行环境(JRE)，因为Kafka Tool基于Java开发。
- 在首次使用时，准确配置Kafka集群的地址和端口，以保证能够成功连接。
