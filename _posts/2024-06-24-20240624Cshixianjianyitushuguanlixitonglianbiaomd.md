# C++实现简易图书管理系统（链表）

## 简介

本仓库提供了一个用C++实现的简易图书管理系统的源代码，该系统基于链表数据结构。源代码中包含了测试数据，方便用户进行测试和验证。

## 功能特点

- 使用链表数据结构管理图书信息。
- 支持图书的添加、删除、查询和显示功能。
- 包含测试数据，便于快速上手和测试。

## 使用方法

1. **克隆仓库**
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **编译源代码**
   ```bash
   cd your-repo-directory
   g++ -o book_management main.cpp
   ```

3. **运行程序**
   ```bash
   ./book_management
   ```

## 文件结构

- `main.cpp`: 主程序文件，包含图书管理系统的核心逻辑。
- `data.txt`: 测试数据文件，包含预设的图书信息。
