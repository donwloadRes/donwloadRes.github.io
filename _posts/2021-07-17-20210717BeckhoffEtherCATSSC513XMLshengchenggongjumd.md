# Beckhoff EtherCAT SSC 5.13 XML生成工具

## 简介

本仓库提供了一个用于生成Beckhoff EtherCAT SSC 5.13 XML文件的工具。该工具在5.12版本的基础上进行了源码优化，提升了生成效率和稳定性。

## 功能特点

- **优化源码**：在5.12版本的基础上进行了源码优化，提升了工具的性能和稳定性。
- **生成XML文件**：能够快速生成符合Beckhoff EtherCAT SSC 5.13标准的XML文件。
- **开源项目**：本项目完全开源，欢迎社区贡献和反馈。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/beckhoff-ethercat-ssc-5.13.git
   ```

2. **安装依赖**：
   ```bash
   cd beckhoff-ethercat-ssc-5.13
   pip install -r requirements.txt
   ```

3. **运行工具**：
   ```bash
   python generate_xml.py
   ```

4. **生成XML文件**：
   工具将自动生成符合Beckhoff EtherCAT SSC 5.13标准的XML文件，并保存到指定目录。
