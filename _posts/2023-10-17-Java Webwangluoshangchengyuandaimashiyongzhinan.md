# Java Web网络商城源代码使用指南

## 项目简介
本项目提供了一个无框架的Java Web网络商城的源代码，包括前后台页面。该项目使用Java、HTML、CSS、JavaScript和Servlet技术，适合作为结课作业或毕业设计使用。

## 功能模块
### 前台页面
1. **商品页面**：实现商品搜索、分类浏览及分页处理。
2. **商品详情页面**：点击商品查看详细信息。
3. **登录页面**：用户登录后可将商品加入购物车。
4. **购物车页面**：管理购物车中的商品，支持商品数量调整和总价计算。

### 后台页面
1. **登录页面**：管理员登录后台管理系统。
2. **商品类型查询**：通过关键字查询商品类型。
3. **商品类型操作**：增删改查商品类型。
4. **商品类型首页设置**：设置网络商城首页的商品类型。
5. **商品查询**：根据商品类型和关键词查询商品。
6. **商品增删改查**：管理商品信息，包括文字、图片等。

## 环境配置
1. **JDK**：安装并配置JDK 8。
2. **Eclipse**：安装并配置Eclipse IDE。
3. **Tomcat**：安装并配置Tomcat服务器。
4. **MySQL**：安装并配置MySQL数据库。
5. **数据库连接工具**：推荐使用Navicat等工具。

## 项目导入步骤
1. 下载项目源代码并解压。
2. 将数据库SQL文件导入MySQL。
3. 在Eclipse中导入项目。
4. 修改`JDBCUtils`中的MySQL账号和密码。
5. 运行项目，右键选择`Run As` -> `Run on Server`。

## 注意事项
- 确保所有环境配置正确。
- 导入数据库时，注意选择正确的数据库连接工具。
- 运行项目前，确保Tomcat服务器已正确配置并启动。
