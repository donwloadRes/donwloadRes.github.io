# Windows Server 2016 安装 .NET 3.5 解决方案

本仓库提供了一个资源文件，专门用于解决在 Windows Server 2016 上无法安装 .NET 3.5 的问题。经过亲测，该方法有效且操作简单。

- **文件标题**: Windows Server 2016，解决不能安装.NET 3.5问题,亲测有效
- **文件描述**: 将文件解压放到某个目录下，在安装.NET 3.5时，在“确认”一步，点击指定备用源路径，在路径里填写你解压得到的sxs文件夹所在路径，我放在c盘下，所以填写c:\sxs\

## 使用步骤

1. **下载资源文件**: 从本仓库下载提供的资源文件。
2. **解压文件**: 将下载的文件解压到你选择的目录下，例如 `C:\`。
3. **安装.NET 3.5**: 在 Windows Server 2016 上尝试安装 .NET 3.5。
4. **指定备用源路径**: 在安装过程中，当系统提示“确认”时，点击“指定备用源路径”。
5. **填写路径**: 在路径框中填写你解压得到的 `sxs` 文件夹所在路径，例如 `C:\sxs\`。
6. **完成安装**: 按照提示完成安装过程。
