# AsmDude-vs2022 汇编语法高亮插件

## 简介
AsmDude 是一款专为 Visual Studio 2022 设计的汇编语言插件，提供了强大的语法高亮和代码辅助功能。该插件支持汇编源文件和反汇编窗口的语法高亮，适用于 Visual Studio 2015、2017 和 2019 版本。通过 AsmDude，您可以更轻松地编写和阅读汇编代码，提高开发效率。

## 功能特点
- **语法高亮**：为汇编代码提供丰富的语法高亮，使代码更易于阅读和理解。
- **代码辅助**：提供代码补全、代码折叠等功能，帮助您更高效地编写汇编代码。
- **文档链接**：方便地访问相关文档，快速查找指令和函数的详细信息。
- **反汇编窗口支持**：在反汇编窗口中同样提供语法高亮和代码辅助功能。

## 安装方法
1. 下载最新版本的 AsmDude.vsix 安装文件（版本号：v1.9.6.14）。
2. 打开 Visual Studio 2022。
3. 在 Visual Studio 中选择“扩展” -> “管理扩展”。
4. 点击“从 VSIX 安装”，选择下载的 AsmDude.vsix 文件进行安装。
5. 安装完成后，重启 Visual Studio 2022。

## 使用说明
安装完成后，AsmDude 将自动为您的汇编源文件和反汇编窗口提供语法高亮和代码辅助功能。您可以在编写汇编代码时享受更便捷的开发体验。

## 其他资源
如果您觉得汇编语言过于复杂，但仍希望访问特定的机器指令，可以考虑使用 Intrinsics-Dude 插件。
