# 树莓派 4B 安装 Windows 资源文件介绍

本资源文件旨在帮助用户在树莓派 4B 上安装 Windows 操作系统。通过提供的工具和步骤，用户可以轻松地将 Windows 系统安装到树莓派 4B 上，并享受与传统 PC 相似的使用体验。

## 资源内容

- **Windows on Raspberry (WoR) 工具**：用于简化树莓派安装 Windows 的过程。
- **镜像文件脚本**：包含适用于树莓派 4B 的 Windows 镜像文件。
- **详细安装指南**：提供从硬件准备到系统安装的完整步骤。

## 安装步骤概述

1. **硬件准备**：
   - 至少32GB可用空间的SD卡或外置硬盘。
   - 树莓派 4B（建议最低RAM 2GB）。
   - 一台可以正常运行的Windows电脑。

2. **下载资源**：
   - 下载 WoR 工具。
   - 下载合适的镜像文件脚本。

3. **安装系统**：
   - 使用 WoR 工具选择设备和存储介质。
   - 选择下载的镜像文件并开始写入系统。
   - 将SD卡或外置硬盘连接到树莓派 4B 并启动系统。

4. **系统配置**：
   - 根据提示完成系统安装过程。
   - 配置系统内存限制（如果需要）。

## 注意事项

- 确保路由器支持科学上网，以便顺利完成安装。
- 目前仅支持网线连接，WiFi暂时无法使用。
- 蓝牙功能可以正常使用。

## 适用版本

- Windows 10/11 的 arm64 版本。
- 版本号建议在 19014~251xx 之间。

## 其他说明

- 本资源文件适用于树莓派 4B，其他型号可能不兼容。
- 安装过程中可能需要重启数次，请耐心等待。
