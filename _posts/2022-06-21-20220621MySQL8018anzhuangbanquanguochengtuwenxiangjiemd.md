# MySQL 8.0.18 安装版全过程图文详解

本文提供了一步一步的MySQL安装指南，包括从官方网站或第三方平台下载安装包，选择合适的安装目录，自定义安装过程，设置安装路径和开发者选项，在安装配置阶段，文章详细解释了如何定义密码，选择服务启动选项，并测试了使用Navicat连接MySQL服务的步骤。

## 目录

1. 下载 MySQL
2. 准备安装 MySQL
3. 安装 MySQL
4. 安装配置
5. 使用 Navicat 连接测试

### 1. 下载 MySQL

从官方网站或第三方平台下载MySQL 8.0.18安装包。选择对应版本下载即可。

### 2. 准备安装 MySQL

准备一个安装目录文件夹，之后安装位置选择这里。安装后的结构如图所示。

### 3. 安装 MySQL

双击安装包开始安装。点击 Back 后选择自定义安装，作为开发者所需安装的产品功能便会被选中。选择自定义安装路径，重复操作直至完成。

### 4. 安装配置

按图标注操作即可，需求不同可以自行修改。定义密码，选择服务启动选项，点击 Execute 完成配置。

### 5. 使用 Navicat 连接测试

使用 Navicat 连接 MySQL 服务，测试连接是否正常。

通过以上步骤，您可以顺利完成MySQL 8.0.18的安装与配置。
