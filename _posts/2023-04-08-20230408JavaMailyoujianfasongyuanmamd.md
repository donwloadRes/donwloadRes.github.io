# Java Mail 邮件发送源码

## 简介

本仓库提供了一个用 Java Mail 发送邮件的源码，支持多种邮局，并且支持单发和群发功能。无论你是需要发送单封邮件还是批量发送邮件，这个源码都能满足你的需求。

## 功能特点

- **支持多种邮局**：源码支持多种常见的邮件服务器，如 Gmail、Outlook、QQ邮箱等。
- **单发和群发**：你可以选择发送单封邮件，也可以一次性发送多封邮件。
- **简单易用**：源码结构清晰，易于理解和使用，适合初学者和有经验的开发者。

## 使用方法

1. **下载源码**：从本仓库下载源码文件。
2. **配置邮件服务器**：根据你使用的邮件服务器，修改源码中的配置信息。
3. **编译运行**：使用 Java 编译器编译源码，并运行程序。
4. **发送邮件**：根据需要选择单发或群发功能，输入收件人地址和邮件内容，即可发送邮件。

## 注意事项

- 请确保你使用的邮件服务器允许通过第三方应用发送邮件。
- 在群发邮件时，请注意邮件服务器的限制，避免触发反垃圾邮件机制。
