# Android 端 WebSocket 简单 Demo

## 项目描述

本项目是一个简单的 Android WebSocket 示例，展示了如何在 Android 应用中使用 WebSocket 进行基本的通信。通过本项目，你可以学习到如何添加依赖和权限、创建 WebSocketClient 客户端类、建立连接并发送消息。

## 功能介绍

1. **添加依赖和权限**：
   - 在 `build.gradle` 文件中添加 WebSocket 相关的依赖。
   - 在 `AndroidManifest.xml` 文件中添加必要的网络权限。

2. **创建 WebSocketClient 客户端类**：
   - 实现一个自定义的 `WebSocketClient` 类，用于处理 WebSocket 的连接、消息接收和发送。

3. **建立连接并发送消息**：
   - 在应用中建立 WebSocket 连接，并发送简单的消息。
   - 处理接收到的消息并展示在界面上。

## 使用说明

1. **克隆项目**：
   ```bash
   git clone https://github.com/yourusername/WebSocketDemo.git
   ```

2. **导入项目**：
   - 使用 Android Studio 打开项目。

3. **运行项目**：
   - 连接 Android 设备或启动模拟器。
   - 点击运行按钮，启动应用。

4. **测试 WebSocket 连接**：
   - 在应用中点击“连接”按钮，建立 WebSocket 连接。
   - 输入消息并点击“发送”按钮，测试消息的发送和接收。

## 注意事项

- 确保设备或模拟器已连接到互联网。
- 根据实际需求修改 WebSocket 服务器的地址和端口。
