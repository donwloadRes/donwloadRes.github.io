# g++ gcc编译器安装包及其依赖

## 资源描述

本仓库提供了一个包含g++和gcc编译器的安装包及其依赖的资源文件。该资源文件适用于需要安装gcc版本4.8.5的环境。

## 安装步骤

1. **下载资源文件**  
   下载本仓库中的`g++.zip`文件。

2. **解压缩文件**  
   使用以下命令解压缩下载的文件：
   ```bash
   unzip g++.zip
   ```

3. **进入解压后的目录**  
   进入解压后的目录：
   ```bash
   cd g++
   ```

4. **安装rpm包**  
   使用以下命令安装所有rpm包：
   ```bash
   rpm -Uvh *.rpm
   ```

5. **验证安装**  
   安装完成后，可以使用以下命令验证gcc版本：
   ```bash
   gcc --version
   ```
   预期输出为：
   ```
   gcc version 4.8.5 20150623 (Red Hat 4.8.5-11) (GCC)
   ```

## 注意事项

- 请确保在安装过程中有足够的权限。
- 如果在安装过程中遇到依赖问题，请确保所有依赖包都已正确安装。
