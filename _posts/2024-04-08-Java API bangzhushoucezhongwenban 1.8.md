# Java API 帮助手册中文版 1.8

## 简介
本仓库提供了一份详细的Java API帮助手册中文版1.8，旨在帮助开发者更方便地查阅和理解Java平台标准版的API规范。该文档根据网页内容生成目录，方便用户快速定位所需信息。

## 功能特点
- **目录生成**：文档自动生成目录，方便用户快速导航。
- **目录调整**：用户可以调整目录的宽度位置（左右），并可根据需要隐藏目录。
- **目录呼出**：目录生成后，页面左上角会有一个小蓝块，点击即可呼出目录。
- **目录重置**：目录的右下角有一个小灰块，点击后目录会重新在左边生成。
- **宽度调整**：目录的灰色条可以拖动，用户可以根据需要设置目录的宽度。

## 使用说明
1. **下载文档**：从本仓库下载Java API帮助手册中文版1.8。
2. **打开文档**：使用支持HTML的浏览器打开文档。
3. **浏览目录**：文档打开后，目录默认生成在左边，点击左上角的小蓝块可以呼出目录。
4. **调整目录**：如果需要调整目录的位置或宽度，可以通过拖动目录的灰色条来实现。
5. **隐藏目录**：如果需要隐藏目录，可以点击目录右下角的小灰块，目录会重新在左边生成。

## 注意事项
- 本文档为Java API帮助手册中文版1.8，适用于Java 1.8版本。
- 文档内容为网页格式，建议使用现代浏览器打开以获得最佳阅读体验。
