# Internet Download Manager (IDM) 6.42 Build 3中文版

## 简介
Internet Download Manager (IDM) 是一款功能强大的下载加速器，旨在为用户提供最快的下载速度。IDM 通常可以将下载速度提升至原来的500%，极大地提高了下载效率。该软件支持多种协议，包括代理服务器、FTP、HTTP、HTTPS等，并且能够无缝集成到主流浏览器中，自动处理下载任务。

## 主要功能
- **多线程下载**：IDM 使用多线程技术，显著提高下载速度。
- **自动恢复下载**：即使在下载过程中遇到中断或错误，IDM 也能自动恢复下载。
- **浏览器集成**：IDM 可以无缝集成到 Microsoft Internet Explorer、Netscape、MSN Explorer、AOL、Opera、Mozilla、Mozilla Firefox、Mozilla Firebird、Avant Browser、MyIE2 等主流浏览器中。
- **计划任务**：用户可以设定下载任务的时间计划，实现自动下载。
- **支持多种协议**：IDM 支持代理服务器、FTP、HTTP、HTTPS 等协议。
- **视频和音频内容处理**：IDM 能够自动捕获网页上的视频和音频内容，方便用户下载。

## 新版本特性
- **Windows 11 兼容性**：6.42 Build 3 版本增加了对 Windows 11 的兼容性。
- **网络播放器下载面板**：新增了 IDM 下载面板，方便用户下载网络播放器中的内容。
- **页面抓取器**：改进了页面抓取器功能，提升下载效率。
- **重新开发的计划程序**：优化了计划任务功能，使用更加便捷。
- **MMS 协议支持**：新增了对 MMS 协议的支持。
- **与 IE 11 集成**：增强了与 IE 11 及基于 IE 的浏览器的集成。
- **改进的下载引擎**：重新设计和增强了下载引擎，提升下载速度和稳定性。
- **高级集成**：与所有最新浏览器的独特高级集成，提升用户体验。
- **改进的工具栏**：改进了工具栏，增加了更多实用功能。

## 安装教程
1. 下载并解压安装包。
