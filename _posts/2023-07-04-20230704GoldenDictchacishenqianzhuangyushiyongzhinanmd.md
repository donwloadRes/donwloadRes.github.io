# GoldenDict 查词神器安装与使用指南

## 简介
GoldenDict 是一款功能强大的跨平台查词工具，支持 Windows、Mac 和 Linux 系统。它不仅免费、无广告，还支持多种词典格式，能够满足用户在生活、学习和工作中的查词需求。本文将详细介绍如何在不同操作系统上安装和使用 GoldenDict。

## 主要功能
- **跨平台支持**：适用于 Windows、Mac 和 Linux 系统。
- **多种词典格式**：支持多种词典格式，方便用户管理多个词典。
- **在线查词**：支持查维基百科、在线翻译等功能。
- **屏幕取词**：支持屏幕取词和划词释义，提高查词效率。
- **发音功能**：支持听取发音，帮助用户更好地学习语言。

## 安装步骤
### Windows
1. 下载最新版本的 GoldenDict 安装包。
2. 双击安装包，按照提示完成安装。

### macOS
1. 下载最新版本的 GoldenDict 安装包。
2. 双击安装包，按照提示完成安装。

### Linux
1. 打开终端，执行以下命令安装 GoldenDict：
   ```bash
   sudo apt install goldendict -y  # Ubuntu / Debian
   sudo yum install goldendict -y  # CentOS
   sudo dnf install goldendict -y  # Fedora
   ```

## 使用指南
### 基本操作
1. **打开 GoldenDict**：在 Windows 和 macOS 上，双击图标打开；在 Linux 上，可以使用快捷键 `Super（Win）` 键搜索 `gold` 并按 `Enter` 键打开。
2. **修改语言**：在菜单栏选择【编辑】>【首选项】，修改界面语言和显示风格。

### 管理词典
1. **下载词典**：选择合适的词典进行下载，并存放到指定位置。
2. **导入词典**：在菜单栏选择【编辑】>【词典】>【词典来源】>【文件】> 添加，选择词典位置并点击重新扫描。
3. **词典分组**：在【群组】选项卡中，将词典按类分组，方便检索。

### 查词功能
1. **复制查词**：选中需要翻译的词句，按住 `Ctrl` 再按两下 `C` 键，即可呼唤 GoldenDict 进行翻译。
2. **屏幕取词**：在特定场景下，如阅读生词较多的英文文章时，可以开启屏幕取词功能，提高查词效率。

### 在线查词与翻译
1. **添加在线词典**：在菜单栏选择【编辑】>【词典】>【词典来源】>【网站】> 添加，配置在线词典。
2. **使用搜索引擎**：可以添加搜索引擎，如谷歌、百度等，进行在线搜索。

### 文本翻译
1. **安装 translate-shell**：在终端中执行安装命令，配置 GoldenDict 进行文本翻译。
2. **使用 CopyTranslator**：安装并配置 CopyTranslator，提高文本翻译效率。
