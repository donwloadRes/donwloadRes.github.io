# VC++ 6.0 安装指南（解决 Win10 和 Win11 无法打开的问题）

本仓库提供了一个资源文件，用于解决在 Windows 10 和 Windows 11 系统上安装和运行 VC++ 6.0 时遇到的问题。以下是详细的安装步骤和注意事项。

## 安装步骤

### 1. 下载
- 下载本仓库提供的资源文件。

### 2. 安装
1. 解压下载的压缩包。
2. 运行安装程序。
3. 按照安装向导的提示，逐步完成安装。
4. 在安装过程中，可以选择更改安装路径（建议安装到非系统盘，如D盘）。
5. 点击“安装”并等待安装完成。

### 3. 更改兼容性
1. 右键点击桌面上的 VC++ 6.0 图标，选择“打开文件位置”。
2. 重命名安装目录中的可执行文件（例如，在文件名后添加一个字母或数字）。
3. 同步桌面快捷方式的目标名称。
4. 右键点击桌面快捷方式，选择“属性”，在“兼容性”选项卡中勾选“以兼容模式运行这个程序”和“以管理员身份运行此程序”，然后点击“应用”。

### 4. 打开
- 回到桌面，双击 VC++ 6.0 图标即可正常打开。如果遇到其他问题，请留言或私信交流。

## 注意事项
- 本安装指南适用于 Windows 10 和 Windows 11 系统。
- 安装过程中请确保以管理员身份运行安装程序。
- 如果遇到“dskquoui.dll”文件缺失的错误提示，请将安装光盘中的“DISK1\SETUP\DSKQUOUI”目录下的“DSKQUOUI.DL_”文件复制到Windows\System32目录下，并将其重命名为“DSKQUOUI.DLL”。

通过以上步骤，您应该能够成功在 Windows 10 和 Windows 11 系统上安装并运行 VC++ 6.0。
