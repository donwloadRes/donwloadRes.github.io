# QT 实现登陆页面跳转到主页面源码

## 简介

本仓库提供了一个使用 QT 框架实现的登陆页面跳转到主页面的源码。通过该源码，您可以学习如何在 QT 中实现简单的页面跳转功能，适用于初学者和需要快速实现类似功能的开发者。

## 功能描述

- **登陆页面**：用户输入用户名和密码进行登陆。
- **主页面**：登陆成功后跳转到主页面，显示欢迎信息。
- **错误处理**：如果用户名或密码错误，会提示错误信息。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **打开项目**：
   使用 QT Creator 打开项目文件（.pro）。

3. **编译运行**：
   编译并运行项目，即可看到登陆页面。输入正确的用户名和密码后，页面将跳转到主页面。

## 文件结构

- `main.cpp`：主程序入口。
- `loginwindow.cpp` 和 `loginwindow.h`：登陆页面的实现。
- `mainwindow.cpp` 和 `mainwindow.h`：主页面的实现。
- `ui_loginwindow.h` 和 `ui_mainwindow.h`：UI 文件的自动生成代码。

## 注意事项

- 本项目仅为示例代码，实际使用时请根据需求进行修改和扩展。
- 请确保您已经安装了 QT 开发环境。
