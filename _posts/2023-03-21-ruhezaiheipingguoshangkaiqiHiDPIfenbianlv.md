# 如何在黑苹果上开启HiDPI分辨率

## 简介

本资源文件提供了在黑苹果（Hackintosh）上开启HiDPI分辨率的详细教程和相关脚本。HiDPI（High Dots Per Inch）技术可以显著提升显示效果，使字体和图标在高分辨率屏幕上更加清晰和细腻。

## 使用方法

### 1. 准备工作

在开始之前，请确保你已经安装了黑苹果系统，并且具备基本的终端操作知识。你需要准备以下文件：

- HiDPI启用脚本 `enable-HiDPI.sh`
- RDM工具（可选）
- 更改原始分辨率工具 `SwitchResX`

### 2. 启用HiDPI

1. 打开终端，并导航到 `enable-HiDPI.sh` 所在的目录。
2. 赋予脚本执行权限：
   ```bash
   chmod +x enable-HiDPI.sh
   ```
3. 执行脚本：
   ```bash
   ./enable-HiDPI.sh
   ```
4. 脚本运行后，会提示你输入二倍缩放后的分辨率。根据你的屏幕尺寸和分辨率选择合适的值（例如，13寸1080p屏幕建议输入1600×900）。
5. 输入完毕后，输入 `0` 退出，并输入登录密码以完成操作。

### 3. 调整分辨率

1. 重启电脑后，打开 `SwitchResX` 工具。
2. 在侧边栏中找到你的显示器名称，点击后选择 `Custom Resolutions`。
3. 点击左下角的加号，添加一个新的分辨率配置。
4. 在上方选择 `Scaled resolution`，并将高度和宽度设置为命令行中设置的HiDPI的两倍。
5. 保存设置并退出。

### 4. 完成设置

再次重启电脑，选择设置好的分辨率即可。现在你可以使用RDM工具来调整分辨率，享受HiDPI带来的清晰显示效果。

## 注意事项

- 开启HiDPI会增加系统的渲染压力，但通常不会对性能产生显著影响。
- 如果你的显示器分辨率较低（如1080p），建议谨慎开启HiDPI，以免影响显示效果。
