# C/C++模拟银行排队叫号系统

## 项目简介

本项目是一个用C/C++语言编写的模拟银行排队叫号系统。该系统旨在模拟银行客户到达、排队、叫号以及业务办理的整个流程。通过随机生成客户到达时间和业务办理时长，系统能够动态展示客户在银行中的排队情况，并统计每个窗口的服务客户数量、客户最长的等待时间以及客户的平均等待时间。

## 功能特点

1. **客户到达模拟**：系统随机生成客户到达银行的时间和业务办理时长。
2. **排队叫号**：根据客户到达时间，系统为客户发放号码牌，并根据窗口空闲情况进行叫号。
3. **业务办理**：模拟客户在窗口办理业务的过程，记录办理时长。
4. **统计分析**：统计每个窗口服务客户数量、客户最长的等待时间以及客户的平均等待时间。
5. **动态展示**：程序运行时，实时输出客户到达、等待、办理业务的信息。

## 基本要求

1. 银行上午9点开门，下午5点关门，期间每个小时的客流量不超过35人。
2. 每个客户的基本信息包括到达银行时间和业务办理时长，这两项数据均由系统随机生成。
3. 程序运行时，由键盘输入银行的窗口数量，然后输出客户到达、等待、办理业务的信息。
4. 提示用户可以继续输入窗口数量，若输入0，程序结束。
5. 观察银行的窗口数量与接待客户总量与客户等待时间之间的关系。

## 输出样例

程序运行时，输出格式如下：

```
第几号客户几点到达，在等待了多少分钟后，在几号窗口办理业务，持续多少分钟。
若客户是提前离开的，对应的输出就是：第几号客户几点到达，提前离开。
```
