# HTML+CSS+JS网页设计期末课程大作业

## 项目简介

本项目是一个基于HTML、CSS和JavaScript的网页设计期末课程大作业。该项目旨在通过实际操作，帮助学生掌握前端开发技术，包括网页布局、样式设计和交互功能的实现。通过完成这个项目，学生能够深入理解HTML、CSS和JavaScript在网页设计中的应用，提升自己的前端开发能力。

## 项目内容

该项目包含以下几个主要部分：

1. **首页设计**：
   - 使用HTML和CSS进行页面布局，包括头部、导航栏、图片轮播、内容区域和底部版权区。
   - 使用JavaScript实现图片轮播的动画效果。

2. **游戏介绍页面**：
   - 展示游戏的基本信息和特色。
   - 使用HTML和CSS进行页面布局，确保信息清晰易读。

3. **游戏攻略页面**：
   - 提供游戏的攻略和技巧。
   - 使用HTML和CSS进行页面布局，确保攻略内容易于查找和阅读。

4. **手办商城页面**：
   - 展示游戏相关的手办商品。
   - 使用HTML和CSS进行页面布局，确保商品展示美观且易于浏览。

5. **问卷调查页面**：
   - 设计一个简单的问卷调查表单。
   - 使用HTML和CSS进行页面布局，确保表单易于填写。

6. **加入我们页面**：
   - 提供加入团队的联系方式和信息。
   - 使用HTML和CSS进行页面布局，确保信息清晰易懂。

## 技术栈

- **HTML**：用于构建网页的基本结构。
- **CSS**：用于网页的样式设计，包括布局、颜色、字体等。
- **JavaScript**：用于实现网页的交互功能，如图片轮播、表单验证等。

## 使用工具

- **EditPlus**：用于编写和编辑HTML、CSS和JavaScript代码。

## 项目效果

项目模仿了一个游戏官网的设计，通过实际操作展示了如何使用HTML、CSS和JavaScript进行网页设计。每个页面都经过精心布局和设计，确保用户体验良好。
