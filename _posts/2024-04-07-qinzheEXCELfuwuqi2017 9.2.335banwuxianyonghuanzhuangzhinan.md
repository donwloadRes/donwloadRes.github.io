# 勤哲EXCEL服务器2017 9.2.335版无限用户安装指南

## 资源简介
本资源包含勤哲EXCEL服务器2017的9.2.335版本，提供无限用户许可。该版本需配合SQL Server 2005数据库使用，适用于需要强大Excel数据管理和协作的企业或个人。特别注意，本版本若要实现全面功能，如手机APP、微信及WEB接口，需在两台计算机上完成特定配置，或利用单机加虚拟机环境。

## 安装步骤

### 主机安装（数据库与核心服务器）
1. **安装SQL Server 2005**。
2. **安装勤哲9.2.335版**，采用默认设置。
3. **注册**: 按照提供的步骤进行产品注册。
4. **数据库恢复**: 使用SQL Server Management Studio Express，根据压缩包内的“数据库还原说明”还原四个数据库文件。

### 副机安装（扩展功能服务器）
- **适用系统**: 推荐Windows 2003系统，其他系统可能需要额外配置。
- **安装勤哲2016版**，选择自定义安装，排除应用服务器和数据库选项。
- **配置扩展功能**: 根据“手机微信填报等说明.pdf”文档进行详细配置。

## 注册机与注意事项
- 注册机的使用方法详阅9.2补丁相关说明。
- **重要提示**: 主机主要用于数据存储，客户端操作应在其它配置的机器上执行，避免直接在主机上操作客户端或控制台以免引起不必要的问题。
- 关于客户端显示试用天数仅剩1天的问题，这是正常显示，实际使用不受影响，无需担心过期。

## 结语
通过遵循上述指南，您将能够成功搭建并配置勤哲EXCEL服务器，享受其强大的Excel数据处理与协同办公能力。请注意，在实施过程中仔细阅读每一步骤，并按照指示操作，确保系统稳定运行。如果您计划利用移动设备或其他高级功能，仔细配置副机至关重要。祝您安装顺利！
