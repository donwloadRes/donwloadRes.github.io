# C# 串口数据采集与实时显示工具

## 简介

本项目是一个实用的C#小工具，旨在通过串口与下位机设备进行通信，实时采集、显示并记录下位机采集的数据。界面设计灵感来源于Windows任务管理器，力求简洁直观，便于用户操作和监控数据。

## 功能特点

- **串口通信**：支持通过串口与下位机设备进行数据交互。
- **实时显示**：采集的数据会实时显示在界面上，方便用户即时查看。
- **数据记录**：采集的数据会被记录下来，便于后续分析和处理。
- **界面设计**：界面参考Windows任务管理器，简洁明了，易于操作。

## 使用说明

1. **配置串口**：在工具界面中选择正确的串口和波特率。
2. **启动采集**：点击“开始”按钮，工具将开始通过串口采集数据。
3. **实时监控**：采集的数据会实时显示在界面上，用户可以即时查看。
4. **数据记录**：采集的数据会被自动记录，用户可以导出或查看历史数据。

## 系统要求

- .NET Framework 4.5 或更高版本
- Windows 7 或更高版本
