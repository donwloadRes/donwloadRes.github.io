# PyQt5 和 PyQt5-sip 快速安装资源

本仓库提供了一个快速安装 PyQt5 和 PyQt5-sip 的资源文件，帮助你避免直接从网络下载时的缓慢速度。你可以通过以下步骤快速安装这两个库。

## 资源文件说明

- **文件名**: `pyqt5-<版本号>.whl` 和 `pyqt5_sip-<版本号>.whl`
- **描述**: 这些文件是 PyQt5 和 PyQt5-sip 的预编译二进制文件，可以直接通过 `pip install` 命令进行安装。

## 安装步骤

1. **下载资源文件**: 从本仓库中下载 `pyqt5-<版本号>.whl` 和 `pyqt5_sip-<版本号>.whl` 文件。

2. **安装 PyQt5**:
   ```bash
   pip install pyqt5-<版本号>.whl
   ```

3. **安装 PyQt5-sip**:
   ```bash
   pip install pyqt5_sip-<版本号>.whl
   ```

## 注意事项

- 请确保你的 Python 环境与下载的 `.whl` 文件版本兼容。
- 如果你在安装过程中遇到任何问题，请检查你的 Python 版本和操作系统是否与文件兼容。
