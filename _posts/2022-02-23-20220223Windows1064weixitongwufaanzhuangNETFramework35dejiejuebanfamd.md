# Windows10 64位系统无法安装 .NET Framework 3.5的解决办法

## 简介

本资源文件提供了一个解决Windows10 64位系统无法安装.NET Framework 3.5的方法。许多软件，如AutoCAD等，需要.NET Framework 3.5才能正常运行。然而，在全新安装的Windows10系统中，直接安装.NET Framework 3.5可能会遇到问题。本资源文件包含了一个zip工具包，帮助用户顺利安装.NET Framework 3.5。

## 使用方法

1. **下载并解压文件**：
   - 下载提供的zip工具包。
   - 解压下载的文件，解压后会看到一个名为【sxs】的文件夹。

2. **复制文件夹**：
   - 将解压后的【sxs】文件夹复制到C盘根目录（直接拖到C盘）。

3. **以管理员身份运行**：
   - 右键点击【sxs】文件夹中的NET Framework 3.5安装程序，选择“以管理员身份运行”。

4. **等待安装完成**：
   - 运行后会进入一个黑框，等待进度条达到100%，安装即完成。

5. **删除文件夹**：
   - 安装完成后，可以将之前复制到C盘的【sxs】文件夹删除。

## 注意事项

- 确保在安装过程中网络连接正常，以便系统能够下载必要的文件。
- 如果安装过程中遇到任何问题，可以参考提供的说明文件或重新尝试上述步骤。

## 适用范围

本方法适用于Windows10 64位系统，特别是那些在安装.NET Framework 3.5时遇到问题的用户。

## 其他解决方案

如果本方法无法解决问题，可以尝试以下其他解决方案：
- 检查系统更新，确保Windows10系统为最新版本。
- 使用Windows自带的“启用或关闭Windows功能”选项，尝试重新安装.NET Framework 3.5。
