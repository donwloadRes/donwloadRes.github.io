# Python 图片查找与点击工具

## 简介

本仓库提供了一个用于在Python中进行图片查找并点击的资源文件。通过使用该工具，您可以轻松地在屏幕上查找指定的图片，并在找到后自动点击该图片。

## 功能描述

该资源文件的核心功能是通过Python代码实现以下操作：

1. **图片查找**：在屏幕上查找指定的图片文件。
2. **自动点击**：在找到图片后，自动点击该图片。

## 使用示例

以下是资源文件中的示例代码片段，展示了如何使用该工具进行图片查找并点击：

```python
# 指定图片路径
imagePath = r'D:\图片.png'
print(imagePath)

# 创建ImageSearch对象
ImageSearchs = ImageSearch.ImageSearchs()

# 查找并点击图片
ImageSearchs.click(imagePath, 2, timeOut=10)
```

## 参数说明

- `imagePath`：指定要查找的图片文件路径。
- `2`：点击次数，表示点击图片的次数。
- `timeOut=10`：查找图片的超时时间，单位为秒。如果在指定时间内未找到图片，则停止查找。

## 注意事项

1. **图片路径**：请确保指定的图片路径正确无误，否则可能导致查找失败。
2. **超时时间**：根据实际情况调整超时时间，以确保在合理的时间内完成查找操作。
