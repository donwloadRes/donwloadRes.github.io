# Qt 绘制流程图示例程序——Diagram

## 简介

本仓库提供了一个基于Qt的绘制流程图示例程序——Diagram。该程序展示了如何使用Qt框架实现节点的添加以及节点之间的连线操作。通过这个示例，您可以学习到如何在Qt中创建自定义图形界面，并实现基本的图形编辑功能。

## 功能特性

- **节点添加**：用户可以通过简单的操作在画布上添加节点。
- **节点连线**：支持在不同节点之间创建连线，形成流程图的基本结构。
- **图形编辑**：提供了基本的图形编辑功能，如移动节点、调整连线等。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/Diagram.git
   ```

2. **打开项目**：
   使用Qt Creator打开项目文件（`.pro`文件）。

3. **编译运行**：
   在Qt Creator中编译并运行项目，即可看到示例程序的运行效果。

## 依赖项

- Qt 5.x 或更高版本
