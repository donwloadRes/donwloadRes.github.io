## 使用方法

1. **下载文件**：首先，下载本仓库中的`java-1.8.0-openjdk-1.8.0.402.b06-1.el7_9.x86_64.rpm`文件。

2. **解压文件**：将下载的文件解压到目标目录。

3. **安装升级包**：在解压后的目录中，执行以下命令进行安装：
   ```bash
   yum localinstall ./*.rpm
   ```

4. **验证安装**：安装完成后，可以通过以下命令验证Java版本：
   ```bash
   java -version
   ```
