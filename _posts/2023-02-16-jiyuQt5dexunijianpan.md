# 基于Qt5的虚拟键盘

## 简介
本仓库提供了一个基于Qt5.4.2的虚拟键盘插件制作工程源码。该虚拟键盘支持谷歌拼音输入法，适用于需要在嵌入式设备或桌面应用中集成虚拟键盘的开发者。

## 功能特点
- 基于Qt5.4.2开发，兼容性强。
- 支持谷歌拼音输入法，提供良好的中文输入体验。
- 可自定义键盘布局和样式。
- 适用于嵌入式设备和桌面应用。

## 使用方法
1. **克隆仓库**
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **编译项目**
   ```bash
   cd your-repo-directory
   qmake
   make
   ```

3. **运行虚拟键盘**
   ```bash
   ./your-virtual-keyboard
   ```

## 依赖项
- Qt5.4.2 或更高版本
- 谷歌拼音输入法（已集成在源码中）
