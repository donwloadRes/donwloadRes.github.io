- **直接调用**：代码结构清晰，可以直接集成到你的项目中，无需额外配置。

## 示例代码

以下是一个简单的示例，展示了如何使用本代码进行UTF8到GBK的转换：

```c
#include "utf8_gbk_converter.h"

int main() {
    char utf8_str[] = "这是一个UTF8编码的字符串";
    char gbk_str[100];

    utf8_to_gbk(utf8_str, gbk_str, sizeof(gbk_str));

    printf("GBK编码的字符串: %s\n", gbk_str);

    return 0;
}
```
