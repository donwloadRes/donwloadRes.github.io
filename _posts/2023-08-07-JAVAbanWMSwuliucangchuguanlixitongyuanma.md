# JAVA版WMS物流仓储管理系统源码

## 项目描述

本项目提供了一个基于JAVA开发的物流仓库管理系统（WMS）源码，支持自营和第三方物流仓储企业。该系统包含PDA端和Web端，经过多家公司上线运转后，为了降低物流仓储企业的信息化成本，针对有特殊信息化需求的企业，提供高性价比的实施服务。

## 技术架构

- **开发语言**: JAVA
- **技术架构**: 
  - SpringMVC
  - Hibernate
  - Minidao（类Mybatis）
  - Easyui（UI库）
  - Jquery
  - Boostrap
  - Ehcache
  - Redis
  - Ztree

## 适用范围

- 第三方物流仓储企业
- 自营仓储企业

## 技术特性

- **基于JAVA的WEB后台**
- **基于ANDROID开发的PDA系统**

## 功能特性

- 订单管理系统（OMS）
- 仓储管理系统（WMS）
- 计费管理系统（BMS）
- 现场作业系统（RF）
- 第三方接口模块

## 接口支持

- 已对接：
  - SAP ECC
  - SAP HANA 数据库
  - 用友U8
  - 百胜E3
  - UAS
- 对接自主研发ERP管理系统
- 增加进销存模块
- 增加BOM

## 使用说明

1. **克隆仓库**: 使用 `git clone` 命令克隆本仓库到本地。
2. **环境配置**: 根据项目要求配置JAVA开发环境及相关依赖。
3. **运行项目**: 按照项目文档中的说明启动Web端和PDA端。
