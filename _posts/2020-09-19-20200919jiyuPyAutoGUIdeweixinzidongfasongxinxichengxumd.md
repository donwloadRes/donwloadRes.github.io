# 基于PyAutoGUI的微信自动发送信息程序

## 简介

本项目是一个基于Python库PyAutoGUI编写的简单微信自动发送信息程序。该程序主要用于自动化微信消息发送，适用于需要频繁发送消息的场景，如与朋友互相刷屏发送表情等。

## 功能特点

- **自动发送消息**：程序能够自动打开微信，搜索指定联系人，并发送预设的消息内容。
- **支持多条消息发送**：可以将多条消息内容存储在文本文件中，程序会逐条发送。
- **自定义发送间隔**：用户可以设置消息发送的时间间隔，避免过于频繁的发送。

## 使用步骤

1. **安装PyAutoGUI**：
   在终端输入以下命令安装PyAutoGUI库：
   ```bash
   pip install pyautogui
   ```

2. **引入库**：
   在编写的Python文件中输入以下代码引入PyAutoGUI库：
   ```python
   import pyautogui
   ```

3. **编写关键代码**：
   参考文章中的代码示例，编写自动发送消息的逻辑。主要步骤包括：
   - 打开微信
   - 搜索联系人
   - 发送消息

4. **运行程序**：
   运行编写好的Python脚本，程序将自动执行消息发送任务。

## 注意事项

- 由于PyAutoGUI不支持中文输入，因此在发送中文消息时需要使用其他方法（如pyperclip库）进行复制粘贴。
- 请确保微信已登录且处于活动状态，以便程序能够正常操作。
