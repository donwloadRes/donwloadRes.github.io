# Unity3D 使用 Meth 实现温度云图

## 描述
本资源文件提供了一个在 Unity3D 中使用 Meth 实现温度云图的解决方案。网上已有一些使用 Shader 实现的温度云图，但大多数都是写死的，并且代码存在问题，无法正常显示效果。因此，本资源提供了一个使用 Meth 实现的温度云图，温度点位可以动态生成，行列数也可以动态创建，只需给定一个数值即可。

## 使用方法
1. 在 Unity3D 中创建一个空物体。
2. 将提供的代码附加到该空物体上。
3. 为该空物体添加 `MethRenderer` 和 `MethFilter` 组件。
4. 运行项目，即可看到温度云图效果。
