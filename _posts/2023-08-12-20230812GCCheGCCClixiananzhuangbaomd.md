# GCC 和 GCC-C++ 离线安装包

本仓库提供了一个从 mini 安装版 CentOS 7.5 提取出来的 GCC 和 GCC-C++ 离线安装包。该安装包适用于在没有网络连接的环境下安装 GCC 和 GCC-C++ 编译器。

## 资源文件描述

- **文件名**: gcc和gcc-c++离线安装包
- **文件内容**: 包含 GCC 和 GCC-C++ 的 RPM 安装包

## 使用方法

1. **下载文件**: 下载本仓库中的资源文件。
2. **解压文件**: 将下载的文件解压到服务器上的某个目录。
3. **进入目录**: 使用 `cd` 命令进入解压后的目录。
4. **安装 RPM 包**: 执行以下命令安装所有 RPM 包：
   ```bash
   rpm -Uvh *.rpm
   ```
5. **验证安装**: 安装完成后，运行以下命令验证 GCC 和 GCC-C++ 是否安装成功：
   ```bash
   gcc -v
   g++ -v
   ```
   如果命令输出显示版本信息，则表示安装成功。

## 注意事项

- 请确保服务器上没有安装过 GCC 和 GCC-C++，否则可能会导致安装冲突。
- 该安装包仅适用于 CentOS 7.5 系统，其他系统版本可能不兼容。

## 其他说明

- 该安装包是从 mini 安装版 CentOS 7.5 中提取的，适用于离线环境下的安装需求。
- 如果在安装过程中遇到任何问题，请检查系统环境或联系管理员。
