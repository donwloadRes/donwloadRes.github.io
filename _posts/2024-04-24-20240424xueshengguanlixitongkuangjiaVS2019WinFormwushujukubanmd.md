# 学生管理系统框架 (VS2019 + WinForm + 无数据库版)

## 项目简介

本项目提供了一个基于Visual Studio 2019和WinForm开发的学生管理系统框架，该框架不依赖于数据库，主要通过文本文件来存储和管理学生信息。项目包含两个主要界面：教师登录页面和学生管理界面。数据的主要来源是两个文本文件：`学生信息管理表.txt` 和 `教师密码.txt`。

## 功能模块

### 教师登录模块
- 教师通过输入用户名和密码进行登录。
- 用户名和密码存储在 `教师密码.txt` 文件中。
- 后期可采用MD5加密，或是直接换成SQL数据库的 + MD5加密的形式来处理登录。

### 学生成绩管理模块
- 学生信息管理界面允许教师查看和管理学生信息。
- 学生信息存储在 `学生信息管理表.txt` 文件中。

### 退出模块
- 提供退出系统的功能。

## 使用说明

1. **环境要求**：
   - Visual Studio 2019 或更高版本。
   - .NET Framework 4.5 或更高版本。

2. **文件结构**：
   - `学生信息管理表.txt`：存储学生信息。
   - `教师密码.txt`：存储教师登录信息。

3. **运行项目**：
   - 打开项目解决方案文件（.sln）。
   - 编译并运行项目。

4. **登录信息**：
   - 默认教师用户名：`admin`
   - 默认教师密码：`123456`

## 未来改进

- **数据库支持**：后续版本可以考虑引入SQL数据库，并使用MD5加密来增强安全性。
- **功能扩展**：可以增加更多功能模块，如学生成绩统计、课程管理等。

## 注意事项

- 本项目为无数据库版，所有数据存储在文本文件中，适合学习和初步开发使用。
- 请妥善保管 `教师密码.txt` 文件，避免泄露敏感信息。
