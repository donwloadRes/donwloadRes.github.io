# EndNote X 9.3.3 安装指南

本仓库提供EndNote X 9.3.3的安装文件及相关资源。EndNote是一款功能强大的文献管理软件，广泛应用于学术研究和论文写作中。通过本指南，您可以轻松完成EndNote X 9.3.3的安装。

## 安装步骤

1. **下载安装文件**  
   从本仓库下载EndNote X 9.3.3的安装文件。

2. **运行安装程序**  
   双击下载的`.exe`文件，启动安装程序。

3. **选择安装路径**  
   如果您不想将EndNote安装在C盘，可以点击“Browse”按钮，选择其他路径进行安装。

4. **关闭Office软件**  
   在安装过程中，如果您的电脑上打开了Word、PPT或Excel等Office软件，请先关闭它们，以避免安装过程中出现错误提示。

5. **完成安装**  
   等待安装程序完成，点击“Finish”按钮结束安装。

6. **添加参考文献格式**  
   安装完成后，将提供的参考文献格式文件复制到EndNote的Styles文件夹中，以便在写作时使用。

## 注意事项

- 在安装过程中，请确保您的电脑已连接到互联网。
- 安装完成后，建议重启电脑以确保所有设置生效。
