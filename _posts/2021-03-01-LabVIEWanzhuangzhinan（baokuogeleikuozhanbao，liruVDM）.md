# LabVIEW安装指南（包括各类扩展包，例如VDM）

## 简介
本资源文件提供了LabVIEW及其相关扩展包（如VDM）的安装指南。LabVIEW是一款图形化编程环境，广泛应用于自动化测试、测量和控制系统开发。通过本指南，您可以轻松完成LabVIEW及其扩展包的安装。

## 安装步骤

### 1. 下载NI Package Manager
首先，您需要下载NI Package Manager。NI Package Manager是NI公司提供的软件包管理工具，用于安装、更新和管理NI软件。

### 2. 下载NI License Activator
接下来，下载NI License Activator。该工具用于激活NI软件的许可证。

### 3. 打开NI Package Manager
打开NI Package Manager后，您可以按照以下步骤进行安装：

#### 3.1 下载LabVIEW
- 在NI Package Manager中选择LabVIEW进行下载。
- 安装路径默认为`C:\Program Files\National Instruments`。
- 安装过程中，请阅读并同意许可协议。
- 安装完成后，选择不要立即激活。

#### 3.2 下载VDM
- 同样在NI Package Manager中选择VDM进行下载。
- 安装路径默认为`C:\Program Files\National Instruments`。
- 安装过程中，请阅读并同意许可协议。
- 安装完成后，选择不要立即激活。

### 4. 打开NI License Activator
打开NI License Activator后，右键点击并选择“激活”以激活您的NI软件。

## 总结
通过以上步骤，您可以成功安装LabVIEW及其扩展包（如VDM）。请确保在安装过程中保持网络连接，以便顺利完成下载和激活。
