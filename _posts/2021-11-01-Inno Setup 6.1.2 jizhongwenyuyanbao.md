# Inno Setup 6.1.2 及中文语言包

## 简介
Inno Setup 是一款免费的安装包制作软件，操作简单、功能稳定，能够快速制作出标准Windows风格的安装界面，足以完成一般安装任务。使用Inno Setup打包可执行的exe软件，可在任何电脑中运行。

## 资源内容
- **Inno Setup 6.1.2**: 最新版本的Inno Setup，提供了更强大的功能和更好的兼容性。
- **中文语言包**: 为Inno Setup 6.1.2提供的中文语言支持，方便中文用户使用。

## 使用说明
1. **下载资源**: 从本仓库下载Inno Setup 6.1.2及中文语言包。
2. **安装Inno Setup**: 运行Inno Setup 6.1.2的安装程序，按照提示完成安装。
3. **安装中文语言包**: 将中文语言包文件放置在Inno Setup的安装目录下，通常为`C:\Program Files (x86)\Inno Setup 6`。
4. **配置语言**: 打开Inno Setup，进入设置菜单，选择中文语言包作为默认语言。
