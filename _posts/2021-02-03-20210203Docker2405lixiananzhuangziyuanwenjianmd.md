# Docker 24.0.5 离线安装资源文件

本仓库提供了一个用于离线安装 Docker 24.0.5 的资源文件，包含所有必要的依赖项和安装脚本。通过执行提供的脚本，您可以在没有互联网连接的环境中轻松安装 Docker。

## 资源文件内容

- **docker 24.0.5 离线安装包**：包含 Docker 24.0.5 及其所有依赖项的 RPM 包。
- **install_docker.sh**：安装脚本，执行该脚本即可完成 Docker 的安装。

## 安装步骤

1. **下载资源文件**：将本仓库中的所有文件下载到您的目标服务器。
2. **执行安装脚本**：在终端中执行以下命令：
   ```bash
   ./install_docker.sh
   ```
   **注意**：该脚本需要执行两遍才能完成安装。

3. **验证安装**：安装完成后，您可以通过以下命令验证 Docker 是否成功安装：
   ```bash
   docker --version
   ```
