# Vue3多功能应用：待办列表、日记与便签、LOL英雄资料-课程大作业

## 项目简介

本项目是一个综合性的Web应用程序，利用Vue3构建，旨在提供便捷的个人管理工具。涵盖的功能包括全面的用户管理系统（注册、登录、找回密码、个人信息修改）、高效待办事项管理、日记与便签编写与维护，以及一个独特的LOL英雄资料查询模块。项目详细操作指南及开发背景可在[项目文档](https://blog.csdn.net/qq_34060370/article/details/126172749)中查阅。为了本地运行项目，需要预先安装[Json-Server](https://github.com/typicode/json-server)，并确保在代码配置中调整URL指向正确的本地服务器路径。

## 核心功能

### 用户系统
- **注册**: 简洁明了的用户注册流程。
- **登录与找回密码**: 引入验证码机制增强安全，支持快速找回丢失的账户密码。
- **个性化设置**: 允许用户随时修改个人资料。

### 待办列表管理
- **列表展示**: 清晰展示所有待办事项。
- **增删改查**: 支持新增待办、轻松修改和删除条目，以及通过拖拽或设置完成状态来变更待办项的状态。
- **智能排序**: 提供多种排序选项，帮助用户高效管理任务优先级。

### 日记与便签
- **创作空间**: 用户可以创建日记记录生活点滴，或留下便签快速记事。
- **编辑回忆**: 方便地编辑以往的日记与便签内容，保留每一份珍贵记忆。
- **有序存档**: 实现日记与便签的分类管理和轻松检索。

### LOL英雄资料库
- **探索英雄世界**: 特色模块，集成最新的LOL英雄数据，满足游戏爱好者的需求。

## 开发与运行指南

1. 安装Node.js环境。
2. 使用`npm install json-server -g`全局安装Json-Server。
3. 在项目根目录下启动Json-Server服务：`json-server --watch db.json`。
4. 使用终端或命令提示符进入项目目录，执行`npm install`安装项目依赖。
5. 运行项目：`npm run serve`，浏览器将自动打开应用界面。

请根据文档中的指引进行具体的操作细节调整，享受开发过程的乐趣和成就感！
