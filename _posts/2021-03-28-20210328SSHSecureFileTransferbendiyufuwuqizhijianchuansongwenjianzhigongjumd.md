# SSH Secure File Transfer -- 本地与服务器之间传送文件之工具

## 简介

SSH Secure File Transfer 是一款用于在本地计算机与远程服务器之间进行文件传输的工具。该工具基于SSH协议，提供了安全、可靠的文件传输功能，适用于Windows和Linux系统。

## 功能特点

- **安全传输**：所有文件传输过程均通过SSH协议加密，确保数据的安全性。
- **跨平台支持**：支持Windows和Linux系统之间的文件传输。
- **简单易用**：提供直观的用户界面，方便用户进行文件的上传和下载操作。
- **快速连接**：支持快速连接功能，用户可以快速建立与远程服务器的连接。

## 使用方法

1. **安装**：下载并安装SSH Secure File Transfer Client。
2. **连接服务器**：
   - 打开SSH Secure Shell Client。
   - 点击“Quick Connect”按钮。
   - 输入服务器地址、用户名和端口号。
   - 点击“Connect”按钮，输入密码进行连接。
3. **文件传输**：
   - 连接成功后，选择“New file transfer window”按钮。
   - 在文件传输窗口中，双击文件夹目录以显示文件。
   - 从本地文件目录拖放文件到右侧窗口，完成文件上传。

## 注意事项

- 确保服务器和客户端的SSH服务正常运行。
- 使用强密码或SSH密钥进行身份验证，提高安全性。
- 定期更新SSH Secure File Transfer Client以获取最新功能和安全补丁。

## 常见问题

- **连接失败**：检查服务器地址、用户名和端口号是否正确，确保网络连接正常。
- **文件传输失败**：确保目标目录有足够的权限，检查文件大小是否超出限制。

通过SSH Secure File Transfer工具，您可以轻松、安全地在本地与远程服务器之间进行文件传输，提高工作效率。
