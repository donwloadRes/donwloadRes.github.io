# Intel I219-V网卡芯片MAC修改工具

## 简介
本仓库提供了一个用于修改Intel I219-V网卡芯片MAC地址的工具。通过该工具，您可以轻松地修改网卡的MAC地址，以满足特定的网络需求。

## 使用方法
1. **下载资源文件**：首先，下载本仓库中的资源文件。
2. **解压文件**：将下载的文件解压到您制作好的DOS启动U盘的根目录中。
3. **进入DOS系统**：将U盘插入计算机并启动，进入DOS系统。
4. **输入命令**：在DOS命令行中输入以下命令来修改MAC地址：
   ```
   c:\eeupdate /nic=[A] /mac=[B]
   ```
   其中：
   - `[A]` 是网卡NIC列表的序号。
   - `[B]` 是您希望修改的MAC地址。

   例如：
   ```
   c:\eeupdate /nic=1 /mac=BCAEC51267E7
   ```

## 注意事项
- 请确保您已经正确制作了DOS启动U盘，并且文件已正确解压到U盘根目录。
- 在输入命令时，请仔细核对网卡序号和MAC地址，确保输入正确。
