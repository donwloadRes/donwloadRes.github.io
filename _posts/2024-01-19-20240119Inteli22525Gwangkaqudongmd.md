## 安装指南

### Windows

1. 下载驱动程序压缩包。
2. 解压缩文件到任意目录。
3. 打开设备管理器，找到 Intel i225 2.5G 网卡。
4. 右键点击网卡，选择“更新驱动程序”。
5. 选择“浏览我的计算机以查找驱动程序软件”。
6. 选择解压缩后的驱动程序目录，点击“下一步”进行安装。

### Linux

1. 下载驱动程序压缩包。
2. 解压缩文件到任意目录。
3. 打开终端，进入解压缩后的目录。
4. 运行以下命令进行安装：
   ```bash
   sudo ./install.sh
   ```
