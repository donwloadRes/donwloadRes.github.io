# Mac M1 (ARM) 安装 Redis 的四种方式

本文详细介绍了在 Mac M1 (ARM) 芯片上安装 Redis 的四种方法，包括本机安装、Homebrew 安装、虚拟机安装和 Docker 安装。每种方法都提供了详细的步骤和注意事项，帮助用户根据自己的需求选择最合适的安装方式。

## 1. 本机安装 Redis

### 步骤：
1. 下载 Redis 安装包。
2. 解压安装包。
3. 执行编译测试指令。
4. 正式安装 Redis。
5. 启动 Redis 服务。
6. 连接 Redis 并进行测试。

### 注意事项：
- 建议使用 6.2 版本，5.0 版本可能会有报错。

## 2. Homebrew 安装 Redis

### 步骤：
1. 确保已安装 Homebrew。
2. 使用 Homebrew 安装 Redis。
3. 启动 Redis 服务。
4. 连接 Redis 并进行测试。

### 注意事项：
- 安装过程中可以选择特定版本，如 5.0 版本。

## 3. 虚拟机安装 Redis

### 步骤：
1. 创建 Redis 目录。
2. 下载 Redis 安装包。
3. 解压并编译 Redis。
4. 安装 Redis。
5. 启动 Redis 服务。
6. 修改 Redis 配置文件。
7. 设置 Redis 开机自启。
8. 开启 Redis 端口。

### 注意事项：
- 需要安装 C 环境，如 GCC 和 automake。

## 4. Docker 安装 Redis

### 步骤：
1. 进入 Docker Hub 查找支持 ARM 的 Redis 镜像。
2. 指定版本号安装 Redis。
3. 启动 Redis 容器。
4. 连接 Redis 并进行测试。

### 注意事项：
- 选择支持 ARM 架构的 Redis 版本。

通过以上四种方法，用户可以根据自己的需求和环境选择最适合的 Redis 安装方式。每种方法都提供了详细的步骤和注意事项，帮助用户顺利完成 Redis 的安装和配置。
