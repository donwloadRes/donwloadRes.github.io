# Modbus RTU与Modbus TCP/IP控件包（含示例源码）

## 资源介绍

本仓库提供了一个强大的Modbus功能控件包，支持Modbus RTU和Modbus TCP/IP协议，能够快速有效地获取设备仪表数据。该控件包适用于多种开发环境，包括VC、VB、VC#、VF#、BCB、DEPHI等，并且内含丰富的示例源码，帮助开发者快速上手。

## 功能列表

- **01: Read coil status** - 读线圈状态
- **02: Read input status** - 读输入状态
- **03: Read holding register** - 读保持寄存器
- **04: Read input registers** - 读输入寄存器
- **05: Force single coil** - 强制单线圈
- **06: Preset single register** - 预置单寄存器
- **15: Force multiple coils** - 强制多线圈
- **16: Preset multiple registers** - 预置多寄存器
- **17: Report slave ID** - 报告从设备ID
- **22: Mask write register** - 屏蔽写寄存器
- **23: Read/Write registers** - 读/写寄存器

1. **下载资源文件**：点击仓库中的`Modbus RTU与Modbus TCP/IP控件包（含示例源码）.rar`文件进行下载。

4. **集成到项目**：将控件包中的相关文件集成到自己的项目中，根据示例源码进行配置和调用。
