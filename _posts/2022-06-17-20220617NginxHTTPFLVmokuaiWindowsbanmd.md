# Nginx HTTP-FLV 模块（Windows版）

## 简介

本仓库提供了一个已编译好的 `nginx-http-flv-module`（Windows版）资源文件，方便用户在Windows环境下快速部署和使用。该模块基于以下组件构建：

- OpenSSL 1.1.1g
- PCRE 8.44
- Zlib 1.2.11
- Nginx HTTP Module v1.2.10
- Nginx 1.21.6

## 使用方法

1. **下载资源文件**：
   - 点击仓库中的 `nginx-http-flv-module(windows版).zip` 文件进行下载。

2. **解压文件**：
   - 将下载的压缩包解压到你希望安装Nginx的目录。

3. **启动Nginx**：
   - 打开命令提示符（CMD），导航到解压后的Nginx目录。
   - 运行以下命令启动Nginx：
     ```sh
     nginx.exe
     ```

4. **验证安装**：
   - 打开浏览器，访问 `http://localhost`，如果看到Nginx的欢迎页面，说明安装成功。

## 注意事项

- 确保你的系统环境符合Nginx的运行要求。
