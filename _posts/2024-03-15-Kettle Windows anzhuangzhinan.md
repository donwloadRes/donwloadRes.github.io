# Kettle Windows 安装指南

本文档提供了在 Windows 系统上安装 Kettle 的详细步骤。Kettle 是一款开源的 ETL（Extract, Transform, Load）工具，适用于数据抽取、转换和加载任务。以下是安装过程的详细说明。

## 一、检查 JDK 安装

在安装 Kettle 之前，请确保您的系统上已经安装了 Java Development Kit (JDK)。您可以通过以下步骤检查 JDK 是否已安装：

1. 打开命令提示符（Win+R，输入 `cmd`，然后按回车）。
2. 输入 `java -version` 并按回车。
3. 如果显示 Java 版本信息，则说明 JDK 已安装。如果没有，请先安装 JDK。

## 二、下载并解压 Kettle 安装包

1. 下载 Kettle 安装包 `pdi-ce-7.1.0.0-12.zip`。
2. 将下载的压缩包解压到您选择的目录中。

## 三、进入 data-integration 目录

解压后，进入 `data-integration` 目录。这是 Kettle 的主目录，包含所有必要的文件和脚本。

## 四、启动 Kettle

1. 在 `data-integration` 目录中，找到 `Spoon.bat` 文件。
2. 双击 `Spoon.bat` 文件以启动 Kettle。

## 五、案例演示

以下是一些使用 Kettle 的常见案例：

### （一）Kettle 将 Excel 数据导入 MySQL

1. 在新建转换中添加 Excel 输入与表输出并连接。

### （二）Kettle 将 Excel 数据转换为文本

1. 新建转换，添加 Excel 输入和文本文件输出，并连接与 Excel 输入。

### （三）Kettle 将 MySQL 数据库数据输出到 Excel

1. 新建转换，添加表输入和 Excel 输出，并连接。

通过以上步骤，您应该能够在 Windows 系统上成功安装并运行 Kettle。如果在安装过程中遇到任何问题，请参考官方文档或相关社区论坛获取帮助。
