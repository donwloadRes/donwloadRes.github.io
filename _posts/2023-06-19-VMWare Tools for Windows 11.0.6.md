# VMWare Tools for Windows 11.0.6

## 资源简介

本仓库提供的是VMWare Tools的特定版本——11.0.6，专为Windows操作系统设计。VMWare Tools是VMware虚拟机软件的重要组件，它增强了虚拟机与宿主机之间的集成和交互能力，提供了更好的性能和用户体验。通过安装此工具，您可以享受到诸如拖放文件、屏幕自适应调整、时间同步、优化图形性能等便利功能。

## 下载与安装指南

### 步骤一：下载资源
直接从本仓库下载提供的VMWare Tools for Windows 11.0.6文件。确保您下载的是最新且适用于您虚拟机的版本。

### 步骤二：挂载至虚拟机
1. 打开您的VMware虚拟机。
2. 选择“虚拟机”菜单 -> “CD/DVD(SATA)” -> “连接ISO镜像文件”。
3. 浏览并选择刚刚下载的VMWare Tools ISO文件，点击确定以挂载。

### 步骤三：安装VMWare Tools
1. 在虚拟机内部，您会看到一个新的可移动设备（如：“DVD驱动器”或显示ISO名称）。
2. 双击打开这个新出现的“光盘驱动器”，找到并运行里面的`setup.exe`（对于Windows系统）。
3. 安装向导将引导您完成安装过程。通常情况下，保持默认设置，并按“下一步”继续即可。
4. 当提示重启虚拟机时，建议选择“立即重新启动”以完成安装流程。

## 注意事项
- 确保您的VMware虚拟机软件已更新至支持此版本VMware Tools的版本。
- 在安装过程中可能需要管理员权限，请提前确认。
- 如果您在之前已经安装了旧版本的VMware Tools，请先卸载旧版本，然后再进行安装新的版本。
