# CentOS 安装 Docker 和 docker-compose（V1和V2两个版本）

本仓库提供了一个资源文件，详细介绍了如何在CentOS系统上安装Docker和docker-compose，并提供了V1和V2两个版本的安装方法。

## 内容概述

本文档包含了以下主要内容：

1. **安装 Docker**
   - 更新Docker的yum源为阿里云仓库
   - 安装必要的系统工具
   - 查看docker-ce版本
   - 安装指定版本的Docker
   - 切换Docker镜像下载源（使用阿里云镜像）
   - 启动测试Docker
   - Docker启动关闭操作
   - 安装runlike查看run参数工具
   - 卸载/更新已经安装的Docker

2. **安装 docker-compose**
   - 安装指定版本 docker-compose
   - 添加 docker-compose 可执行权限
   - 验证是否安装成功
   - 卸载 docker-compose

## 使用说明

1. **安装 Docker**
   - 按照文档中的步骤，逐步执行命令以完成Docker的安装。
   - 注意选择合适的Docker版本进行安装。

2. **安装 docker-compose**
   - 根据需求选择V1或V2版本进行安装。
   - 安装完成后，验证安装是否成功。

## 注意事项

- 在安装过程中，请确保网络连接正常，以便顺利下载所需的软件包。
- 安装Docker和docker-compose时，建议使用root权限执行命令。
- 如果遇到问题，可以参考文档中的常见问题解答部分，或查阅相关社区资源。
