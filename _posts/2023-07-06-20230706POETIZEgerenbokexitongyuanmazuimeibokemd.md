# POETIZE个人博客系统源码 最美博客

## 项目简介

POETIZE个人博客系统源码 最美博客是一个基于SpringBoot、Vue2和Vue3开发的综合性个人博客系统。该系统不仅支持移动端自适应，还提供了完备的前台和后台管理功能。系统分为两个主要模块：博客系统和聊天室系统。

## 功能特点

### 博客系统

1. **文章管理**：支持文章的发布、编辑、删除等功能。
2. **表白墙**：用户可以在表白墙上发布表白信息。
3. **图片墙**：展示用户上传的图片，支持图片分类和标签。
4. **收藏夹**：用户可以收藏自己喜欢的文章或图片。
5. **乐曲播放**：内置音乐播放器，支持在线播放音乐。
6. **视频播放**：支持上传和播放视频内容。
7. **留言板**：用户可以在留言板上进行互动交流。
8. **友链管理**：管理个人博客的友情链接。
9. **时间线**：以时间轴的形式展示博客的更新历史。
10. **后台管理**：提供完善的后台管理功能，包括用户管理、内容管理、系统设置等。

### 聊天室系统

1. **朋友圈（时间线）**：用户可以发布动态，查看好友的动态。
2. **好友管理**：支持添加、删除好友，查看好友列表。
3. **群聊功能**：支持创建群聊，邀请好友加入群聊。

## 技术栈

- **后端**：SpringBoot
- **前端**：Vue2、Vue3
- **数据库**：MySQL
- **缓存**：Redis
- **消息队列**：RabbitMQ

## 安装与使用

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/poetize-blog.git
   ```

2. **配置数据库**：
   在`application.properties`文件中配置数据库连接信息。

3. **启动后端服务**：
   ```bash
   cd poetize-blog
   mvn spring-boot:run
   ```

4. **启动前端服务**：
   ```bash
   cd poetize-blog/frontend
   npm install
   npm run serve
   ```

5. **访问系统**：
   打开浏览器，访问`http://localhost:8080`即可进入博客系统。
