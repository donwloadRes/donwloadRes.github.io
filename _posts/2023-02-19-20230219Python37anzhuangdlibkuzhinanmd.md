# Python 3.7 安装 dlib 库指南

本仓库提供了一个适用于 Python 3.7 的 dlib 库安装文件，无需安装 Visual Studio (VS) 即可完成安装。以下是详细的安装步骤和注意事项。

## 安装步骤

1. **下载 dlib 安装文件**  
   下载本仓库提供的 `dlib-19.17.99-cp37-cp37m-win_amd64.whl` 文件。

2. **打开命令提示符（CMD）**  
   以管理员身份打开命令提示符。

3. **安装 dlib 库**  
   在命令提示符中输入以下命令并执行：
   ```
   pip install dlib-19.17.99-cp37-cp37m-win_amd64.whl
   ```

4. **等待安装完成**  
   安装过程可能需要一些时间，请耐心等待。

## 注意事项

- 确保你的 Python 版本为 3.7，否则可能无法使用本安装文件。
- 安装过程中需要管理员权限，请确保以管理员身份运行命令提示符。
- 如果在安装过程中遇到任何问题，请参考 CSDN 博客文章中的详细说明。

通过以上步骤，你可以在不安装 Visual Studio 的情况下成功安装 dlib 库，方便进行人脸识别等计算机视觉应用的开发。
