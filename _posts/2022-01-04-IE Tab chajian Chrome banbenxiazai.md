## 简介
IE Tab 是一款专为 Chrome 浏览器设计的插件，它能够在 Chrome 浏览器中嵌入 IE 浏览器的内核。对于经常使用 Chrome 浏览器的用户来说，IE Tab 插件无疑是一个非常实用的工具。当你遇到只能兼容 IE 浏览器的网站时，无需再切换到 IE 浏览器，直接在 Chrome 中使用 IE Tab 即可正常访问。

## 功能特点
- **无缝切换**：在 Chrome 浏览器中直接打开 IE 内核的标签页，无需切换浏览器。
- **兼容性强**：支持访问那些只能通过 IE 浏览器正常访问的网站，如网上银行、政府网站、企业网站等。
- **便捷操作**：只需在 Chrome 中创建一个 IE 标签页，即可享受 IE 浏览器的兼容性。
- **快速体验**：在 Chrome 浏览器中使用 IE Tab，既能享受 Chrome 的快速浏览体验，又能解决兼容性问题。

## 使用场景
- **网上银行**：访问某些只支持 IE 浏览器的网上银行网站。
- **政府网站**：访问一些只兼容 IE 浏览器的政府服务网站。
- **企业内部系统**：访问企业内部的一些只支持 IE 浏览器的管理系统。
