# Arduino UNO 控制全彩RGB点阵条屏单元板HUB75接口显示数字汉字

## 简介

本资源文件提供了使用Arduino UNO控制全彩RGB点阵条屏单元板（HUB75接口）显示数字、字母和汉字的方法。通过描点法实现汉字的显示，其他显示方法正在探索中。资源中包含了程序代码、两个必要的库文件（RGB Matrix Panel library 和 Adafruit GFX Library），以及相关的文档说明。

## 功能特点

- **显示内容**：支持显示数字、字母和汉字。
- **显示方式**：汉字采用描点法显示，其他显示方式正在开发中。
- **硬件接口**：使用HUB75接口连接全彩点阵屏。
- **库文件**：包含RGB Matrix Panel library和Adafruit GFX Library，确保程序的正常运行。
- **兼容性**：适用于32*32和32*64的点阵屏，但目前仅实现了32*32的显示。

## 使用说明

1. **硬件连接**：
   - 将Arduino UNO与全彩RGB点阵条屏单元板通过HUB75接口连接。
   - 确保使用5V大电流电源供电，以避免彩色显示不正常的问题。

2. **软件配置**：
   - 下载并安装RGB Matrix Panel library和Adafruit GFX Library。
   - 打开资源文件中的程序代码，编译并上传到Arduino UNO。
   - 如果编译过程中出现错误，请在代码中添加`#include <Adafruit_GFX.h>`。

3. **显示效果**：
   - 程序上传成功后，点阵屏将显示预设的数字、字母和汉字。
   - 若需要显示其他内容，可根据描点法自行修改程序代码。

## 注意事项

- 确保使用5V大电流电源供电，以保证彩色显示的正常。
- 在32*64的点阵屏上显示时，内容会重复显示，目前仅支持32*32的显示效果。
- 若在编译过程中遇到问题，请参考库文件的官方文档或社区支持。
