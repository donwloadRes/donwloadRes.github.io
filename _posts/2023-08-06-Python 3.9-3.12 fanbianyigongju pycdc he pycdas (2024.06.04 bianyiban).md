## 编译说明
如果你希望手动编译 pycdc 和 pycdas，可以参考以下步骤：

1. 克隆 pycdc 的 GitHub 仓库：
   ```bash
   git clone https://github.com/zrax/pycdc.git
   ```

2. 安装 CMake 和其他必要的编译工具。

3. 使用 CMake 进行编译：
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```
