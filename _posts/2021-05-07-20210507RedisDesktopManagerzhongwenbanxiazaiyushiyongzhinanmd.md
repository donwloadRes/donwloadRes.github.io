## Redis Desktop Manager 安装
- 下载完成后，双击安装文件，按照提示完成安装过程。安装过程中，您可以选择安装路径和其他选项。

## Redis Desktop Manager 汉化
- 安装完成后，打开 RDM 软件，在主界面点击 `Settings`，选择 `language` 语言为 `zh-CN`，然后点击 `OK`。软件界面将切换为中文。

## Redis Desktop Manager 配置
- 在 RDM 中，您可以配置 Redis 服务器的连接信息，包括主机地址、端口号、密码等。配置完成后，您可以连接到 Redis 服务器并进行管理操作。

## Redis Desktop Manager 卸载
- 如果您需要卸载 RDM，可以通过控制面板中的“程序和功能”找到 Redis Desktop Manager，选择卸载即可。
