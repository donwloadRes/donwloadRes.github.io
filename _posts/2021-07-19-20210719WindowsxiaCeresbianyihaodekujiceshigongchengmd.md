# Windows 下 Ceres 编译好的库及测试工程

本仓库提供了一个在 Windows 环境下编译好的 Ceres 库，以及一个简单的测试工程，方便开发者快速集成和使用 Ceres 库。

## 资源文件内容

- **Ceres 库**: 使用 VS2019 和 CMake 编译完成的 Ceres 库，可直接在 Windows 项目中使用。
- **测试工程**: 一个简单的测试工程，展示了如何链接和使用编译好的 Ceres 库。

## 使用说明

1. **下载资源**: 克隆或下载本仓库到本地。
2. **集成 Ceres 库**: 将编译好的 Ceres 库文件（包括头文件和库文件）添加到你的项目中。
3. **配置项目**: 在 Visual Studio 中配置项目，确保正确链接 Ceres 库。
4. **运行测试工程**: 打开测试工程，编译并运行，验证 Ceres 库的正确性。

## 环境要求

- **Visual Studio 2019**: 本资源文件基于 VS2019 编译，建议使用相同版本的 Visual Studio。
- **CMake**: 用于生成 Visual Studio 项目文件。

## 注意事项

- 本资源文件仅适用于 Windows 环境，其他操作系统需自行编译。
- 如果需要使用不同版本的 Ceres 库，请参考 Ceres 官方文档进行编译。
