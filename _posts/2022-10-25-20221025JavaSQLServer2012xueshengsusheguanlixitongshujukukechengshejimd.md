# Java+SQL Server 2012 学生宿舍管理系统 数据库课程设计

欢迎使用“学生宿舍管理系统”，这是一个专为大学宿舍管理设计的简单控制台应用程序。本系统旨在通过Java编程语言结合SQL Server 2012数据库，实现对学生宿舍信息的高效管理。它覆盖了增删查改以及统计分析的基本功能需求，非常适合高校计算机科学或软件工程专业学生的数据库课程设计项目。

## 主要功能包括：

- **增加信息**：能够添加新的宿舍分配记录、学生入住信息。
- **删除信息**：支持删除不再有效的宿舍分配或已离校的学生信息。
- **查询功能**：灵活的查询机制，可以根据宿舍号、学生姓名等条件查找信息。
- **修改数据**：允许对现有宿舍信息或学生信息进行更新。
- **统计分析**：提供简单的统计功能，如空闲床位统计、各楼栋学生分布情况等。

## 开发环境及技术栈

- **开发环境**: Java JDK 1.8及以上版本
- **数据库**: Microsoft SQL Server 2012 或更高版本
- **集成开发环境**: 推荐使用IDEA或Eclipse
- **技术框架**: 标准Java SE，利用原生JDBC进行数据库操作
- **数据库设计**: 包含清晰的实体关系图（ER图），合理划分表结构

## 快速启动指南

1. **环境准备**: 确保已安装好Java环境和SQL Server 2012，并配置好JDBC驱动。
2. **导入项目**: 将压缩包解压后导入到你的IDE中。
3. **数据库设置**: 导入数据库脚本创建表结构，并根据需要配置连接字符串。
4. **运行应用**: 编译并运行主类，即可在控制台看到系统的启动界面。

## 注意事项

- 在实际部署前，请确保调整数据库连接参数以匹配你的本地数据库设置。
- 请备份数据，以防测试过程中发生意外数据丢失。
- 源代码中的注释有助于理解系统架构和逻辑，建议阅读。
