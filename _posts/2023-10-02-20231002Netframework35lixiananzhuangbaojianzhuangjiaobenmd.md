# Netframework 3.5 离线安装包及安装脚本

## 概述

本仓库提供了Microsoft .NET Framework 3.5的完整离线安装包，特别适合无网络环境或需要批量部署的场景。随包附带自定义安装脚本，简化了传统的安装流程，让在各种Windows系统上部署.NET Framework 3.5变得轻松快捷。

## 使用说明

1. **下载与解压**：首先，从本仓库下载提供的压缩包，并将其解压到您选择的磁盘目录下。

2. **编辑安装脚本**：找到解压后的`install.bat`脚本文件。此脚本设计了自动化安装流程。请用文本编辑器（如记事本）打开它，找到标有`Source`的行，修改其后的路径为您实际解压.NET Framework 3.5安装文件的目录路径。例如，如果你将文件解压到了`D:\ offlineInstall\dotnet3.5`，则需将脚本中的相应路径调整为这个位置。

3. **以管理员身份运行**：完成路径配置后，右键点击`install.bat`脚本，选择“以管理员身份运行”。这一步至关重要，因为安装过程中可能需要系统级别的权限。

4. **自动安装**：脚本将自动执行.NET Framework 3.5的安装过程。请耐心等待直至安装完成，期间可能会看到一些安装进度窗口或者命令提示符界面的快速闪动，这是正常现象。

## 注意事项

- **兼容性**：确保您的操作系统支持.NET Framework 3.5。它通常适用于Windows 7及以上版本的Windows操作系统。
  
- **系统更新**：对于Windows 8和Windows 10，有时操作系统可以通过Windows Update来激活.NET Framework 3.5，但此方法不适用于离线安装情况。

- **安全软件**：在运行安装脚本前，确保您的杀毒软件不会误报，以免干扰正常的安装过程。
