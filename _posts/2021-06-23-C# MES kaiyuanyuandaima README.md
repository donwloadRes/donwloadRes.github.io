# C# MES 开源源代码 README

## 项目简介

欢迎来到 C# MES 开源源代码项目！本项目旨在提供一个基于C#语言开发的制造执行系统（Manufacturing Execution System, MES）解决方案。MES 系统位于企业资源计划（ERP）系统与生产自动化系统之间，对于优化车间管理、提高生产效率和质量控制具有至关重要的作用。此开源项目希望能够帮助中小企业及开发者快速搭建和定制符合自身需求的生产管理系统。

## 特性概述

- **模块化设计**：项目采用模块化设计思路，易于扩展和维护。
- **C#编程**：完全基于.NET Framework或.NET Core/5+，适合现代软件开发环境。
- **数据库支持**：支持多种数据库系统（如SQL Server, MySQL等），灵活配置。
- **生产调度与跟踪**：实现生产订单的全程跟踪，从排产到成品入库。
- **质量管理**：内置质量管理模块，确保产品质量控制。
- **物料与库存管理**：高效处理物料消耗与库存变动。
- **报表与数据分析**：提供丰富的报表工具，助力决策分析。
- **用户界面**：友好且直观的用户界面，提升操作体验。

## 快速入门

1. **环境准备**：确保你的开发环境中安装了Visual Studio或Visual Studio Code，并配置好相应的.NET环境。
2. **获取代码**：点击"Clone or download"按钮将本项目下载至本地。
3. **数据库设置**：根据项目文档配置数据库连接字符串，并运行数据库脚本来创建表结构。
4. **编译与运行**：打开解决方案文件，在Visual Studio中编译并运行项目。
5. **开始探索**：登录系统，根据提供的示例数据开始探索功能。
