# VirtualBox 安装 Kali Linux 保姆级教程（1）

本资源文件提供了一个详细的教程，指导用户如何在 VirtualBox 上安装 Kali Linux。教程内容涵盖了从下载 Kali Linux 镜像到完成安装的每一个步骤，适合初学者和有经验的用户参考。

## 教程内容概述

1. **Kali 下载**  
   介绍了如何从官方网站下载 Kali Linux 的 VirtualBox 版本，以及如何使用百度云链接下载。

2. **Kali 安装**  
   详细说明了如何在 VirtualBox 中导入下载的 Kali Linux 镜像文件，并进行安装。

3. **设置 root 用户并登录**  
   指导用户如何设置 root 用户密码，并切换到 root 用户进行登录。

4. **换源及更新**  
   介绍了如何更换 Kali Linux 的软件源为国内源，并进行系统更新。

## 使用说明

1. **下载资源**  
   下载本仓库中的资源文件，包含教程所需的 Kali Linux 镜像文件。

2. **参考教程**  
   按照教程中的步骤，在 VirtualBox 中安装和配置 Kali Linux。

3. **问题反馈**  
   如果在安装过程中遇到任何问题，欢迎在仓库中提出 Issue，我们会尽快回复并提供帮助。

## 注意事项

- 确保 VirtualBox 已正确安装并配置。
- 在安装过程中，请耐心等待系统更新和软件包安装完成。
- 建议在安装前备份重要数据，以防意外情况发生。

通过本教程，您将能够顺利在 VirtualBox 上安装并配置 Kali Linux，开始您的网络安全学习和实践之旅。
