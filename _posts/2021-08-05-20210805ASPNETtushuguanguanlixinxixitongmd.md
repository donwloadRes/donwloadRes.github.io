# ASP.NET图书馆管理信息系统

## 项目简介

本项目是一个基于ASP.NET技术开发的图书馆管理信息系统，旨在为现代化的图书馆管理提供一个高效、便捷的解决方案。系统涵盖了图书馆管理的核心功能，包括图书入库、借阅、归还、续借、预约、查询等，旨在提高图书馆的工作效率和服务质量。

## 系统功能

### 管理员功能

1. **个人中心**：管理员可以管理自己的个人信息。
2. **读者管理**：管理员可以添加、修改、查询和删除读者信息。
3. **图书信息管理**：管理员可以添加、修改、查询和删除图书信息。
4. **图书分类管理**：管理员可以管理图书的分类信息。
5. **书架信息管理**：管理员可以管理书架信息。
6. **图书借阅管理**：管理员可以管理图书的借阅信息。
7. **图书归还管理**：管理员可以管理图书的归还信息。
8. **留言板管理**：管理员可以管理读者的留言信息。
9. **系统管理**：管理员可以管理系统设置和权限。

### 读者功能

1. **个人中心**：读者可以管理自己的个人信息。
2. **图书借阅管理**：读者可以借阅图书。
3. **图书归还管理**：读者可以归还图书。
4. **留言板管理**：读者可以留言。
5. **我的收藏管理**：读者可以管理自己的收藏信息。

## 技术架构

- **开发平台**：ASP.NET Framework
- **数据库**：SQL Server 2000
- **数据访问技术**：ADO.NET
- **界面设计**：Windows 窗体和 Web 窗体

## 数据库设计

系统使用SQL Server 2000作为数据库管理系统，数据库设计遵循规范化原则，确保数据的完整性和一致性。主要数据表包括图书信息表、读者信息表、借阅信息表等。

## 系统特点

1. **界面友好**：系统界面设计美观，操作简便。
2. **数据安全**：系统采用严格的数据库保护措施，确保数据安全。
3. **功能全面**：系统涵盖了图书馆管理的所有核心功能。
4. **易于维护**：系统采用模块化设计，便于维护和升级。

## 使用说明

1. **安装环境**：确保服务器上安装了ASP.NET Framework和SQL Server 2000。
2. **数据库配置**：按照提供的SQL脚本创建数据库和数据表。
3. **系统部署**：将系统文件部署到服务器上，并进行必要的配置。
4. **用户权限**：根据需要设置管理员和读者的权限。
