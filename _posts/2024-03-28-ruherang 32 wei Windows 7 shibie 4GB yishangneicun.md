# 如何让 32 位 Windows 7 识别 4GB 以上内存

## 简介

本资源旨在指导用户启用物理地址扩展 (PAE)，从而允许 32 位 Windows 7 系统识别并使用超过 4GB 的内存。默认情况下，32 位 Windows 7 系统仅支持 4GB 内存，但通过本资源提供的补丁和配置方法，您可以突破这一限制，最大化您的内存利用率。

## 资源详情

本资源包含以下内容：

1. **PAE 补丁文件：**修改系统内核，使其支持 PAE 功能。
2. **使用指南：**详细说明如何应用补丁和配置系统以识别 4GB 以上内存。

## 适用范围

本资源仅适用于 32 位 Windows 7 系统。其他系统版本可能不兼容。

## 使用方法

1. **下载资源：**从我们的网站下载本资源文件。
2. **解压缩文件：**解压缩下载的文件，将获得包含补丁文件和使用指南的文件夹。
3. **阅读使用指南：**仔细阅读使用指南，了解详细的操作步骤。
4. **应用补丁：**按照使用指南的指示，将补丁文件复制到系统目录并运行必要的命令。
5. **配置系统：**使用 BOOTICE 工具编辑 BCD 文件，添加一个新的启动项以启用 PAE 功能。
6. **重启系统：**重启计算机，选择新的 PAE 启动项。系统将能够识别和使用超过 4GB 的内存。
