# DisplayPort-Alt-Mode-on-USB-C 标准文档

## 简介

本仓库提供了一个名为 `DisplayPort-Alt-Mode-on-USB-C.pdf` 的资源文件下载。该文件详细描述了在使用 USB Type-C 连接器的情况下，如何在支持 USB Type-C 的设备之间使用 DisplayPort 协议。

## 文件内容概述

该标准文档涵盖了以下几个主要方面：

1. **USB Type-C 连接器的使用**：描述了如何通过 USB Type-C 到 USB Type-C 电缆组件在设备之间使用 DisplayPort 协议。

2. **电缆组件的适配**：详细说明了如何将 USB Type-C 连接器适配到传统的 DisplayPort 连接器，以及如何通过电缆组件和适配器将 DisplayPort 适配到其他视频协议。

3. **同时使用 USB 和 DisplayPort**：描述了在 USB Type-C 连接器上同时使用 USB 增强超高速和 DisplayPort 协议的情况，特别是在扩展坞应用中的使用。

4. **连接器引脚分配和信号速率**：详细说明了在使用上述电缆组件和适配器时，USB Type-C 连接器的引脚分配和 DisplayPort 信号速率在不同场景下的使用方式。

5. **USB-PD 替代模式发现和进入过程**：描述了如何将 USB-PD 替代模式发现和进入过程应用于 USB Type-C 连接器上的 DisplayPort。
