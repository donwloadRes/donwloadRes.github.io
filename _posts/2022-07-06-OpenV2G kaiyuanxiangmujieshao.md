# OpenV2G 开源项目介绍

## 项目概述

OpenV2G 是一个开源项目，专注于实现 ISO IEC 15118 和 DIN 70121 标准中定义的车辆到电网 (V2G) 通信接口的编码功能。该项目旨在为开发者提供一个可靠的工具，用于将 C 结构序列化为 V2G 兼容的 (EXI) 消息，并能够将这些消息反序列化为 C 结构。

## 主要功能

- **数据绑定器功能**：OpenV2G 堆栈提供了强大的数据绑定器功能，允许用户将 C 结构与 V2G 兼容的 EXI 消息之间进行双向转换。
- **标准兼容性**：项目严格遵循 ISO IEC 15118 和 DIN 70121 标准，确保生成的消息符合行业规范。
- **开源社区支持**：作为一个开源项目，OpenV2G 鼓励社区参与和贡献，共同推动 V2G 技术的发展。

## 注意事项

- **即将推出的标准版本**：对于即将推出的 15118-2-2016 标准（ISO 版本 2）的 V2G 实现，将不会作为开源项目提供。如果您对该版本的解决方案感兴趣，请联系我们：exi.ct@siemens.com。

## 如何使用

1. **下载资源文件**：请从本仓库下载 OpenV2G 的资源文件。
2. **集成到项目中**：将下载的文件集成到您的 C 项目中，并根据需要进行配置。
3. **序列化和反序列化**：使用 OpenV2G 提供的数据绑定器功能，将 C 结构与 V2G 兼容的 EXI 消息进行转换。
