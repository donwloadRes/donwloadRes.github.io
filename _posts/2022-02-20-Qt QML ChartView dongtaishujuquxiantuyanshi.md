# Qt QML ChartView 动态数据曲线图演示

## 简介

本仓库提供了一个基于 Qt QML 的 ChartView 动态数据曲线图演示。通过该演示，您可以学习如何在 QML 中使用 ChartView 组件来显示动态更新的数据曲线图。

## 功能特点

- **动态数据更新**：演示了如何在 QML 中实时更新数据并显示在曲线图上。
- **多种曲线类型**：支持多种曲线类型，如折线图、面积图等。
- **自定义样式**：提供了自定义曲线图样式的示例，包括颜色、线条粗细等。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/your-repo.git
   ```

2. **打开项目**：
   使用 Qt Creator 打开项目文件 `main.qml`。

3. **运行项目**：
   在 Qt Creator 中点击运行按钮，即可看到动态数据曲线图的演示效果。

## 依赖项

- Qt 5.12 或更高版本
- Qt Charts 模块
