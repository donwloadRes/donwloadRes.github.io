# 二进制文件合并工具软件 PackagingTool

## 简介
PackagingTool 是一款功能强大的二进制文件合并工具软件，专为嵌入式开发人员设计。它能够帮助用户轻松合并多个二进制文件（如 `.bin` 文件），并支持自由添加、删除、调整文件顺序，以及设定每个文件的偏移地址。该工具特别适用于与瑞佑芯片 RA8889 结合使用，能够显著提升开发和生产效率。

## 主要功能
1. **多文件合并**：
   - 支持合并多个 `.bin` 文件。
   - 可以自由添加、删除文件，并调整文件的排列顺序。
   - 能够设定每个文件的偏移地址，方便嵌入式系统调用。

2. **文件信息保存**：
   - 可以将文件信息保存于生成的 `.bin` 文件头，方便 MCU 直接调用。

3. **IAP BootLoader 与 Application Code 合并**：
   - 支持合并 IAP BootLoader Code 和 Application Code，并预设偏移地址（如 `0x00010000`）。

4. **生成 `.h` 文件**：
   - 除了生成 `.bin` 文件外，还会生成一份 `.h` 文件。
   - 该 `.h` 文件已构建枚举，MCU 可以直接引用索引调用。

5. **重复导入列表**：
   - 支持重复导入列表，只需导入生成的 `.h` 文件即可，大大提高开发和生产效率。

## 使用说明
1. 打开 PackagingTool 软件。
2. 添加需要合并的 `.bin` 文件。
3. 调整文件顺序，并设定每个文件的偏移地址。
4. 点击“合并”按钮，生成合并后的 `.bin` 文件和对应的 `.h` 文件。
5. 将生成的 `.h` 文件导入到 MCU 项目中，即可直接引用索引调用。
