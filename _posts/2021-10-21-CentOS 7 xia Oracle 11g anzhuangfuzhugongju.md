# CentOS 7 下 Oracle 11g 安装辅助工具

## 资源简介

**CentOS7Oracle11gInstallHelper.zip** 是一款专为简化 CentOS 7 系统上 Oracle 11g 数据库安装过程而设计的脚本集合。此资源主要针对的是 CentOS Linux release 7.9.2009 (Core) 版本的系统和 Oracle 11.2.0.4 版本数据库。

## 主要功能

这个脚本套件精心设计，以自动化以下关键安装步骤：

1. **创建必要用户与组**：自动创建用于运行 Oracle 数据库的服务账户 `oracle` 及相关用户组。
   
2. **配置图形化操作环境**：通过 VNC 设置远程访问能力，确保可以通过图形界面进行数据库的安装与管理，开放 VNC 默认服务端口 5901。
   
3. **防火墙规则设置**：自动调整防火墙规则，确保 Oracle 使用的 1521 端口及 VNC 的 5901 端口对外可访问。
   
4. **依赖包安装**：预先处理好安装 Oracle DB 所需的所有依赖软件包。
   
5. **中文字体支持**：安装必要的中文字体包，避免在 Oracle 应用中出现中文乱码问题。
   
6. **特别说明**：此脚本集还包含了对 `pdksh-5.2.14` 的单独安装，解决特定需求或兼容性问题。
