# PHP+MySQL+Bootstrap 美食主题博客项目

## 项目简介

本项目是一个基于PHP、MySQL和Bootstrap开发的美食主题博客系统。该项目旨在为用户提供一个展示和分享美食的平台，用户可以发布美食相关的文章、图片和视频，同时支持用户之间的互动，如点赞、评论和分享。

## 技术栈

- **前端**: HTML, CSS, JavaScript, Bootstrap, jQuery
- **后端**: PHP
- **数据库**: MySQL
- **架构**: MVC（Model-View-Controller）

## 功能特性

1. **用户管理**:
   - 用户注册与登录
   - 用户信息展示与修改
   - 用户头像上传与管理

2. **博客功能**:
   - 发布美食文章
   - 文章分类与标签
   - 文章搜索与关键字标红

3. **互动功能**:
   - 文章点赞与评论
   - 用户动态发布与展示
   - 好友添加与删除

4. **API接口**:
   - 调用外部美食数据接口
   - 实现菜谱推荐、分类、详情等功能

## 项目结构

- **public**: 存放静态样式文件
- **upload**: 存放用户上传的图片
- **util**: 存放工具类文件，如验证码生成
- **Controller**: 控制层，处理用户请求
- **View**: 视图层，展示页面内容

## 使用说明

1. **环境配置**:
   - 安装PHP和MySQL
   - 配置Apache服务器

2. **数据库设置**:
   - 导入项目提供的SQL文件
   - 修改数据库连接配置

3. **运行项目**:
   - 启动Apache服务器
   - 访问项目首页

## 注意事项

- 验证码生成路径需根据实际环境进行调整。
- 项目中使用了外部API接口，确保网络连接正常。
