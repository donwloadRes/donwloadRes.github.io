# 基于知识图谱的推荐系统项目

## 项目介绍

本项目提供了一个基于知识图谱的推荐系统的完整代码，并附带了相应的数据集。数据集位于`data`文件夹下，以txt格式存储，包括训练集、验证集和测试集。

## 数据集说明

- **训练集**：用于模型训练的数据集。
- **验证集**：用于模型调参和验证的数据集。
- **测试集**：用于最终评估模型性能的数据集。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```

3. **运行代码**：
   ```bash
   python main.py
   ```
