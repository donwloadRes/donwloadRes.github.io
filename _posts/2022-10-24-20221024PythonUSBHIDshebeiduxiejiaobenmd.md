# Python USB HID 设备读写脚本

## 简介

本仓库提供了一个用于读取和写入 USB HID 设备的 Python 脚本。该脚本基于 Python 2.7.9 开发，支持在 Windows 7 64 位系统上进行 USB HID 设备的原始读取和写入操作。

## 功能描述

- **USB HID 设备读取**：脚本能够从连接的 USB HID 设备中读取原始数据。
- **USB HID 设备写入**：脚本支持向 USB HID 设备写入原始数据。

## 使用说明

1. **环境要求**：
   - Python 2.7.9
   - Windows 7 64 位操作系统

2. **安装依赖**：
   - 确保系统中已安装 Python 2.7.9。
   - 安装所需的 Python 库（如有）。

3. **运行脚本**：
   - 下载本仓库中的脚本文件。
   - 在命令行中导航到脚本所在目录。
   - 运行脚本，按照提示进行 USB HID 设备的读取或写入操作。

## 注意事项

- 该脚本在 Windows 7 64 位系统上测试通过，其他操作系统或版本可能需要进行适配。
- 使用前请确保已正确连接 USB HID 设备。
