# Notepad++ 安装包

## 描述

本仓库提供了一个经过优化的 Notepad++ 安装包，解决了在百度上下载的 Notepad++ 安装包中常见的插件管理器（Plugin Manager）问题。具体来说，有些安装包在安装后没有 Plugin Manager，或者 Plugin Manager 无法正常加载插件列表。本安装包完美解决了这些问题，Plugin Manager 中包含了你需要的所有插件，并且经过亲测，确保可用。

## 使用说明

1. **下载安装包**：点击仓库中的安装包文件进行下载。
2. **安装 Notepad++**：运行下载的安装包，按照提示完成 Notepad++ 的安装。
3. **使用 Plugin Manager**：安装完成后，打开 Notepad++，你将看到 Plugin Manager 已经正常加载，并且可以轻松浏览和安装各种插件。
