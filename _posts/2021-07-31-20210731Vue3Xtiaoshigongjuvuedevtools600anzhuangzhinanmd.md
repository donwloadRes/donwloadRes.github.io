# Vue 3.X 调试工具 vue-devtools-6.0.0 安装指南

## 简介
本仓库提供了一个用于 Vue 3.X 开发的调试工具 `vue-devtools-6.0.0` 的安装包。该工具是 Vue 开发者必备的调试利器，能够显著提升调试效率，使得 Vue 3.X 应用的调试变得更加直观和高效。

## 安装步骤
1. **下载安装包**：从本仓库下载 `vue-devtools-6.0.0` 安装包。
2. **解压安装包**：将下载好的安装包解压到你希望安装的目录。
3. **导入 Chrome 浏览器**：在 Chrome 浏览器中，打开扩展程序页面（输入 `chrome://extensions/`），开启开发者模式，然后将解压后的安装包拖入页面中。
4. **启用调试工具**：安装完成后，打开 Vue 应用的页面，在 Chrome 开发者工具中会出现一个名为 `Vue` 的标签，通过该标签可以查看 Vue 应用的运行信息，方便进行调试。

## 注意事项
- 该工具适用于 Vue 3.X 版本，能够支持 Vue 3.X 的新特性和变化。
- 建议在开发环境中使用，生产环境中不推荐使用。
