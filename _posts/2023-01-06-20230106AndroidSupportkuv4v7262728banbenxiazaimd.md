本仓库提供了Android Support库的v4和v7版本的26、27、28三个版本的下载资源。这些资源文件对于开发Android应用程序时使用旧版本的Support库非常有用。

## 资源文件列表

- `android-support-v4.jar`：适用于Android Support库v4的26、27、28版本。
- `appcompat-v7.jar`：适用于AppCompat库v7的26、27、28版本。

## 使用说明

1. 下载所需的`android-support-v4.jar`或`appcompat-v7.jar`文件。
2. 将下载的文件放置到您的Android项目中的`libs`目录下。
3. 在项目的`build.gradle`文件中添加依赖项，例如：
   ```gradle
   dependencies {
       implementation files('libs/android-support-v4.jar')
       implementation files('libs/appcompat-v7.jar')
   }
   ```
4. 同步项目并开始使用这些Support库进行开发。
