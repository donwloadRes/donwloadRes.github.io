# PCL实现的曲面重建技术演示

## 概述

本仓库提供了基于Point Cloud Library (PCL) 的曲面重建项目，旨在展示三种主流的三维重建算法：**泊松重建(Poisson Surface Reconstruction)**、**贪婪三角化(Greedy Triangulation)** 和 **移动立方体算法(Moving Least Squares, 实际上MLS更多用于点云平滑，但其概念在某些曲面重建流程中被应用)**。通过集成Qt作为图形用户界面(GUI)，实现了对这三种算法处理结果的便捷可视化，特别选用了经典的“Bunny”模型作为示例，以直观展示每种算法的特点和效果。

## 技术栈

- **Point Cloud Library (PCL)**: 一个开源的3D处理库，支持从简单的点云操作到复杂的曲面重建等高级算法。
- **Qt**: 一个跨平台的应用程序开发框架，用于创建用户友好的图形界面。
- **C++**: 项目的编程语言，确保高效执行计算密集型任务。

## 功能特点

1. **泊松重建**: 利用泊松方程解决不规则点云表面重建问题，生成连续光滑的表面。
2. **贪婪三角化**: 快速构建点云的三角网格，适用于快速原型或基础形状重建。
3. **移动立方体算法**: 虽然主要提及的是MLS，但在这个上下文中理解为点云平滑及局部拟合，间接辅助重建过程，提升表面质量。

## 使用指南

1. **环境搭建**: 确保你的开发环境中安装了PCL库和Qt，并配置好相应的C++编译器。
2. **编译项目**: 打开项目解决方案文件，在Qt Creator或其他IDE中进行编译。
3. **运行演示**: 编译成功后，运行应用程序。软件会加载预设的"Bunny"点云数据。
4. **选择算法**: 在界面上选择想要测试的曲面重建算法。
5. **可视化结果**: 算法处理完毕后，将直接在GUI中显示重建后的曲面模型。

## 注意事项

- 确保所有依赖库版本兼容，避免因库版本差异导致的问题。
- 由于点云数据和算法的复杂性，处理较大的数据集时可能会消耗较长时间或较多系统资源。
- 修改源代码前，请备份原文件，以防不当修改影响功能。
