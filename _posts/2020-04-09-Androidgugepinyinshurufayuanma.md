# Android谷歌拼音输入法源码

## 资源描述

本仓库提供了一个完整的Android谷歌拼音输入法源码，适用于对输入法定制有需求的开发者。该输入法支持中英文切换，并且已经通过C++代码和cmake编译，整个工程可以编译成一个输入法APK。

## 开发环境

- **开发语言**: Java
- **开发环境**: Android Studio Chipmunk 2021.2.1

## 内容概要

- **谷歌拼音输入法**: 支持中英文切换。
- **C++代码**: 通过cmake编译通过。
- **工程编译**: 整个工程可以编译成一个输入法APK。
- **安装使用**: 安装后需要在设置里添加此输入法，再选择使用即可。

## 适合人群

- C++/Java开发者
- 对输入法定制有需求的人群

## 使用说明

1. 克隆或下载本仓库的源码。
2. 使用Android Studio Chipmunk 2021.2.1打开工程。
3. 编译工程，生成输入法APK。
4. 安装生成的APK，并在系统设置中添加并选择此输入法。

## 注意事项

- 确保开发环境为Android Studio Chipmunk 2021.2.1，以保证代码的兼容性。
- 在编译和运行过程中，请确保所有依赖项已正确配置。
