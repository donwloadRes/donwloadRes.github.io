# Python Excel Sheet 列提取与汇总工具

## 简介

本资源文件提供了一个用Python编写的工具，用于从一个Excel文件中的多个Sheet中提取相同的列，并将这些列的信息汇总到一个新的Excel文件中。该工具使用`openpyxl`库来处理Excel文件，并支持为提取的列添加别名。

## 功能描述

1. **读取Excel文件**：使用`openpyxl`库读取Excel文件中的多个Sheet。
2. **设定提取列**：用户可以设定需要从多个Sheet中提取的列。
3. **提取与汇总**：运行脚本后，程序会提取指定列的信息，并将其汇总到一个新的Excel文件中。用户还可以为提取的列添加别名。
4. **输出结果**：最终生成一个新的Excel文件，其中包含汇总后的数据。

## 文件说明

- **源码**：包含Python脚本的源代码，可供参考和学习。
- **summary.xlsx**：示例源数据文件，包含多个Sheet，用于测试和演示。
- **result.xlsx**：输出的结果文件，包含从多个Sheet中提取并汇总的数据。

## 使用方法

1. 下载并解压资源文件。
2. 打开Python脚本，根据需要修改要提取的列和别名。
3. 运行脚本，生成汇总后的Excel文件。

## 注意事项

- 请确保已安装`openpyxl`库，可以使用`pip install openpyxl`进行安装。
- 示例文件`summary.xlsx`仅供测试使用，实际使用时请替换为实际的Excel文件。
