# 全球城市数据库（含经纬度及国家省份）

## 简介

本仓库提供了一个包含全球78707个主要城市信息的MySQL数据库文件。该数据库文件已经导出成SQL格式，方便用户直接导入到MySQL数据库中使用。每个城市记录包含了经纬度坐标值以及所属的国家和省份信息。

## 文件说明

- **cities.sql**: 包含全球78707个主要城市信息的SQL文件，文件大小约为10M。

## 使用方法

1. **下载文件**: 从本仓库下载`cities.sql`文件。
2. **导入数据库**: 使用MySQL命令行或图形化工具（如phpMyAdmin）将`cities.sql`文件导入到你的MySQL数据库中。

### 导入命令示例

```bash
mysql -u your_username -p your_database_name < cities.sql
```

## 数据字段说明

- **id**: 城市唯一标识符
- **name**: 城市名称
- **latitude**: 纬度坐标
- **longitude**: 经度坐标
- **country**: 国家名称
- **province**: 省份名称
