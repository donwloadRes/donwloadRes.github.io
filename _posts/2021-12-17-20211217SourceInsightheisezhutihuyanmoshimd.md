# Source Insight 黑色主题 - 护眼模式

## 简介

本仓库提供了一个专为 Source Insight 软件设计的黑色主题，旨在为用户提供更加舒适的编程环境，特别是在长时间使用时，能够有效减轻眼睛疲劳，提升工作效率。

## 资源文件

- **文件名称**: `source_insight_dark_theme.zip`
- **文件描述**: 该文件包含了 Source Insight 的黑色主题配置文件，用户只需将其导入到 Source Insight 中即可应用。

## 使用方法

1. **下载文件**: 点击仓库中的 `source_insight_dark_theme.zip` 文件进行下载。
2. **解压缩**: 将下载的压缩包解压到任意目录。
3. **导入主题**: 打开 Source Insight 软件，进入 `Options` -> `Preferences` -> `Colors`，点击 `Import` 按钮，选择解压后的主题文件进行导入。
4. **应用主题**: 在 `Colors` 选项卡中，选择导入的主题并点击 `Apply` 按钮，即可应用黑色主题。

## 注意事项

- 请确保 Source Insight 软件版本与主题文件兼容。
- 如果在导入过程中遇到问题，请参考 Source Insight 的官方文档或联系技术支持。
