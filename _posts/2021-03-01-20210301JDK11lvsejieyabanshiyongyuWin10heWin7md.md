# JDK 11 绿色解压版（适用于 Win10 和 Win7）

## 简介

本仓库提供了一个方便快捷的 JDK 11 绿色解压版，适用于 Windows 10 和 Windows 7 操作系统。该版本无需安装，解压后即可使用，配置简单，非常适合开发者在不同环境中快速部署和使用 JDK 11。

## 特点

- **解压即用**：无需繁琐的安装步骤，只需解压文件即可开始使用 JDK 11。
- **配置方便**：内置了必要的配置文件，用户只需简单设置环境变量即可快速上手。
- **兼容性强**：适用于 Windows 10 和 Windows 7 操作系统，满足不同用户的需求。

## 使用方法

1. **下载资源文件**：从本仓库下载 `jdk11绿色解压版win10win7.zip` 文件。
2. **解压文件**：将下载的压缩包解压到你希望安装 JDK 11 的目录中。
3. **配置环境变量**：
   - 右键点击“此电脑”或“计算机”，选择“属性”。
   - 点击“高级系统设置”，然后选择“环境变量”。
   - 在“系统变量”中找到 `Path` 变量，点击“编辑”。
   - 在 `Path` 变量中添加 JDK 11 的 `bin` 目录路径（例如：`C:\path\to\jdk11\bin`）。
4. **验证安装**：打开命令提示符，输入 `java -version`，如果显示 JDK 11 的版本信息，则表示配置成功。

## 注意事项

- 请确保解压路径中不包含中文或特殊字符，以免影响正常使用。
- 如果在使用过程中遇到问题，请检查环境变量配置是否正确。
