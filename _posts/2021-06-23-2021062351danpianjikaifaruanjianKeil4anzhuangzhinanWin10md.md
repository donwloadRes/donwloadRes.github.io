# 51单片机开发软件Keil4安装指南（Win10）

## 简介
本仓库提供了一个详细的安装指南，帮助用户在Windows 10系统上安装51单片机开发软件Keil4。Keil4是一款广泛使用的嵌入式开发工具，适用于51系列单片机的开发。

## 安装步骤

### 1. 下载安装包
首先，从提供的资源链接中下载Keil4的安装包。下载完成后，解压缩文件。

### 2. 安装Keil4
1. 打开解压后的文件夹，找到并运行`C51V901.exe`文件。
2. 按照安装向导的提示，逐步完成安装。在“Customer Information”页面，可以随意输入信息。
3. 安装完成后，不要选择“Show Release Notes”，但确保勾选其他选项，然后点击“Finish”。

### 3. 注册Keil4
1. 以管理员身份运行`KEIL_Lic.exe`文件，打开Keil Generic Keygen窗口。
2. 在窗口中选择“Target”为C51。
3. 从电脑桌面打开Keil uVision4，选择“File” -> “License Management”。
4. 复制“License Management”窗口中“CID”对应的内容，粘贴到Keil Generic Keygen窗口中“CID”对应的位置。
5. 点击“Generate”，生成注册码。
6. 将生成的注册码复制到“License Management”窗口中“New License ID Code(UC)”对应的方框中。
7. 点击“Add LIC”，完成注册。

## 注意事项
- 安装过程中，请确保所有路径中不包含中文字符。
- 注册时，确保以管理员身份运行相关程序，以避免权限问题。
