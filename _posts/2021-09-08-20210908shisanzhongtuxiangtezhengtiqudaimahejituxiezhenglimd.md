# 十三种图像特征提取代码合集（吐血整理）

## 简介

本仓库提供了一个包含13种图像特征提取代码的资源文件合集。这些代码涵盖了多种常用的图像特征提取方法，适用于图像处理、计算机视觉和机器学习等领域。每个特征提取方法都提供了详细的代码实现，方便用户快速上手和应用。

## 包含的特征提取方法

1. **01_Histogram** - 直方图特征提取
2. **02_GLCM** - 灰度共生矩阵特征提取
3. **03_Color** - 颜色特征提取
4. **04_ShapeContext** - 形状上下文特征提取
5. **05_SIFT** - 尺度不变特征变换（SIFT）特征提取
6. **06_HOG** - 方向梯度直方图（HOG）特征提取
7. **07_LBP** - 局部二值模式（LBP）特征提取
8. **08_Gabor** - Gabor滤波器特征提取
9. **09_SURF** - 加速稳健特征（SURF）特征提取
10. **10_Harris** - Harris角点检测特征提取
11. **11_FAST** - FAST角点检测特征提取
12. **12_BRIEF** - 二进制鲁棒独立基本特征（BRIEF）特征提取
13. **13_ORB** - 定向FAST和旋转BRIEF（ORB）特征提取

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo-url.git
   ```

2. **进入仓库目录**：
   ```bash
   cd your-repo-name
   ```

3. **选择所需的特征提取方法**：
   每个特征提取方法都存放在独立的文件夹中，文件夹名称与特征提取方法名称对应。

4. **运行代码**：
   根据每个文件夹中的README文件或代码注释，运行相应的代码进行特征提取。
