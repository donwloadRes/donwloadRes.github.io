# PyQt5 简单视频播放器

## 介绍

本仓库提供了一个基于 PyQt5 的简单视频播放器资源文件。该播放器利用 PyQt5 中的 QMediaPlayer 和 QVideoWidget 组件，实现了基本的视频播放功能。用户可以通过该播放器播放视频文件，并进行播放、暂停操作，同时还可以查看播放进度。

## 功能特点

- **视频播放**：支持播放本地视频文件。
- **播放控制**：提供播放、暂停功能，方便用户控制视频播放进度。
- **进度条**：显示当前播放进度，用户可以通过拖动进度条快速跳转到视频的任意位置。
- **列表功能**：支持视频文件列表，用户可以方便地切换不同的视频进行播放。

## 使用说明

1. **下载资源文件**：从本仓库下载提供的资源文件。
2. **安装依赖**：确保已安装 PyQt5 库。如果未安装，可以使用以下命令进行安装：
   ```bash
   pip install PyQt5
   ```
3. **运行播放器**：解压下载的资源文件，运行其中的主程序文件即可启动视频播放器。

## 注意事项

- 请确保系统中已安装必要的视频解码器，以支持播放不同格式的视频文件。
- 如果在使用过程中遇到任何问题，请参考 PyQt5 的官方文档或相关社区寻求帮助。
