# Redis Desktop Manager 安装及使用教程

## 简介
Redis Desktop Manager（简称RDM）是一款跨平台的Redis可视化管理工具，支持Windows、macOS和Linux系统。它提供了直观的图形界面，方便用户管理和操作Redis数据库。本教程将详细介绍如何安装和使用Redis Desktop Manager。

## 安装步骤

### 1. 下载安装包
可以从官网或百度网盘下载Redis Desktop Manager的安装包。

### 2. 安装过程
1. 双击运行安装包，进入安装向导界面。
2. 点击“Next”进入下一步。
3. 阅读并同意许可协议，点击“I Agree”。
4. 选择安装位置（可选），建议安装在非C盘。
5. 点击“Install”开始安装，耐心等待安装完成。
6. 安装完成后，勾选启动Redis Desktop Manager选项，点击“Finish”打开软件。

## 使用方法

### 1. 新建连接
1. 进入Redis Desktop Manager主界面。
2. 点击“新建连接”，填写连接信息：
   - Name：连接名称
   - Host：Redis服务器IP地址
   - Port：Redis服务器端口号
   - Auth：密码（如果设置了密码）
3. 点击“Test Connection”测试连接，确认无误后点击“OK”完成连接。

### 2. 操作Redis
1. 连接成功后，可以查看和管理Redis中的键值对。
2. 支持根据筛选条件查询、添加新键、重命名键、删除键等操作。
3. 支持命令控制台操作，执行常用的Redis命令。

## 功能特点
- 跨平台支持
- 直观的图形界面
- 支持命令控制台操作
- 丰富的Redis操作功能

通过本教程，您可以轻松安装和使用Redis Desktop Manager，提高Redis数据库的管理效率。
