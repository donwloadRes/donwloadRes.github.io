# 安装Visual Studio Code及配置C/C++运行环境MINGW64

本资源文件提供了在Windows 11系统上安装Visual Studio Code并配置C/C++运行环境MINGW64的详细步骤。通过本指南，您可以轻松地在Windows 11上搭建一个高效的C/C++开发环境。

## 内容概述

1. **安装Visual Studio Code**
   - 下载并安装Visual Studio Code。
   - 安装完成后，启动Visual Studio Code。

2. **配置MINGW64**
   - 下载MINGW64软件包。
   - 设置环境变量，确保系统能够识别MINGW64路径。
   - 检查MINGW64是否配置成功。

3. **安装必备插件**
   - 安装C/C++ Extension Pack，提供C/C++开发所需的核心功能。
   - 安装Code Runner，方便在Visual Studio Code中直接运行C/C++代码。

4. **最终效果**
   - 配置完成后，您可以在Visual Studio Code中编写C/C++代码，并通过右键点击“Run Code”来运行代码。

## 使用说明

1. **下载资源文件**
   - 下载本仓库提供的资源文件，包含MINGW64的安装包及相关配置文件。

2. **安装Visual Studio Code**
   - 按照指南中的步骤下载并安装Visual Studio Code。

3. **配置MINGW64**
   - 解压下载的MINGW64安装包，并将其放置在合适的目录下。
   - 设置系统环境变量，确保MINGW64路径被正确识别。

4. **安装插件**
   - 在Visual Studio Code中安装C/C++ Extension Pack和Code Runner插件。

5. **验证配置**
   - 编写简单的C/C++代码，通过右键点击“Run Code”来验证配置是否成功。

通过以上步骤，您将能够在Windows 11上顺利配置并使用Visual Studio Code进行C/C++开发。
