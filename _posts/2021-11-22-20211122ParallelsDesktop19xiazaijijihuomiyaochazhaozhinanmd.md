# Parallels Desktop 19 下载及激活密钥查找指南

本文将详细介绍如何下载 Parallels Desktop 19 以及如何查找您的 Parallels Desktop for Mac 激活密钥。

## 1. Parallels Desktop 19 下载

Parallels Desktop 19 是一款适用于 Mac 的虚拟化软件，允许您在 Mac 计算机上同时运行多个操作系统。以下是下载步骤：

1. **下载链接**：访问提供的下载链接，下载 Parallels Desktop 19 安装包。
2. **安装步骤**：
   - 直接点击安装包，软件将自动开始安装。
   - 安装完成后，准备一个 Windows 系统的安装镜像文件。
   - 按照 Parallels Desktop 的界面指引，创建一个新的虚拟机并在其中安装 Windows 操作系统。

## 2. 查找 Parallels Desktop for Mac 激活密钥

激活密钥是激活 Parallels Desktop 副本的必要条件。如果您丢失了激活密钥，可以使用以下方法重新获取：

### 2.1 在 Parallels 账户中查找

1. 打开您的 Parallels 账户仪表板。
2. 单击 Parallels Desktop for Mac 下的“激活的订阅”和永久许可证。
3. 在“许可证密钥”框中查看产品许可证。

### 2.2 通过 Parallels 在线商店查找

如果您是通过 Parallels 在线商店购买的 Parallels Desktop，请访问 Parallels 在线商店找回密钥，或搜索您的电子邮箱，查看来自 cleverbridge (no-reply@cleverbridge.com) 的邮件。

### 2.3 在电子邮件或盒子中查找

1. 如果您是通过 Parallels 代理商购买的，密钥会通过电子邮件发送到您购买时使用的电子邮件地址。
2. 如果您购买了盒装的 Parallels Desktop 副本，密钥应位于盒子内部。

### 2.4 使用终端命令查找

1. 启动终端（Finder >“应用程序”>“实用工具”）。
2. 执行以下命令：`prlsrvctl info --license`。
3. 在 Parallels Desktop 15 及更高版本中，输出将为您提供许可证密钥的一部分。

## 3. 激活 Parallels Desktop

1. 首次在 Parallels Desktop for Mac 中启动虚拟机时，将提示创建或登录 Parallels 账户。
2. 如果您没有 Parallels 账户，请选择“我是一名新用户”，单击“创建账户”并按照屏幕上的说明操作。
3. 如果已有账户，请选择“我有密码”，然后输入账户凭据。
4. 在下一步中输入的激活密钥将被注册到您登录到的账户。

通过以上步骤，您可以顺利下载并激活 Parallels Desktop 19，享受在 Mac 上运行多个操作系统的便利。
