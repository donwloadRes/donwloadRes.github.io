## 安装步骤
1. **升级pip**：
   首先，确保你的pip是最新版本。打开命令提示符或终端，输入以下命令：
   ```bash
   pip install --upgrade pip
   ```

2. **安装Gensim**：
   使用pip安装Gensim库。输入以下命令：
   ```bash
   pip install gensim
   ```

3. **安装依赖库**：
   在安装Gensim之前，需要先安装NumPy和SciPy库。安装顺序为：NumPy -> SciPy -> Gensim。确保按照顺序安装，以避免依赖问题。
