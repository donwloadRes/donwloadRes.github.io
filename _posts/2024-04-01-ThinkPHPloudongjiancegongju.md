# ThinkPHP漏洞检测工具

## 简介

本仓库提供了一个用于检测ThinkPHP框架漏洞的工具。该工具旨在帮助开发者和安全研究人员快速识别和修复ThinkPHP应用程序中的潜在安全漏洞。

## 功能特点

- **自动化检测**：自动扫描ThinkPHP应用程序中的常见漏洞。
- **报告生成**：生成详细的漏洞检测报告，便于分析和修复。
- **易于使用**：简单易用的命令行界面，适合各种技术水平的用户。

## 使用方法

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-repo/thinkphp-vulnerability-scanner.git
   ```

2. **安装依赖**：
   ```bash
   cd thinkphp-vulnerability-scanner
   pip install -r requirements.txt
   ```

3. **运行检测工具**：
   ```bash
   python scanner.py --target http://your-thinkphp-app.com
   ```

4. **查看报告**：
   检测完成后，工具会生成一个HTML格式的报告文件，路径为`reports/report.html`。
