# SpringBoot 基于 Java 的在线问诊系统

## 项目简介

本项目是一个基于 SpringBoot 和 Java 开发的在线问诊系统，旨在为用户提供便捷的在线医疗咨询服务。系统分为前台和后台两部分，涵盖了用户登录、医生管理、药品展示、咨询服务、订单管理等多个功能模块。

## 功能模块

### 前台功能

1. **用户登录模块**：
   - 用户注册
   - 用户登录
   - 用户退出
   - 修改密码

2. **医生模块**：
   - 医生注册
   - 医生登录
   - 上传资格证图片
   - 编写工作经验
   - 拟定价格

3. **药店药品显示模块**：
   - 药品分类
   - 查看药品名称和数量
   - 查看药品详细情况

4. **医生显示模块**：
   - 医生姓名
   - 医生工作经验
   - 价格
   - 医生评价（用户对该医生的评价，通过打分和评语）
   - 预约问诊

5. **咨询模块**：
   - 选择医生人选
   - 填写病症
   - 咨询结果和医生建议（对用户病症开出的药方和注意事项）

6. **订单模块**：
   - 查看订单
   - 取消未支付订单
   - 确认收货
   - 订单支付（支付时可以选择是到药店自取还是由药店配送）

### 后台功能

1. **登录界面**：
   - 管理员登录

2. **药店药品管理**：
   - 查询药品
   - 查看药品详情
   - 新添加功能：
     - 添加药品
     - 删除药品
     - 编辑药品详情页
     - 上传药品图片

3. **订单管理**：
   - 查看用户的订单
   - 选择取消订单
   - 订单发货

## 资源内容

本仓库提供了完整的项目源码和数据库文件，方便开发者进行二次开发和学习。

## 使用说明

1. **下载源码**：
   - 克隆或下载本仓库的源码到本地。

2. **导入数据库**：
   - 使用提供的 SQL 文件导入数据库。

3. **配置环境**：
   - 配置好 Java 开发环境和数据库连接信息。

4. **运行项目**：
   - 运行 SpringBoot 项目，访问系统前台和后台。

## 注意事项

- 请确保本地环境已安装 Java 和 MySQL 数据库。
- 在运行项目前，请根据实际情况修改数据库连接配置。
