# Source Insight 工具安装及使用方法

## 简介
Source Insight 是一款功能强大的代码阅读器，能够使大量的代码产生联系，方便阅读，并且支持各种语言的程序代码。本文将详细介绍 Source Insight 的安装及使用方法。

## 安装步骤
1. **下载 Source Insight**  
   从官方网站或指定下载地址下载 Source Insight 安装包。

2. **安装 Source Insight**  
   按照安装向导的步骤进行安装，可以选择自定义安装目录。

3. **激活 Source Insight**  
   - **方法一**：使用 key 文件激活。将下载的 key 文件拷贝至 Source Insight 安装目录下，运行激活工具生成 key 文件。
   - **方法二**：替换原主程序。下载破解文件并替换原主程序，然后导入激活文件完成激活。

## 新建工程
1. **创建新工程**  
   打开 Source Insight，选择 `Project → New Project`，输入工程名字和保存路径，点击 OK。

2. **添加源代码**  
   输入要编辑的程序源代码的路径，选择 `Add All` 添加所有文件，或手动选择添加文件。

## 使用方法
1. **代码阅读**  
   Source Insight 提供了强大的代码浏览功能，支持代码导航、符号搜索、快速跳转等。

2. **代码编辑**  
   使用代码补全、语法高亮、自动缩进等功能提高编码效率。

3. **代码分析**  
   使用符号浏览器查看函数和变量的定义和引用，分析代码的调用关系和继承关系。

## 注意事项
- 确保安装目录路径中没有中文或特殊字符。
- 在添加文件时，注意选择需要添加的源代码文件类型。

通过以上步骤，您可以顺利安装并使用 Source Insight 进行代码阅读和编辑。
