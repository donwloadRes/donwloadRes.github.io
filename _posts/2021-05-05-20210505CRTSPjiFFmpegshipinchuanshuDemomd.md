# C# RTSP 及 FFmpeg 视频传输 Demo

## 简介

本仓库提供了一个基于 C# 和 FFmpeg 的视频传输 Demo，主要用于演示如何通过 RTSP 协议进行视频流的传输。该 Demo 结合了 C# 的强大功能和 FFmpeg 的多媒体处理能力，为开发者提供了一个简单易用的视频传输解决方案。

## 功能特点

- **RTSP 视频流传输**：支持通过 RTSP 协议从摄像头或其他视频源获取实时视频流。
- **FFmpeg 集成**：利用 FFmpeg 进行视频流的解码和处理，确保视频传输的稳定性和高效性。
- **C# 实现**：使用 C# 语言编写，适合 .NET 开发者快速上手和集成。

## 使用说明

1. **环境准备**：
   - 确保已安装 .NET 开发环境。
   - 下载并配置 FFmpeg 库。

2. **运行 Demo**：
   - 克隆本仓库到本地。
   - 打开项目解决方案文件（.sln）。
   - 编译并运行项目。

3. **配置 RTSP 源**：
   - 在代码中配置 RTSP 视频流的 URL。
   - 启动程序，观察视频流的传输效果。

## 注意事项

- 请确保网络环境稳定，以保证 RTSP 视频流的传输质量。
- 根据实际需求，可以对代码进行扩展和优化。
