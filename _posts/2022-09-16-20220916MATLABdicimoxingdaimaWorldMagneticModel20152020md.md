# MATLAB地磁模型代码 - World Magnetic Model 2015-2020

该项目是世界电磁模型的C/C++实现，适用于2015年至2020年的地磁模型计算。代码已经通过MATLAB的验证，可以直接添加到您的项目中使用。

## 项目描述

本项目提供了一个基于C/C++的地磁模型代码，适用于2015年至2020年的世界电磁模型。代码已经通过MATLAB的验证，可以直接添加到您的项目中使用。代码是开源的，您可以自由分叉、分发，但不提供任何担保，并且完全免费。

### 输入参数

- **纬度**：-90.00 至 +90.00 度
- **经度**：-180.00 至 +180.00 度
- **高度**：参照WGS84椭球或平均海平面（MSL）
- **日期**：2015.0 至 2020.0

### 输出参数

代码将计算出以下七个地磁成分：

- **F**：地磁场的总强度
- **H**：地磁场的水平强度
- **X**：地磁场的北部分量
- **Y**：地磁场的东部分量
- **Z**：地磁场的垂直分量
- **I（DIP）**：地磁倾角
- **D（DEC）**：地磁偏角（磁变化）

## 使用方法

1. 克隆项目到本地：
   ```bash
   git clone cd WorldMagneticModel2015-2020
   ```

2. 进入项目目录并编译：
   ```bash
   cd WorldMagneticModel2015-2020
   make
   ```

3. 运行程序：
   ```bash
   ./WMM
   ```
