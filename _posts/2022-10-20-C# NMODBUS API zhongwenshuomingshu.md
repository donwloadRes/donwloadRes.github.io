# C# NMODBUS API 中文说明书

## 资源介绍

本仓库提供了一个名为“C# NMODBUS API 中文说明书.pdf”的资源文件，该文件详细介绍了如何在C#项目中使用NMODBUS库进行Modbus通信。

## 资源内容

1. **安装NMODBUS库**  
   在项目工程中，通过NuGet包管理器搜索并安装`nmodbus4`包。安装完成后，即可在项目中使用NMODBUS库。

2. **引用命名空间**  
   安装完成后，只需在代码中引用`using modbus.device`命名空间。根据具体需求，还可以添加其他必要的命名空间。

3. **实现功能**  
   根据项目需求，使用NMODBUS库提供的API实现Modbus通信功能。

## 使用说明

1. 下载本仓库中的“C# NMODBUS API 中文说明书.pdf”文件。
2. 按照说明书中的步骤，在C#项目中安装并使用NMODBUS库。
3. 根据项目需求，实现Modbus通信功能。
