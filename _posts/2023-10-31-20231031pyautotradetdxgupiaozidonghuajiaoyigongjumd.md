# pyautotrade_tdx: 股票自动化交易工具

## 概览

**pyautotrade_tdx** 是专为华泰证券通达信版用户设计的一款高效能股票自动化交易平台。该工具能够同时监控多达5只股票，基于预设的交易条件即时下单，确保交易响应时间低于2秒。为了便于交流和问题解决，我们设立了QQ群：486224275，欢迎加入讨论。

## 环境需求及安装

- **操作系统**: Windows 10 64位
- **Python版本**: 推荐使用Python 3.6 64位（32位亦可）
- **依赖库**: 
    - `pywin32`：用于Windows API的Python接口。
    - `tushare`: 获取市场数据的Python库。
    
安装依赖库推荐使用pip命令：
```
pip install pywin32 tushare
```

## 软件组成

- **pyautotrade.pyw**: 主程序文件，执行自动化交易的核心逻辑。
- **stockInfo.dat**: 存储股票监控信息的数据文件。
- **winguiauto.py**: 包含定制化的WinAPI函数，支持窗口操作和交互。

## 使用指南

1. **启动前准备**：首先配置好`stockInfo.dat`中的监控股票信息。
2. 运行**pyautotrade.pyw**，选择“对买对卖”模式，并保持界面在最前端。
3. 不设置时间条件默认下单时间为次日凌晨1点，价格条件需自行设定以符合交易策略。
4. 股票买卖数量须为100股的整数倍，如输入非标准数额，系统自动向下取整至最近的100股。
5. 默认情况下，若股票数量未指定，则视为0股，即不进行交易。
