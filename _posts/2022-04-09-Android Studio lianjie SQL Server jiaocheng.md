# Android Studio 连接 SQL Server 教程

本仓库提供了一个资源文件 `Android studio 连接SQL Server.rar`，该文件详细介绍了如何在 Android Studio 中连接 SQL Server 数据库，并读取其中的数据。通过本教程，您将学会如何创建线程来获取 SQL Server 中的数据，并在配置文件 `Androidmanifest.xml` 中添加必要的权限，以确保网络连接的正常运行。

## 资源文件内容

- **Android studio 连接SQL Server.rar**: 该文件包含了实现 Android Studio 连接 SQL Server 数据库的完整代码示例。通过解压该文件，您可以快速了解如何在 Android 应用中实现数据库连接和数据读取。

## 主要功能

1. **数据库连接**: 详细介绍了如何在 Android Studio 中配置 SQL Server 数据库连接。
2. **数据读取**: 通过创建线程，实现了从 SQL Server 数据库中读取数据的功能。
3. **权限配置**: 在 `Androidmanifest.xml` 文件中添加了 `<uses-permission android:name="android.permission.INTERNET"/>` 权限，确保应用能够正常进行网络连接。

## 使用说明

1. **下载资源文件**: 下载并解压 `Android studio 连接SQL Server.rar` 文件。
2. **导入项目**: 将解压后的项目导入到 Android Studio 中。
3. **配置数据库**: 根据您的 SQL Server 数据库信息，修改项目中的数据库连接配置。
4. **运行应用**: 运行应用，验证是否能够成功连接 SQL Server 并读取数据。

## 注意事项

- 确保您的 Android 设备或模拟器已连接到互联网。
