# C# WinForm 低功耗蓝牙通讯示例

## 项目简介

本项目是一个使用C#开发的低功耗蓝牙（BLE）通讯的Winform上位机应用程序。该应用程序实现了蓝牙设备的搜索、连接、读写数据以及正常断开连接等功能。通过本项目，您可以学习如何在C#中使用低功耗蓝牙技术进行设备间的通讯。

## 功能特性

- **蓝牙设备搜索**：自动搜索并列出附近的低功耗蓝牙设备。
- **蓝牙设备连接**：选择设备后，可以与其建立连接。
- **数据读写**：连接成功后，可以读取和写入蓝牙设备的数据。
- **正常断开连接**：在通讯结束后，可以安全地断开蓝牙连接。

## 使用说明

1. **克隆仓库**：
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```

2. **打开项目**：
   使用Visual Studio打开项目文件（`.sln`）。

3. **编译运行**：
   编译并运行项目，应用程序将启动并显示蓝牙设备搜索界面。

4. **操作步骤**：
   - 点击“搜索设备”按钮，列出附近的低功耗蓝牙设备。
   - 选择一个设备并点击“连接”按钮，建立蓝牙连接。
   - 连接成功后，可以使用“读取数据”和“写入数据”功能与设备进行通讯。
   - 通讯结束后，点击“断开连接”按钮，安全断开蓝牙连接。
