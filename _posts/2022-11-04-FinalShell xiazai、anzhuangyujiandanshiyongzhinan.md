# FinalShell 下载、安装与简单使用指南

## 简介
FinalShell 是一款功能强大的 SSH 客户端工具，支持 Windows、macOS 和 Linux 平台。它不仅提供了远程终端操作功能，还集成了文件传输、服务器管理等多种实用功能，是开发者和运维人员的好帮手。

## 下载
1. 访问 FinalShell 官方网站或相关下载页面。
2. 选择适合你操作系统的版本进行下载。
3. 下载完成后，双击安装包开始安装。

## 安装
1. 双击下载的安装包，启动安装向导。
2. 阅读并接受许可协议，点击“下一步”。
3. 选择安装位置，点击“安装”。
4. 安装过程中可能会提示安装 Winpcap 依赖，点击“确定”继续。
5. 安装完成后，点击“关闭”退出安装向导。

## 简单使用
1. 打开 FinalShell，点击左侧的“文件夹”图标。
2. 选择“SSH 连接(Linux)”，填写连接信息（名称、主机、用户名、密码）。
3. 双击创建的连接，弹出窗口点击“接受并保存”。
4. 连接成功后，即可进行远程操作。

## 注意事项
- 如果 ifconfig 命令无法使用，需安装 net-tools 软件包。
- 虚拟机 IP 地址可能会变化，建议设置固定 IP 地址。

## 功能特点
- 多平台支持：Windows、macOS、Linux。
- 多标签、批量服务器管理。
- 支持 SSH 和 Windows 远程桌面。
- 内置多种配色方案，支持平滑字体显示。
- 支持 SFTP 文件传输，拖放操作简单快捷。
- 服务器网络性能实时监控，支持双边 TCP 加速。

FinalShell 是一款功能全面、操作简便的 SSH 客户端工具，适合各类开发和运维人员使用。通过本指南，你可以轻松完成 FinalShell 的下载、安装和基本使用。
