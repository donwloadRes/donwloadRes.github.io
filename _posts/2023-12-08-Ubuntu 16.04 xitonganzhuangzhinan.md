# Ubuntu 16.04 系统安装指南

## 深度学习炼丹手册 - 第一篇

本资源文件是《深度学习炼丹手册》的第一篇，主要内容为 Ubuntu 16.04 系统的安装指南。本文详细介绍了从系统镜像下载到安装完成的整个过程，并提供了显卡驱动的安装方法，适合初次接触 Ubuntu 系统的深度学习爱好者和开发者。

### 内容概述

1. **系统镜像下载**
   - 提供了 Ubuntu 16.04 版本的下载链接。
   - 介绍了如何使用软碟通软件制作 Ubuntu U 盘启动盘。

2. **系统安装**
   - 详细说明了系统安装的步骤，包括分区挂载、启动引导器设置等。
   - 针对有/无核显的情况，提供了不同的安装建议。

3. **显卡驱动安装**
   - 推荐手动安装显卡驱动的方法。
   - 提供了参考博客链接，帮助解决安装过程中可能遇到的问题。

4. **特殊情况处理**
   - 针对老旧硬件设备，提供了 Legacy 引导和 MBR 磁盘格式的安装建议。
   - 解决了可能出现的 The system is running in low-graphics mode 问题。

### 适用人群

- 初次接触 Ubuntu 系统的用户。
- 需要安装 Ubuntu 16.04 进行深度学习开发的开发者。
- 对显卡驱动安装有需求的用户。
