# Python实现多功能天气预报系统

## 项目简介

本项目是一个使用Python编写的多功能天气预报系统。用户可以在主界面输入想要查看未来天气情况的城市，系统会自动爬取网站数据，保存数据表格并显示天气信息。此外，系统还支持将未来天气情况绘制成折线图和饼状图，方便用户直观地了解天气变化趋势。代码注释丰富，简单实用，非常适合学习和参考。

## 功能特点

- **城市天气查询**：用户可以输入任意城市名称，系统会自动获取该城市的未来天气情况。
- **数据保存**：获取的天气数据会自动保存为表格文件，方便用户后续查看和分析。
- **图表展示**：系统支持将天气数据绘制成折线图和饼状图，帮助用户更直观地了解天气变化。
- **代码注释**：代码中包含丰富的注释，便于初学者理解和学习。

## 使用说明

1. **安装依赖**：在运行代码之前，请确保已安装所需的Python库。可以使用以下命令安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

2. **运行程序**：直接运行主程序文件，按照提示输入城市名称即可获取天气信息。

3. **查看结果**：系统会自动生成天气数据表格和图表，用户可以在指定目录下查看结果。

## 适用人群

- **Python初学者**：代码注释详细，适合初学者学习Python编程和数据处理。
- **数据分析爱好者**：系统支持数据保存和图表展示，适合对天气数据分析感兴趣的用户。
